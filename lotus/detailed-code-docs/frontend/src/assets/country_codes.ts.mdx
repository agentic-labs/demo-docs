## High-level description
This code defines a TypeScript constant `country_json` which is an array of objects. Each object represents a country with its two-letter country code (`Code`) and the full country name (`Name`).

## Symbols

### `country_json`
#### Description
This symbol is a constant array of objects containing country codes and names. It is exported for use in other parts of the application.

#### Inputs
None

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| `country_json` | `{ Code: string; Name: string }[]` | An array of objects, each containing a `Code` (two-letter country code) and a `Name` (full country name). |

#### Internal Logic
The array is hardcoded with data for various countries. Some country names include Unicode escape sequences (e.g., `\u00c5land Islands`, `C\u00f4te d'Ivoire`) to represent special characters.

## References
This code is referenced in `frontend/src/components/Customers/CustomerInfo.tsx` and `frontend/src/components/Settings/settings/tabs/GeneralTab.tsx` to populate country selection dropdowns.
