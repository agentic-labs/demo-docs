## High-level description
The `CopyText` component is a reusable React component that provides a user-friendly way to copy text to the clipboard. It displays the text to be copied and provides a visual indication of the copy status (copied or not copied).

## Code Structure
The `CopyText` component is a functional component that uses the `useState`, `useRef`, and `useEffect` hooks. It renders a `Tooltip` component from the Ant Design library to display the copy status. The text to be copied is displayed within a `code` element, which can be styled according to the provided `language` prop.

## References

- `Tooltip` component from the Ant Design library.

## Symbols

### `CopyText`
#### Description
A functional React component that allows users to copy text to the clipboard.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| textToCopy | string | The text to be copied to the clipboard. |
| className | string | Optional. Additional CSS class names to apply to the component. |
| showIcon | boolean | Optional. Whether to show the copy icon. Defaults to `false`. |
| onlyIcon | boolean | Optional. Whether to only show the copy icon and hide the text. Defaults to `false`. |
| language | string | Optional. The language of the code to be copied, used for syntax highlighting. |

#### Outputs
The component renders a `div` element containing the text to be copied and a `Tooltip` component that displays the copy status.

#### Internal Logic
1. The component uses the `useState` hook to track the copy status (`copySuccess`).
2. The `copyToClipBoard` function handles the copy action:
    - It uses the `navigator.clipboard.writeText` API to copy the text to the clipboard.
    - It sets the `copySuccess` state to `true` to indicate a successful copy.
    - It uses a timeout to reset the `copySuccess` state to `false` after 3 seconds.
3. The `Tooltip` component displays a different message based on the `copySuccess` state:
    - If `copySuccess` is `true`, it displays a "Copied" message with a checkmark icon.
    - If `copySuccess` is `false`, it displays a "Click to Copy" message with a copy icon.
4. The text to be copied is rendered within a `code` element, which can be styled according to the provided `language` prop.
