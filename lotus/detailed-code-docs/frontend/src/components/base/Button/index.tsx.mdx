## High-level description
The code defines a React component named `Button` that wraps the Ant Design `Button` component. It allows for customization of the button's appearance through a `level` prop, which determines the button's style.

## Code Structure
The `Button` component is a simple functional component that takes `level` and `children` props. It uses the Ant Design `Button` component (`BTN`) and applies custom styling based on the `level` prop.

## References
- `antd`: The Ant Design library is used for the `Button` component.
