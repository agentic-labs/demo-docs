## High-level description
The code defines a React component called `Tooltip` and its two variations: `TooltipTop` and `TooltipBottom`. These components are used to display tooltips on hover, with the option to position them at the top or bottom of the content.

## Code Structure
The `Tooltip` component acts as a wrapper and provides basic styling. The `TooltipTop` and `TooltipBottom` components extend the basic functionality by adding specific positioning and styling for top and bottom tooltips respectively. These variations are then attached to the main `Tooltip` component as static properties.
