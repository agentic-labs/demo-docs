## High-level description
This code defines a React component called `PricingUnitDropDown` that renders a dropdown menu for selecting pricing units (currencies). It fetches a list of available currencies from an API and allows the user to select one, optionally showing an "All Currencies" option.

## Code Structure
The main component `PricingUnitDropDown` uses the `useQuery` hook to fetch currency data and the antd `Select` component to render the dropdown. It also includes helper functions for processing the currency data and handling selection changes.

## Symbols

### PricingUnitDropDown
#### Description
A React functional component that renders a dropdown menu for selecting pricing units (currencies).

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| defaultValue | string | Optional. The default selected value for the dropdown. |
| setCurrentCurrency | function | A callback function to set the selected currency code. |
| setCurrentSymbol | function | Optional. A callback function to set the selected currency symbol. |
| shouldShowAllOption | boolean | Optional. If true, adds an "All Currencies" option to the dropdown. |
| disabled | boolean | Optional. If true, disables the dropdown. Default is false. |
| size | "large" \| "small" \| "middle" | Optional. Sets the size of the dropdown. Default is "small". |
| className | string | Optional. Additional CSS class names for the dropdown. |

#### Internal Logic
1. Fetches currency data using the `useQuery` hook.
2. Processes the currency data with the `getCurrencies` function, optionally adding an "All Currencies" option.
3. Renders an antd `Select` component with the processed currency options.
4. Handles currency selection changes, updating the selected currency and symbol.
