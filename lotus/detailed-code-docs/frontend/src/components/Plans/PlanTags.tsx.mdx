## High-level description
This code defines a React component called `PlanTags` that displays tags associated with a plan. It handles rendering different tag layouts based on the number of tags and provides an option to add new tags.

## Code Structure
The main `PlansTags` component takes props for tags and an optional flag to show an "Add Tag" button. It uses conditional rendering to display tags differently based on their count and the demo environment status.

## Symbols

### PlansTags
#### Description
This is the main functional component that renders the plan tags.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| tags | components["schemas"]["PlanDetail"]["tags"] \| components["schemas"]["Plan"]["tags"] | An array of tag objects |
| showAddTagButton | boolean | Optional flag to show an "Add Tag" button |

#### Internal Logic
1. Checks if there are no tags and if it's not a demo environment. If true, renders an "Add Tag" badge.
2. If there are tags:
   - For 3 or more tags, displays the first 2 tags and shows a count of remaining tags.
   - For fewer than 3 tags, displays all tags.
3. Optionally renders an "Add Tag" button based on conditions.
