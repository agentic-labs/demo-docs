## High-level description
This CSS file defines styles for a plan card component, likely used in a subscription or pricing page. It includes styles for hover effects, active subscriptions, plan details, and an archive option.

## Symbols

### `.planCard:hover`
#### Description
Applies a box shadow effect when hovering over the plan card, creating a subtle elevation effect.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| box-shadow | CSS property | Adds a shadow effect to the card on hover |

### `.activeSubscriptions`
#### Description
Defines the text color for active subscriptions.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| color | CSS property | Sets the text color to a specific shade of brown (#c3986b) |

### `.planDetailsValue`
#### Description
Styles the value of plan details, making them bold and centered.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| font-weight | CSS property | Sets the font weight to 600 (semi-bold) |
| color | CSS property | Sets the text color to a dark gray (#1d1d1f) |
| align-self | CSS property | Centers the content vertically within its container |

### `.planDetails`
#### Description
Defines the overall style for plan details, including font size, padding, and alignment.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| font-size | CSS property | Sets the font size to 16 pixels |
| padding | CSS property | Adds 8 pixels of vertical padding |
| display | CSS property | Uses flexbox for layout |
| align-items | CSS property | Centers items vertically within the container |

### `.planMenuArchiveIcon`
#### Description
Styles the archive icon in the plan menu, setting its color and layout properties.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| color | CSS property | Sets the icon color to a shade of red (#c20022) |
| align-items | CSS property | Centers items vertically |
| justify-content | CSS property | Centers items horizontally |
| display | CSS property | Uses flexbox for layout |
| padding | CSS property | Adds 5 pixels of padding around the icon |

### `.archiveLabel`
#### Description
Positions the archive label relative to its container.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| margin-left | CSS property | Adds 5 pixels of left margin |
| margin-top | CSS property | Adds 7 pixels of top margin |

## Side Effects
This CSS file does not have any side effects as it only defines styles and does not modify any global state or external resources.
