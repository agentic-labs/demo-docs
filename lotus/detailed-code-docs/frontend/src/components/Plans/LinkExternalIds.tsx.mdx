## High-level description
This code defines a React component called `LinkExternalIds` that allows users to manage external IDs linked to a plan. It provides functionality to display, add, and remove external IDs using tags.

## Code Structure
The main component `LinkExternalIds` is defined as a functional component. It uses various hooks and sub-components to manage state and render the UI. The component is then memoized and exported as the default.

## Symbols

### LinkExternalIds
#### Description
This is the main functional component that renders the UI for managing external IDs.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| externalIds | string[] | Array of existing external IDs |
| setExternalLinks | (links: string[]) =&gt; void | Optional function to update external links |
| createExternalLink | (link: string) =&gt; void | Optional function to create a new external link |
| deleteExternalLink | (link: string) =&gt; void | Optional function to delete an external link |

#### Internal Logic
1. Manages state for input visibility, input value, and refs for input elements.
2. Renders existing external IDs as tags.
3. Provides functionality to add new external IDs.
4. Handles tag removal.

The component is memoized using `React.memo` to optimize re-renders when props haven't changed.
