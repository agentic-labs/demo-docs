## High-level description
This CSS file (CustomPagination.css) defines the styles for a custom pagination component. It provides styling for movement buttons, disabled states, and the current page number display, enhancing the visual appearance and user interaction of the pagination controls.

## Symbols

### `.movementButton`
#### Description
Defines the style for the movement buttons in the pagination component.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| N/A | CSS Class | Applied to button elements |

#### Internal Logic
- Sets the cursor to a pointer, indicating the button is clickable
- Sets the font size to 12 pixels

### `.movementButton[disabled]`
#### Description
Defines the style for disabled movement buttons in the pagination component.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| N/A | CSS Selector | Applied to disabled button elements with class `movementButton` |

#### Internal Logic
- Sets the color to a light gray (#666666)
- Changes the cursor to 'not-allowed', indicating the button is not clickable

### `.currentPageNumber`
#### Description
Defines the style for the current page number display in the pagination component.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| N/A | CSS Class | Applied to the element displaying the current page number |

#### Internal Logic
- Sets a fixed height and width of 30 pixels
- Centers the content both vertically and horizontally using flexbox
- Sets the font weight to 400 (normal)
- Centers the text
- Adds a light gray border with rounded corners

## References
This CSS file is directly referenced and used by the `CustomPagination.tsx` component, which imports it using:
```typescript
import "./CustomPagination.css";
```

The styles defined in this CSS file are applied to the corresponding elements in the `CustomPagination` component:

1. The `.movementButton` class is applied to the navigation buttons (Start, Previous, and Next).
2. The `.currentPageNumber` class is applied to the div displaying the current page number.

The disabled state styling (`.movementButton[disabled]`) is automatically applied when the `disabled` attribute is set on the button elements in the React component.

This CSS file works in conjunction with the `CustomPagination.tsx` component to create a visually appealing and functional pagination control, enhancing the user interface for navigating through paginated content.
