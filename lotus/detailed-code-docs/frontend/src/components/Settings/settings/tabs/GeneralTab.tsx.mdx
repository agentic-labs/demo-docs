## High-level description
The GeneralTab component is a React functional component that displays and allows editing of organization settings. It includes fields for company name, default currency, billing address, tax rate, timezone, subscription filters, and tax providers. The component uses various Ant Design components for the user interface and react-query for data fetching and mutation.

## Code Structure
The component is structured as a single functional component with several hooks for state management and API calls. It uses the Ant Design Form component for handling form inputs and submissions. The component also includes a modal for editing organization settings.

## Symbols

### GeneralTab
#### Description
The main component that renders the organization settings view and edit functionality.

#### Internal Logic
1. Fetches organization data and pricing units using react-query.
2. Manages state for various organization settings.
3. Renders a view of current organization settings.
4. Provides an edit modal for updating organization settings.
5. Handles form submission and API calls for updating organization settings.

### useQuery hooks
#### Description
Used to fetch organization data and pricing units from the API.

### useMutation hook
#### Description
Used to handle the update of organization settings.

### Form handlers and state setters
#### Description
Various functions to handle form input changes and set component state.

|:-------|:-----|:--------|:------------|

## Error Handling
The component uses react-toastify to display error messages when API calls fail. It also includes client-side validation for form inputs.

## API/Interface Reference
The component interacts with the following API endpoints:
| Endpoint | Method | Description |
|:---------|:-------|:------------|
| app/pricing_unit_list | GET | Fetches available pricing units |
| app/organization | GET | Fetches organization details |
| app/organizations/{org_id} | PATCH | Updates organization settings |

Note: The actual API calls are abstracted through the imported API functions.
