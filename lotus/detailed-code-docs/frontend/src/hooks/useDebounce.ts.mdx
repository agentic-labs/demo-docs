## High-level description
The `useDebounce` hook in `useDebounce.ts` provides a debounced version of a given value, which only updates after a specified delay. This is useful for optimizing performance by preventing rapid updates to state or expensive operations.

## Symbols

### `useDebounce`
#### Description
This hook takes a value and a delay as input and returns a debounced version of the value. It uses `useState` to store the debounced value and `useEffect` to schedule updates after the specified delay.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| value | `T` | The value to be debounced. |
| delay | `number` | The delay in milliseconds before updating the debounced value. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| debouncedValue | `T` | The debounced version of the input value. |

#### Internal Logic
1. Initializes state for the debounced value using `useState`, setting it to the initial value.
2. Uses `useEffect` to handle updates to the debounced value.
3. Inside the effect, sets a timeout using `setTimeout` to update the debounced value after the specified delay.
4. Returns a cleanup function from the effect to clear the timeout using `clearTimeout` if the component unmounts or the value or delay changes before the timeout completes.
5. Returns the debounced value.
