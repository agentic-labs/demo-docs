## High-level description
This code defines the routing configuration for a React application using React Router. It sets up various routes for different pages and components, including a dashboard, plans, customers, settings, and various integrations. The component also includes a sidebar navigation menu.

## Code Structure
The main component `AppRoutes` defines the routing structure using `react-router-dom`. It uses the `Layout` component from Ant Design to create a layout with a collapsible sidebar. The sidebar content is provided by the `SideBar` component. The routes are defined using the `Routes` and `Route` components from React Router.

## Symbols

### `AppRoutes`
#### Description
The main functional component that defines the routing structure of the application.

#### Internal Logic
1. Sets up state for sidebar collapse
2. Uses `useEffect` to set initial collapse state based on window width
3. Renders a layout with a collapsible sidebar and main content area
4. Defines routes for various pages and components

### `getSettingsTab`
#### Description
A helper function that conditionally renders the settings component or redirects to the home page.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| component | React.ReactNode | The component to render for settings |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| output | React.ReactNode | The component to render or a redirect |
