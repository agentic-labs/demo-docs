## High-level description
This file defines action creators for managing plan-related state in a React application. It exports functions that create action objects with specific types and payloads, which can be used to update the application's state through a reducer.

## Symbols

### `setCurrentPlan`
#### Description
Creates an action to set the current plan.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| data | PlanType \| null | The plan data to set as current |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| action | Object | An action object with type SET_CURRENT and the provided data as payload |
