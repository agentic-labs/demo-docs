## High-level description
This file defines TypeScript interfaces for error responses in a frontend application. It includes structures for general error responses, validation errors, and specific error types related to API queries and responses.

## Symbols

### `ErrorResponse`
#### Description
Defines the structure for a general error response from the API.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| type | URL | The type of error, represented as a URL |
| title | string | The title or summary of the error |
| detail | string | A detailed description of the error |
| validation_errors | ValidationError[] | Optional array of validation errors |

### `ValidationError`
#### Description
Defines the structure for individual validation errors.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| code | string | The error code |
| detail | string | A detailed description of the validation error |
| attr | string | The attribute or field that caused the validation error |

### `QueryErrors`
#### Description
Defines the structure for errors related to API queries.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| response | Resp | The response object containing error data |

### `Resp`
#### Description
Extends the Response interface to include a data property for error information.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| data | { [key: string]: string } | An object containing error messages keyed by field names |

### `response`
#### Description
Extends the Response interface to include a data property of type ErrorResponse.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| data | ErrorResponse | The error response data |

### `ErrorResponseMessage`
#### Description
Extends the Response interface to include a response property of type response.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| response | response | The response object containing error data |
