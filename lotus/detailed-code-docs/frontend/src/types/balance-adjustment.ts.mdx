## High-level description
This code defines TypeScript interfaces for creating and managing credits (balance adjustments) in a financial system. It includes types for creating credits, managing drawdowns, and representing credit information.

## Symbols

### `CreateCreditType`
#### Description
An interface that defines the structure for creating a new credit.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| customer_id | string | Unique identifier for the customer |
| amount | number | The amount of credit |
| description | string (optional) | Description of the credit |
| currency_code | string | Code of the currency used |
| effective_at | string (optional) | Date when the credit becomes effective |
| expires_at | string (optional) | Expiration date of the credit |
| amount_paid | number (optional) | Amount paid for the credit |
| amount_paid_currency_code | string (optional) | Currency code for the amount paid |

### `DrawdownType`
#### Description
An interface that represents a drawdown on a credit.

#### Inputs
| Name | Type | Description |
| credit_id | string | Unique identifier for the credit |
| amount | number | Amount of the drawdown |
| applied_at | string | Date when the drawdown was applied |
| description | string | Description of the drawdown |

### `CreditType`
#### Description
An interface that represents a credit with its full details.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| amount | number | Total amount of the credit |
| amount_paid_currency | CurrencyType | Currency of the amount paid |
| amount_paid | number | Amount paid for the credit |
| amount_remaining | number | Remaining balance of the credit |
| description | string | Description of the credit |
| effective_at | string | Date when the credit becomes effective |
| expires_at | string | Expiration date of the credit |
| credit_id | string | Unique identifier for the credit |
| currency | CurrencyType | Currency of the credit |
| customer | LightweightCustomerType | Customer associated with the credit |
| status | "active" \| "inactive" | Current status of the credit |
| drawdowns | DrawdownType[] | Array of drawdowns on this credit |

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| LightweightCustomerType | Represents a simplified customer type |
| CurrencyType | Represents currency information |

This code provides a structured way to handle credits and balance adjustments in a financial system, allowing for creation, tracking, and management of credits for customers.
