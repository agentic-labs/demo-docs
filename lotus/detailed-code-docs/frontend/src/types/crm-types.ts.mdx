## High-level description
This code defines TypeScript interfaces and types for CRM (Customer Relationship Management) related data structures. It includes types for CRM providers, connection status, settings, and parameters used in CRM integrations within the application.

## Symbols

### `CRMProviders`
#### Description
A constant array containing the supported CRM provider names.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| CRMProviders | readonly ["salesforce"] | An array of supported CRM provider names |

### `CRMProviderType`
#### Description
A type representing the possible CRM provider names.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| CRMProviderType | "salesforce" | A string literal type for CRM provider names |

### `CRMConnectionStatus`
#### Description
An interface representing the connection status of a CRM provider.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| crm_provider_name | CRMProviderType | The name of the CRM provider |
| connected | boolean | Whether the CRM is connected |
| self_hosted | boolean | Whether the CRM is self-hosted |
| working | boolean | Whether the CRM integration is working |
| connection_id | string \| undefined | The connection ID, if available |
| account_id | string \| undefined | The account ID, if available |
| native_org_url | string \| undefined | The native organization URL, if available |

### `CRMSettingsParams`
#### Description
An interface for CRM settings parameters.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| setting_group | "crm" | The setting group, always "crm" |
| setting_name | string \| undefined | The optional setting name |

### `CRMSetting`
#### Description
An interface representing a CRM setting.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| setting_group | "crm" | The setting group, always "crm" |
| setting_id | string | The unique identifier for the setting |
| setting_name | string | The name of the setting |
| setting_values | any | The values associated with the setting |
