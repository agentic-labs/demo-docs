## High-level description
This TypeScript file defines interfaces for webhook endpoints and related operations. It provides structures for creating, updating, and managing webhook endpoints, including their triggers and associated data.

## Symbols

### `WebhookEndpoint`
#### Description
An interface representing a webhook endpoint with its properties.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| webhook_endpoint_id | string | Unique identifier for the webhook endpoint |
| name | string | Name of the webhook endpoint |
| webhook_url | URL | URL of the webhook endpoint |
| webhook_secret | string | Secret key for the webhook endpoint |
| triggers | WebhookTrigger[] | Array of triggers associated with the endpoint |

### `WebhookTrigger`
#### Description
An interface representing a webhook trigger.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| trigger_name | string | Name of the trigger |

### `WebhookEndpointCreate`
#### Description
An interface for creating a new webhook endpoint, omitting certain fields from `WebhookEndpoint`.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| name | string | Name of the webhook endpoint |
| webhook_url | URL | URL of the webhook endpoint |
| triggers_in | string[] | Array of trigger names |

### `WebhookEndpointUpdate`
#### Description
An interface for updating an existing webhook endpoint, omitting the `triggers` field from `WebhookEndpoint`.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| webhook_endpoint_id | string | Unique identifier for the webhook endpoint |
| name | string | Name of the webhook endpoint |
| webhook_url | URL | URL of the webhook endpoint |
| webhook_secret | string | Secret key for the webhook endpoint |
| triggers_in | string[] | Array of trigger names |

## References
This file is referenced in other parts of the codebase, particularly in the API handling (`frontend/src/api/api.ts`) and the Developer settings tab (`frontend/src/components/Settings/settings/tabs/DeveloperTab.tsx`).
