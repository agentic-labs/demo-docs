## High-level description
This directory contains documentation for the subscription lifecycle within the Lotus platform. It covers various aspects of managing subscriptions, including add-ons, alerts, subscription creation, invoicing, and managing entitlements.

## What does it do?
This documentation provides a comprehensive guide for understanding and utilizing the features offered by Lotus to manage the entire lifecycle of a subscription. It explains how to create and manage subscriptions, handle upgrades and downgrades, set up alerts for usage monitoring, configure invoicing and taxes, and control access to product features. This documentation serves as a valuable resource for anyone involved in managing subscriptions and billing within the Lotus platform.

## Dependencies
The documentation does not explicitly mention any external libraries or frameworks. However, it does reference integration with payment gateways and providers such as Stripe and PayPal (Braintree) for invoice payment collection.

## Configuration
The documentation mentions several configurable aspects of the Lotus platform:

1. Add-ons: Customizable pricing models, invoicing timing, and quantity attachments to plans.
2. Alerts: Configurable usage thresholds for triggering webhooks.
3. Subscriptions: Options for handling cancellations, plan switches, and proration settings.
4. Invoicing:
   - Default currency setting for the entire organization
   - Customer-specific currency settings
   - Tax configuration options:
     - Organization-level tax rate
     - Customer-specific tax rates
     - Integration with TaxJar for automatic sales tax calculation
   - Webhook URL configuration for invoice notifications
5. Entitlements: Configurable access controls for product features and components

The documentation suggests that many of these configurations can be set through the Lotus dashboard or via API calls, but specific configuration file formats or environment variables are not mentioned.
