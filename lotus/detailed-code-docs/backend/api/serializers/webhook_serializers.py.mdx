## High-level description
This file defines serializers for webhook events in a billing system. It includes serializers for various events such as customer creation, invoice creation, subscription creation, and usage alerts. These serializers are used to structure the data that will be sent in webhook payloads when specific events occur in the system.

This file defines the structure of the data that will be sent in webhook payloads for various events in the billing system. It uses serializers from Django REST framework to define the fields and their types for each event type. The serializers inherit from `serializers.Serializer` and define the fields that will be included in the webhook payload for each event type.
