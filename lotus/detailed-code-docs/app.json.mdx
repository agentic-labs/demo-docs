## High-level description
This `app.json` file is a configuration file for deploying the Lotus application on Heroku. It defines the application's metadata, required add-ons, buildpacks, environment variables, and post-deployment scripts for setting up the Lotus open-source metering, billing, and pricing platform.

## Symbols

### Root Object
#### Description
The root object in this JSON file contains various configuration settings for the Heroku deployment of the Lotus application.

#### Internal Logic
The file structure includes:
1. Basic application information (name, description, website, repository, logo)
2. Success URL after deployment
3. Required add-ons (PostgreSQL and Redis)
4. Buildpacks (Node.js and Python)
5. Environment variables
6. Post-deployment script (currently empty)
