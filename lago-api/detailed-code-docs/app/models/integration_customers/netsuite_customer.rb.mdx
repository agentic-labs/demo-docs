## High-level description
The `IntegrationCustomers::NetsuiteCustomer` class represents a customer in the context of a Netsuite integration. It inherits from `IntegrationCustomers::BaseCustomer` and adds a `subsidiary_id` setting.

## Code Structure
The `IntegrationCustomers::NetsuiteCustomer` class inherits from `IntegrationCustomers::BaseCustomer`, which provides common functionality for integration customers. It uses the `settings_accessors` method from the `SettingsStorable` concern to define accessors for the `subsidiary_id` setting.

## References

- `SettingsStorable`: Concern providing methods for storing and retrieving settings.
