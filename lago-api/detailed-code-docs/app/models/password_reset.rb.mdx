## High-level description
The `PasswordReset` model represents a request to reset a user's password. It handles the generation, storage, and validation of password reset tokens, ensuring their uniqueness and expiration.

## Code Structure
The `PasswordReset` model is associated with the `User` model through a `belongs_to` relationship. It validates the presence, uniqueness, and expiration of the generated reset tokens.

## References
- `User`: The `PasswordReset` model belongs to the `User` model, indicating that each password reset request is associated with a specific user.
