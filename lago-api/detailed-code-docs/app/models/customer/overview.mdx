## High-level description
The `Customer::AppliedTax` model represents a join table between `Customer` and `Tax` models, indicating which taxes are applied to a specific customer. It tracks the association between customers and the taxes that apply to them.

## Code Structure
The `Customer::AppliedTax` model is defined within the `Customer` class as a nested class. It inherits from `ApplicationRecord` and includes the `PaperTrailTraceable` module. It has `belongs_to` associations with both the `Customer` and `Tax` models.
