## High-level description
The `SlimHelper` class provides a utility method for rendering Slim templates in a Rails application. It defines a constant for a PDF logo filename and offers a class method to render Slim templates with a given context.

## Symbols

### `SlimHelper`
#### Description
This class serves as a helper for rendering Slim templates. It contains a constant for a PDF logo filename and a class method for rendering Slim templates.

#### Internal Logic
The class defines a constant `PDF_LOGO_FILENAME` and a class method `render`.

### `PDF_LOGO_FILENAME`
#### Description
A constant that stores the filename of the logo used in PDF generation.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| PDF_LOGO_FILENAME | String | The filename of the logo image used in PDF generation |

### `render`
#### Description
A class method that renders a Slim template file with the provided context.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| path | String | The path to the Slim template file, relative to `app/views/` |
| context | Object | The context object to be used when rendering the template |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| rendered_content | String | The rendered content of the Slim template |

#### Internal Logic
1. Constructs the full path to the Slim template file using `Rails.root.join`.
2. Reads the content of the template file with UTF-8 encoding.
3. Creates a new `Slim::Template` instance with the file content.
4. Renders the template with the provided context.

## Dependencies
| Dependency | Purpose |

| Slim | Used for rendering Slim templates |

## References
The `SlimHelper` class and its `PDF_LOGO_FILENAME` constant are referenced in the `Utils::PdfGenerator` class (from the related code snippet) for PDF generation purposes.

## Performance Considerations
The `render` method reads the template file from disk each time it's called. For frequently used templates, caching the file content or the rendered output might improve performance.
