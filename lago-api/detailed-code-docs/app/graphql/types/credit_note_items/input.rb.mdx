### `Types::CreditNoteItems::Input`
#### Description
This class defines the input object type for credit note items in GraphQL mutations. It specifies two required arguments: `amount_cents` and `fee_id`.

#### Inputs
This class is not a function and does not accept direct inputs. However, it defines arguments for GraphQL mutations:

| Name | Type | Description |
|:-----|:-----|:------------|
| amount_cents | `GraphQL::Types::BigInt` | The amount in cents for the credit note item. This field is required. |
| fee_id | `ID` | The ID of the fee associated with the credit note item. This field is required. |

#### Outputs
This class does not directly return any outputs. It is used as an input type for GraphQL mutations.
