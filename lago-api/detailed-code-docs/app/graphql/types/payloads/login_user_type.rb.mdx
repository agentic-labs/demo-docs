## High-level description
The `LoginUserType` is a GraphQL object type that represents the payload returned upon successful user login. It contains the user's authentication token and user details.

## Code Structure
The `LoginUserType` is a simple object type with two fields: `token` and `user`. It inherits from `Types::BaseObject`.

## References
- `Types::UserType`: Represents the user object type.
- `Types::BaseObject`: The base object type for all GraphQL objects.
