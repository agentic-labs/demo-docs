## High-level description
The code defines a GraphQL input type `UpdateCouponInput` used for updating existing coupons. It inherits from `CreateCouponInput` and adds a required `id` field.

## Code Structure
The `UpdateInput` class inherits from `Types::Coupons::CreateInput` and adds a new argument `id`. It is used as an input type for the `UpdateCoupon` mutation.

## References
- `Types::Coupons::CreateInput`: The base input type for creating coupons, from which `UpdateInput` inherits.
- `Mutations::Coupons::Update`: The GraphQL mutation that uses `UpdateInput` to receive update parameters.

## Symbols

### `UpdateInput`
#### Description
A GraphQL input type representing the data required to update an existing coupon. It extends the `CreateCouponInput` type by adding a required `id` field.

#### Inputs
Inherits all inputs from `Types::Coupons::CreateInput` and adds:

| Name | Type | Description |
|:-----|:-----|:------------|
| id | String | The ID of the coupon to update. |

#### Outputs
This is an input type, it doesn't have direct outputs. It's used as an argument for the `Mutations::Coupons::Update` mutation.
