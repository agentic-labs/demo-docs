## High-level description
The code defines a GraphQL input type named `CreateCouponInput` used for creating new coupons. It specifies the structure and data types for the input arguments required to create a coupon.

## References
- `Types::BaseInputObject`: This is the base class for the input type.
- `Types::CurrencyEnum`:  Enum type for accepted currencies.
- `Types::Coupons::CouponTypeEnum`: Enum type for coupon types.
- `Types::Coupons::FrequencyEnum`: Enum type for coupon frequencies.
- `Types::Coupons::LimitationInput`: Input type for coupon limitations.
- `Types::Coupons::ExpirationEnum`: Enum type for coupon expiration types.

## Symbols

### `CreateInput`
#### Description
A GraphQL input type defining the structure and data types for arguments required to create a new coupon.

#### Inputs
This is a GraphQL input type, it does not have explicitly defined input parameters. Instead, it defines a set of arguments:

| Name | Type | Description |
|:-----|:-----|:------------|
| amount_cents | GraphQL::Types::BigInt | The amount in cents for a fixed amount coupon. |
| amount_currency | Types::CurrencyEnum | The currency for a fixed amount coupon. |
| code | String | The coupon code. |
| coupon_type | Types::Coupons::CouponTypeEnum | The type of the coupon (e.g., 'fixed_amount', 'percentage'). |
| description | String | A description of the coupon. |
| frequency | Types::Coupons::FrequencyEnum | The frequency at which the coupon can be applied (e.g., 'once', 'recurring'). |
| frequency_duration | Integer | The duration for recurring coupons. |
| name | String | The name of the coupon. |
| percentage_rate | Float | The percentage rate for a percentage-based coupon. |
| reusable | Boolean | Whether the coupon is reusable or single-use. |
| applies_to | Types::Coupons::LimitationInput | Limitations on which plans or billable metrics the coupon applies to. |
| expiration | Types::Coupons::ExpirationEnum | The expiration type of the coupon (e.g., 'on_date', 'after_duration'). |
| expiration_at | GraphQL::Types::ISO8601DateTime | The specific expiration date and time for the coupon. |

#### Outputs
This is a GraphQL input type, it does not have any outputs. It is used as input for creating a new coupon.
