## High-level description
The code defines a GraphQL input type `CreateIntegrationMappingInput` used for creating new integration mappings. It specifies the required and optional arguments for the `createIntegrationMapping` mutation.

## References
- `Types::BaseInputObject`: This is a base class for GraphQL input objects.
- `Types::IntegrationMappings::MappableTypeEnum`: This enum defines the allowed types for the `mappable_type` argument.

## Symbols
### `CreateIntegrationMappingInput`
#### Description
This class defines the input type for the `createIntegrationMapping` mutation. It inherits from `Types::BaseInputObject` and defines the structure of the input arguments.

#### Inputs
None. This is an input type, not a function or method.

#### Outputs
None. This is an input type, not a function or method.

#### Internal Logic
The class defines six arguments:
- `external_account_code` (String, optional): An external account code.
- `external_id` (String, required): An external identifier.
- `external_name` (String, optional): An external name.
- `integration_id` (ID, required): The ID of the integration.
- `mappable_id` (ID, required): The ID of the mappable object.
- `mappable_type` (Types::IntegrationMappings::MappableTypeEnum, required): The type of the mappable object.
