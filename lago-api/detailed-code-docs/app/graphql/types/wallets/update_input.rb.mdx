## High-level description
The code defines a GraphQL input object type named `UpdateInput` within the `Types::Wallets` module. This input object is used to structure data for updating a Wallet object, specifying the fields that can be modified and their types.

### `UpdateInput`
#### Description
A GraphQL input object type defining the arguments for updating a Wallet.

| Name | Type | Description |
|:-----|:-----|:------------|

| `id` | `ID` | Required. The ID of the wallet to update. |
| `invoice_requires_successful_payment` | `Boolean` | Optional. Whether an invoice requires successful payment for the wallet. |

| `recurring_transaction_rules` | `[Types::Wallets::RecurringTransactionRules::UpdateInput]` | Optional. An array of recurring transaction rules to update for the wallet. |
