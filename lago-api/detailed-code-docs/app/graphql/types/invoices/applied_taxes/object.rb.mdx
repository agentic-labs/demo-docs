## High-level description
The `Types::Invoices::AppliedTaxes::Object` class defines a GraphQL type for representing an applied tax on an invoice. It includes fields for tax details, amount, and whether it was applied to the whole invoice.

## Code Structure
The `Object` class inherits from `Types::BaseObject` and implements the `Types::Taxes::AppliedTax` interface. It defines fields for an invoice applied tax, including a boolean indicating if the tax was applied to the whole invoice, the fees amount in cents, and the related invoice.

## References
- `Types::BaseObject`: Parent class for GraphQL object types.
- `Types::Taxes::AppliedTax`: Interface for applied tax types.
- `Types::Invoices::Object`: GraphQL type for invoices.
