## High-level description
This code defines a GraphQL object type for a PaymentRequest in a Ruby on Rails application. It specifies the structure and fields of a PaymentRequest that can be queried through the GraphQL API.

## Code Structure
The `Types::PaymentRequests::Object` class inherits from `Types::BaseObject` and defines various fields that represent the attributes of a PaymentRequest. It also establishes relationships with other object types such as Customer and Invoice.

## Symbols

### `Types::PaymentRequests::Object`
#### Description
This class defines the GraphQL object type for a PaymentRequest. It specifies the fields that can be queried for a PaymentRequest, including scalar values and relationships to other objects.

#### Fields
| Name | Type | Null | Description |
|:-----|:-----|:-----|:------------|
| id | ID | false | Unique identifier for the PaymentRequest |
| amount_cents | GraphQL::Types::BigInt | false | Amount in cents |
| amount_currency | Types::CurrencyEnum | false | Currency of the amount |
| created_at | GraphQL::Types::ISO8601DateTime | false | Creation timestamp |
| email | String | false | Email associated with the PaymentRequest |
| updated_at | GraphQL::Types::ISO8601DateTime | false | Last update timestamp |
