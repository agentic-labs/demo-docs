## High-level description
The `InvoicedUsageSerializer` class is responsible for serializing invoiced usage data into a JSON-compatible hash structure. It specifically formats the 'month', 'code', 'currency', and 'amount_cents' attributes from the provided model.

## Code Structure
The `InvoicedUsageSerializer` inherits from the `ModelSerializer` class. It overrides the `serialize` method to define a custom serialization logic for invoiced usage data.

## References
- `ModelSerializer`: The parent class providing basic serialization functionality.

## Symbols
### `InvoicedUsageSerializer`
#### Description
This class serializes an instance of an invoiced usage model into a hash containing 'month', 'code', 'currency', and 'amount_cents' attributes.

#### Inputs
Uses the `model` instance variable inherited from `ModelSerializer`, which represents the invoiced usage data to be serialized.

#### Outputs
Returns a hash containing the formatted invoiced usage data.

#### Internal Logic
The `serialize` method constructs a hash with the following keys:
- `month`: The 'month' value from the model.
- `code`: The 'code' value from the model.
- `currency`: The 'currency' value from the model.
- `amount_cents`: The 'amount_cents' value from the model.
