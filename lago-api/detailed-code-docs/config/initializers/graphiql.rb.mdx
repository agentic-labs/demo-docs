## High-level description
This code configures the GraphiQL Rails gem to enable the header editor feature in the development environment. It allows developers to modify request headers when using the GraphiQL interface for testing and exploring GraphQL queries.

## Symbols

### `GraphiQL::Rails.config.header_editor_enabled`
#### Description
This configuration option enables the header editor feature in the GraphiQL interface. When set to `true`, it allows users to modify request headers directly within the GraphiQL UI.
