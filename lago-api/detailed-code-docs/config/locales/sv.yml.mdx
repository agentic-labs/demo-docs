## High-level description
This YAML file contains Swedish (sv) localization data for date formats, day names, month names, and money formatting. It provides translations and formatting rules specific to the Swedish language and culture, which can be used in applications for internationalization and localization purposes.

## Symbols

### `sv`
#### Description
The root key for all Swedish localization data in this file.

### `date`
#### Description
Contains various date-related localization data for Swedish.

#### Internal Logic
- Provides abbreviated and full names for days of the week and months
- Defines a default date format
- Specifies the order of date components (year, month, day)

### `date.abbr_day_names`
#### Description
An array of abbreviated day names in Swedish, starting from Sunday.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| abbr_day_names | Array of Strings | Abbreviated day names: ["sön", "mån", "tis", "ons", "tor", "fre", "lör"] |

### `date.abbr_month_names`
#### Description
An array of abbreviated month names in Swedish, with an empty first element to align with 1-based month numbering.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| abbr_month_names | Array of Strings | Abbreviated month names, starting with an empty string: ["", "jan.", "feb.", "mars", "apr.", "maj", "juni", "juli", "aug.", "sep.", "okt.", "nov.", "dec."] |

### `date.day_names`
#### Description
An array of full day names in Swedish, starting from Sunday.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| day_names | Array of Strings | Full day names: ["söndag", "måndag", "tisdag", "onsdag", "torsdag", "fredag", "lördag"] |

### `date.formats.default`
#### Description
Defines the default date format string for Swedish localization.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| default | String | Default date format: "%d %b %Y" |

### `date.month_names`
#### Description

| Name | Type | Description |
|:-----|:-----|:------------|

### `date.order`
#### Description
Specifies the order of date components for Swedish date representation.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| order | Array of Symbols | Date component order: [:year, :month, :day] |
