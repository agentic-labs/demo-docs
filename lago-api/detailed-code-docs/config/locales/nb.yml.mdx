## High-level description
This YAML file contains localization data for the Norwegian Bokmål (nb) language. It defines translations and formatting rules for dates and money, which are essential for internationalizing applications to properly display dates and currency in Norwegian.

## Symbols

### `nb`
#### Description
The root key for all Norwegian Bokmål translations and configurations.

### `date`
#### Description
Contains all date-related translations and configurations for Norwegian Bokmål.

#### Internal Logic
- Defines abbreviated and full names for days of the week and months
- Specifies the default date format
- Sets the order for date components (year, month, day)

### `date.abbr_day_names`
#### Description
An array of abbreviated day names in Norwegian Bokmål, starting from Sunday.

### `date.abbr_month_names`
#### Description
An array of abbreviated month names in Norwegian Bokmål. The first element is empty, representing that months are typically numbered from 1 to 12.

### `date.day_names`
#### Description
An array of full day names in Norwegian Bokmål, starting from Sunday.

### `date.formats.default`
#### Description
Specifies the default date format for Norwegian Bokmål.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| default | string | The default date format string: "%d. %b %Y" |

### `date.month_names`
#### Description
An array of full month names in Norwegian Bokmål. The first element is empty, representing that months are typically numbered from 1 to 12.

### `date.order`
#### Description
Specifies the order of date components for Norwegian Bokmål.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| order | array | The order of date components: [:year, :month, :day] |

### `money`
#### Description
Contains configurations for formatting monetary values in Norwegian Bokmål.

#### Internal Logic
- Defines the decimal mark
- Specifies the format for displaying currency and amount
- Sets the thousands separator

### `money.decimal_mark`
#### Description
Specifies the character used as a decimal mark in monetary values for Norwegian Bokmål.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| decimal_mark | string | The decimal mark character: "," |

### `money.format`
#### Description
Defines the format string for displaying monetary values in Norwegian Bokmål.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| format | string | The format string: "%u%n" where %u is the currency symbol and %n is the number |

### `money.thousands_separator`
#### Description
Specifies the character used as a thousands separator in monetary values for Norwegian Bokmål.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| thousands_separator | string | The thousands separator character: " " (space) |

## Dependencies
This YAML file is typically used with internationalization (i18n) libraries in web applications, such as Rails I18n for Ruby on Rails applications.

## Configuration
The entire file serves as a configuration for Norwegian Bokmål localization. Applications using this file will refer to these keys to display properly formatted and translated date and money information.
