## High-level description
This YAML file contains German (de) localization data for date and money formatting. It defines translations for day and month names, date formats, and money-related symbols specific to the German language and culture.

## Symbols

### `de`
#### Description
The root key for all German localization data in this file.

### `date`
#### Description
Contains all date-related localization data for German.

#### Internal Logic
- Defines abbreviated and full names for days of the week and months
- Specifies the default date format
- Sets the order for date components (year, month, day)

### `date.abbr_day_names`
#### Description
An array of abbreviated day names in German, starting with Sunday.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| abbr_day_names | Array of Strings | Abbreviated day names: ["So", "Mo", "Di", "Mi", "Do", "Fr", "Sa"] |

### `date.abbr_month_names`
#### Description
An array of abbreviated month names in German, with an empty first element to align array indices with month numbers (1-12).

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| abbr_month_names | Array of Strings | Abbreviated month names, starting with an empty string: ["", "Jan.", "Feb.", "Mär.", ..., "Dez."] |

### `date.day_names`
#### Description
An array of full day names in German, starting with Sunday.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| day_names | Array of Strings | Full day names: ["Sonntag", "Montag", "Dienstag", "Mittwoch", "Donnerstag", "Freitag", "Samstag"] |

### `date.formats.default`
#### Description
Defines the default date format string for German localization.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| default | String | Default date format: "%d. %b %Y" (day, abbreviated month, year) |

### `date.month_names`
#### Description
An array of full month names in German, with an empty first element to align array indices with month numbers (1-12).

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| month_names | Array of Strings | Full month names, starting with an empty string: ["", "Januar", "Februar", "März", ..., "Dezember"] |

### `date.order`
#### Description
Specifies the order of date components for formatting in German.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| order | Array of Symbols | Date component order: [:year, :month, :day] |

### `money`
#### Description
Contains money-related formatting information for German localization.

#### Internal Logic
- Defines the decimal mark and thousands separator for currency amounts
- Specifies the format string for displaying currency amounts

### `money.decimal_mark`
#### Description
Defines the character used as a decimal mark in currency amounts.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| decimal_mark | String | Decimal mark character: "," |

### `money.format`
#### Description
Specifies the format string for displaying currency amounts.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| format | String | Currency format: "%u%n" (currency symbol followed by number) |

### `money.thousands_separator`
#### Description
Defines the character used as a thousands separator in currency amounts.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| thousands_separator | String | Thousands separator character: "." |
