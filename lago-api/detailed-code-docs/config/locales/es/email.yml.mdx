## High-level description
This YAML file contains Spanish (es) translations for email-related content in a billing or invoicing system. It includes translations for credit notes and invoices, covering various aspects such as subject lines, document details, and action prompts.

## Symbols

### `es:`
#### Description
This is the root key for Spanish translations in the YAML file.

### `email:`
#### Description
This key groups all email-related translations under the Spanish language.

### `credit_note:`
#### Description
Contains translations related to credit note emails.

#### Internal Logic
- Provides translations for various aspects of a credit note, including subject line, document details, and actions.
- Includes placeholders for dynamic content such as organization name, credit note number, and dates.

### `invoice:`
#### Description
Contains translations related to invoice emails.

#### Internal Logic
- Provides translations for various aspects of an invoice, including subject line, document details, and due date.
- Includes placeholders for dynamic content such as organization name, invoice number, and dates.

### `powered_by_lago:`
#### Description
Translation for a "Powered by" attribution, likely referring to the software or service provider.

### `questions:`
#### Description
Translation for a prompt encouraging users to contact support if they have questions.
