## High-level description
This YAML file contains French translations for email-related content in a billing or invoicing system. It includes translations for credit notes and invoices, covering various aspects such as subject lines, document details, and action prompts.

## Symbols

### `fr`
#### Description
The root key for French translations.

### `email`
#### Description
A nested key under `fr` that groups all email-related translations.

### `credit_note`
#### Description
Contains translations related to credit note emails.

#### Internal Logic
Provides translations for various aspects of a credit note, including the subject line, document details, and actions.

### `invoice`
#### Description
Contains translations related to invoice emails.

#### Internal Logic
Provides translations for various aspects of an invoice, including the subject line, document details, and actions.

### `powered_by`
#### Description
A translation for indicating the system that generated the email.

### `questions`
#### Description
A translation prompting the recipient to contact for any questions.

## Configuration
This YAML file serves as a configuration for French language translations in the application. It uses nested keys to organize translations for different email types and their components.

Note: The translations include placeholders (e.g., `%{organization_name}`, `%{date}`, `%{credit_note_number}`, `%{invoice_number}`) which are likely replaced with actual values when the emails are generated.
