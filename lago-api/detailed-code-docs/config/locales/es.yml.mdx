## High-level description
This YAML file contains Spanish language localization data for date and money formatting. It defines translations and formatting rules for days, months, and currency display in Spanish.

## Symbols

### `es`
#### Description
The root key for all Spanish localization data.

### `date`
#### Description
Contains all date-related localization information.

#### Internal Logic
- Defines abbreviated and full names for days and months
- Specifies date formatting rules
- Sets the order for date components (year, month, day)

### `date.abbr_day_names`
#### Description
List of abbreviated day names in Spanish, starting from Sunday.

### `date.abbr_month_names`
#### Description
List of abbreviated month names in Spanish. The first element is empty, representing that months are 1-indexed.

### `date.day_names`
#### Description
List of full day names in Spanish, starting from Sunday.

### `date.formats.default`
#### Description
Defines the default date format string for Spanish localization.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| format | string | "%-d de %b de %Y" (e.g., "1 de ene de 2023") |

### `date.month_names`
#### Description
List of full month names in Spanish. The first element is empty, representing that months are 1-indexed.

### `date.order`
#### Description
Specifies the order of date components for formatting.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| order | array | [:year, :month, :day] |

### `money`
#### Description
Contains currency formatting rules for Spanish localization.

#### Internal Logic
- Defines decimal mark and thousands separator
- Specifies the format for displaying currency amounts

### `money.decimal_mark`
#### Description
Specifies the character used as a decimal point in currency amounts.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| decimal_mark | string | "." (period) |

### `money.format`
#### Description
Defines the format string for displaying currency amounts.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| format | string | "%u%n" (currency symbol followed by the number) |

### `money.thousands_separator`
#### Description
Specifies the character used as a thousands separator in currency amounts.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| thousands_separator | string | "," (comma) |

## Dependencies
This YAML file is likely used in conjunction with a localization or internationalization library in a larger application, possibly a Ruby on Rails project, given the structure and naming conventions.
