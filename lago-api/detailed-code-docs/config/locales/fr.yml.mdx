## High-level description
This YAML file contains French language localization data for date and money formatting. It defines translations and formatting rules for days, months, and currency display in French.

## Symbols

### `fr`
#### Description
The root key for all French localization data.

### `date`
#### Description
Contains all date-related localization information.

#### Internal Logic
- Defines abbreviated and full names for days and months
- Specifies date formatting rules
- Sets the order for date components (year, month, day)

### `date.abbr_day_names`
#### Description
An array of abbreviated day names in French, starting with Sunday.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| abbr_day_names | Array of Strings | Abbreviated day names: ["Dim", "Lun", "Mar", "Mer", "Jeu", "Ven", "Sam"] |

### `date.abbr_month_names`
#### Description
An array of abbreviated month names in French, with an empty first element to align array indices with month numbers (1-12).

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| abbr_month_names | Array of Strings | Abbreviated month names, starting with an empty string: ["", "janv.", "févr.", "mars", "avr.", "mai", "juin", "juill.", "août", "sept.", "oct.", "nov.", "déc."] |

### `date.day_names`
#### Description
An array of full day names in French, starting with Sunday.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| day_names | Array of Strings | Full day names: ["Dimanche", "Lundi", "Mardi", "Mercredi", "Jeudi", "Vendredi", "Samedi"] |

### `date.formats`
#### Description
Defines the default date format for French localization.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| default | String | Default date format: "%-d %b %Y" |

### `date.month_names`
#### Description
An array of full month names in French, with an empty first element to align array indices with month numbers (1-12).

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| month_names | Array of Strings | Full month names, starting with an empty string: ["", "Janvier", "Février", "Mars", "Avril", "Mai", "Juin", "Juillet", "Août", "Septembre", "Octobre", "Novembre", "Décembre"] |

### `date.order`
#### Description
Specifies the order of date components for French date representation.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| order | Array of Symbols | Date component order: [:year, :month, :day] |

### `money`
#### Description
Contains currency formatting rules for French localization.

#### Internal Logic
- Defines the decimal mark, thousands separator, and overall format for currency display

### `money.decimal_mark`
#### Description
Specifies the character used as a decimal separator in currency amounts.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| decimal_mark | String | Decimal separator: "," |

### `money.format`
#### Description
Defines the format string for displaying currency amounts.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| format | String | Currency format: "%n %u" (number followed by unit) |

### `money.thousands_separator`
#### Description
Specifies the character used as a thousands separator in currency amounts.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| thousands_separator | String | Thousands separator: " " (space) |

## Configuration
This YAML file serves as a configuration for French language localization. It can be used by internationalization libraries or frameworks to properly format dates and currency amounts for French-speaking users.

| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| date.formats.default | String | "%-d %b %Y" | Default date format for French localization |
| money.format | String | "%n %u" | Currency amount format for French localization |
