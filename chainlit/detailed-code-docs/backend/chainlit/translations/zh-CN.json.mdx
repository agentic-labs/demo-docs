## High-level description
This file contains Chinese (Simplified) translations for the Chainlit application. It provides localized text for various components, hooks, and pages of the user interface, enabling a Chinese-language experience for users.

## Code Structure
The JSON structure is organized hierarchically, with main sections for "components", "hooks", and "pages". Each section contains nested objects representing different parts of the application, such as buttons, dialogs, and forms.
