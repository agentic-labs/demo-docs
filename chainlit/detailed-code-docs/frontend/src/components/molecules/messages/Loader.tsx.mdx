## High-level description
The `MessageLoader` component is a React functional component that renders a loading indicator for messages. It displays a linear progress bar within a styled box, which can be shown or hidden based on the `show` prop.

## Symbols

### MessageLoader
#### Description
This is the main and only component exported from this file. It renders a loading indicator for messages using Material-UI components.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| show | boolean | Optional prop to control the visibility of the loader |

#### Outputs
The component returns a JSX element representing the message loader.

#### Internal Logic
1. The component uses the `useLayoutMaxWidth` hook to determine the maximum width for the layout.
2. It renders a `Fade` component from Material-UI to handle the appearance/disappearance animation.
3. Inside the `Fade`, it renders a `Box` component with specific styling properties.
4. Within the `Box`, it renders a custom `BorderLinearProgress` component, which is a styled version of Material-UI's `LinearProgress`.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|

| hooks/useLayoutMaxWidth | Custom hook to determine the layout's maximum width |

### Configuration
The component uses the following style configurations:
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| maxWidth | string | (from useLayoutMaxWidth) | Sets the maximum width of the loader |
| width | string | '100%' | Sets the width of the loader container |
| borderRadius | string | '1rem' | Sets the border radius of the loader container |
| height | number | 3 | Sets the height of the loader container |
| color | string | 'text.primary' | Sets the color of the loader |
