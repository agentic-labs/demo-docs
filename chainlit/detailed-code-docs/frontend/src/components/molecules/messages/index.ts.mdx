## High-level description
This code exports the `MessageContainer` component from the `MessageContainer` module. It serves as an entry point for the message-related components in the application, making the `MessageContainer` component available for use in other parts of the frontend.

## Symbols

### `MessageContainer`
#### Description
The `MessageContainer` is exported from this file, but it is defined in the related `MessageContainer.tsx` file. It is a React component that wraps the `Messages` component with a `MessageContext.Provider`, providing a context for message-related data and functionality.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| actions | IAction[] | An array of action objects |
| context | IMessageContext | The message context object |
| elements | IMessageElement[] | An array of message element objects |
| messages | IStep[] | An array of message step objects |

#### Internal Logic
1. The component is wrapped with `memo` for performance optimization.
2. It sets up a `MessageContext.Provider` with the provided `context` or `defaultMessageContext`.
3. Inside the provider, it renders the `Messages` component with the given props.
