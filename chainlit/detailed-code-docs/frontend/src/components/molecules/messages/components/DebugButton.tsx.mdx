## High-level description
The `DebugButton` component is a React functional component that renders a button with a debug icon. When clicked, it opens a debug URL in a new tab, allowing users to debug a specific step in the Literal AI platform.

## Code Structure
The `DebugButton` component uses Material-UI components (`IconButton` and `Tooltip`) and a custom `DebugIcon`. It takes `debugUrl` and `step` as props to construct the debug URL.

## References
- `IStep` interface from 'client-types/'

- Material-UI components: `IconButton` and `Tooltip`

## Symbols

### DebugButton
#### Description
A functional component that renders a debug button with a tooltip. When clicked, it opens a debug URL in a new tab.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| debugUrl | string | The base URL for debugging |
| step | IStep | The step object containing information about the current step |

#### Internal Logic
1. Extracts the `stepId` from the `step` object, removing the 'wrap_' prefix if present.
2. Constructs the `href` by replacing placeholders in the `debugUrl` with actual values.
3. Renders an `IconButton` wrapped in a `Tooltip`.
4. The `IconButton` uses the custom `DebugIcon` and opens the constructed `href` in a new tab when clicked.
