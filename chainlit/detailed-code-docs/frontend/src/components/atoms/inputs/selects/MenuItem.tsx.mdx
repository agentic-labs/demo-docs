## High-level description
This code defines a custom `MenuItem` component for use in select inputs. It extends Material-UI's MenuItem component with additional styling and functionality, including support for icons, notification counts, and dark mode.

## Code Structure
The `MenuItem` component is the main export of this file. It uses several imported components and utilities to create a customized menu item for select inputs.

## Symbols

### `MenuItem`
#### Description
A custom menu item component for select inputs, extending Material-UI's MenuItem with additional styling and features.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| item | SelectItem | The item data to display |
| value | string \| number | The value of the item |
| selected | boolean | Whether the item is currently selected |
| isDarkMode | boolean | Whether the app is in dark mode |
| icon | JSX.Element (optional) | An optional icon to display |
| ...props | MMenuItemProps | Additional props passed to MMenuItem |

#### Outputs
Returns a JSX.Element representing the styled menu item.

#### Internal Logic
1. Applies custom styling based on the `selected` and `isDarkMode` props.
2. Renders the item's icon (if provided) and label.
3. Displays a notification count (if provided) using the `NotificationCount` component.
