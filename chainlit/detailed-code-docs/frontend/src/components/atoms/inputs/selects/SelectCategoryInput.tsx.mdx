## High-level description
The `SelectCategoryInput` component is a customized select input that allows users to choose from categorized options. It extends the functionality of the `SelectInput` component by grouping items into categories with headers, providing a more organized and hierarchical selection interface.

## Code Structure
The main component `SelectCategoryInput` uses the `SelectInput` component as its base, enhancing it with category-specific rendering logic. It also utilizes the `MenuItem` component for individual items and the `useIsDarkMode` hook for theme-aware styling.

- `useIsDarkMode` hook from 'hooks/useIsDarkMode'

- `SelectInput`, `SelectInputProps`, and `SelectItem` from './SelectInput'

## Symbols

### SelectCategoryInput
#### Description
A React component that renders a select input with categorized options. It takes an array of category items and renders them as grouped options within the select input.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| items | CategoryItem[] | An array of category items to be displayed in the select input |
| value | string \| number | The currently selected value |
| ...rest | Omit&lt;SelectInputProps, 'items'&gt; | Other props to be passed to the underlying SelectInput component |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| JSX.Element | React component | The rendered select input component with categorized options |

#### Internal Logic
1. Uses the `useIsDarkMode` hook to determine the current theme mode.
2. Defines `renderCategoryItem` function to render each category with its header and items.
3. Implements `renderLabel` function to find and return the label of the currently selected item.
4. Renders the `SelectInput` component with the categorized items and custom rendering logic.
