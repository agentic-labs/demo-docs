## High-level description
The `InputLabel` component is a reusable React component that renders a label for input fields. It provides a flexible structure with optional features such as tooltips and notification counts, enhancing the user interface for form inputs.

## Code Structure
The `InputLabel` component is defined as a functional component that takes props of type `InputLabelProps`. It uses various Material-UI components to create a structured label with optional elements.

## References
This component references:
- Material-UI components: `Box`, `MInputLabel`, `Tooltip`, `Info`
- Custom component: `NotificationCount`
- Custom type: `NotificationCountProps`

## Symbols

### `InputLabel`
#### Description
A functional component that renders a label for input fields with optional tooltip and notification count.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string (optional) | HTML id for the label |
| label | string \| number | The text content of the label |
| tooltip | string (optional) | Tooltip text to display on hover |
| notificationsProps | NotificationCountProps (optional) | Props for the NotificationCount component |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| JSX.Element | React component | The rendered InputLabel component |

#### Internal Logic
1. Renders a `Box` component as the main container.
2. Inside the main container, renders another `Box` for the label and optional tooltip.
3. Renders the `MInputLabel` component with custom styling.
4. If a tooltip is provided, renders a `Tooltip` component with an `Info` icon.
5. If `notificationsProps` are provided, renders the `NotificationCount` component.

## Performance Considerations
The component uses conditional rendering for optional elements (tooltip and notification count), which helps to minimize unnecessary DOM elements when these features are not needed.
