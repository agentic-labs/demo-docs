## High-level description
The `ClipboardCopy` component is a reusable React component that provides a button for copying text to the clipboard. It displays a tooltip to indicate the copy action and its success, enhancing user experience in a web application.

## Code Structure
The main `ClipboardCopy` component uses React hooks and Material-UI components to create a copy-to-clipboard functionality with visual feedback.

## Symbols

### ClipboardCopy
#### Description
A functional React component that renders an icon button with copy-to-clipboard functionality and a tooltip for user feedback.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| value | string | The text to be copied to the clipboard |
| edge | IconButtonProps['edge'] | Optional. Defines the edge position of the icon button |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| JSX.Element | React component | Returns a Tooltip-wrapped IconButton component |

#### Internal Logic
1. Uses `useState` to manage the `isCopied` state.
2. Utilizes the `useCopyToClipboard` hook from 'usehooks-ts' for clipboard operations.
3. Defines `handleCopy` function to copy text and update state.
4. Defines `handleTooltipClose` function to reset the copied state.
5. Renders a Tooltip component containing an IconButton with a CopyIcon.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| react | Core React library |
| usehooks-ts | Provides the useCopyToClipboard hook |
| @mui/material | Material-UI components (IconButton, Tooltip) |
| assets/copy | Custom CopyIcon component |

## Error Handling
The component includes basic error handling in the `handleCopy` function, logging any errors that occur during the copy operation to the console.
