## High-level description
The `NotificationCountProps` type definition in `frontend/src/types/NotificationCount.tsx` defines the structure for props used in a notification count component. It allows for displaying a count value and optionally includes input properties for interactive count adjustment.

## Symbols

### `NotificationCountProps`
#### Description
This type defines the structure of props for a notification count component. It includes an optional count value and an optional object for input properties.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| count | number \| string \| undefined | The notification count to be displayed |
| inputProps | object \| undefined | Properties for an input field to adjust the count |

#### Internal Logic
The `inputProps` object, when provided, contains properties for an input field:
- `id`: A required string identifier for the input field
- `max`: An optional maximum value for the input
- `min`: An optional minimum value for the input
- `onChange`: A required function to handle changes in the input value
- `step`: An optional step value for incrementing/decrementing the input
