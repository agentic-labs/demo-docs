## High-level description
This file defines and exports four SVG icon components for thumbs up and thumbs down actions, both in outlined and filled versions. These icons are designed to be used with Material-UI's SvgIcon component, providing a consistent look and feel for feedback mechanisms in a React application.

## Notes
1. All icons are designed with a viewBox of "0 0 24 24", which is standard for Material-UI icons.
2. The outlined icons (ThumbDownIcon and ThumbUpIcon) use stroke="currentColor" to allow for color customization through CSS.
3. The filled icons (ThumbDownFilledIcon and ThumbUpFilledIcon) use fill="currentColor" for the same purpose.
4. All icons are exported as named exports, allowing for selective imports in other parts of the application.
5. The SVG paths are carefully crafted to create the thumb shapes, ensuring a consistent and recognizable design across the application.
6. These icons are likely used in conjunction with the FeedbackButtons component found in the related code snippets, providing visual representation for user feedback actions.
