## High-level description
This code defines a theme configuration for a React application using Material-UI (MUI). It exports functions to create dark and light themes, as well as specialized grey button themes. The themes include customizations for typography, components, shapes, and color palettes.

## Code Structure
The code defines several constants for typography, components, shape, and colors. It then uses these to create theme functions (`darkTheme`, `lightTheme`, `makeTheme`) and specialized button themes (`darkGreyButtonTheme`, `lightGreyButtonTheme`).

## References
This code references color palettes from `./palette`, and uses MUI's `createTheme` function.

## Symbols

### `typography`
#### Description
Defines the font family for the theme.

#### Internal Logic
Sets the font family to 'Inter' with a fallback to 'sans-serif'.

### `components`
#### Description
Customizes default props and styles for various MUI components.

#### Internal Logic
Modifies default properties for MuiButton, MuiLink, MuiFormHelperText, and MuiTooltip components.

### `shape`
#### Description
Defines the default border radius for the theme.

### `success` and `error`
#### Description
Define color schemes for success and error states.

### `darkTheme(fontFamily?: string, breakpoints?: BreakpointsOptions)`
#### Description
Creates a dark theme configuration.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| fontFamily | string (optional) | Custom font family |
| breakpoints | BreakpointsOptions (optional) | Custom breakpoints |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| theme | Theme | MUI theme object |

#### Internal Logic
Uses `createTheme` to generate a dark theme with customized typography, components, shape, breakpoints, and color palette.

### `lightTheme(fontFamily?: string, breakpoints?: BreakpointsOptions)`
