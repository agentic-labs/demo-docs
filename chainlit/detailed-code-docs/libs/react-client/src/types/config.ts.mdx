## High-level description
This file defines TypeScript interfaces for configuration-related types used in the Chainlit React client. It includes interfaces for starters, chat profiles, audio configuration, authentication configuration, and the main Chainlit configuration.

## Symbols

### `IStarter`
#### Description
Defines the structure for a starter, which is likely used to provide pre-defined conversation starters or quick actions in the chat interface.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| label | string | The display label for the starter |
| message | string | The message content associated with the starter |
| icon | string (optional) | An optional icon for the starter |

### `ChatProfile`
#### Description
Defines the structure for a chat profile, which likely represents different chat modes or personas available in the application.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| default | boolean | Indicates if this is the default profile |
| icon | string (optional) | An optional icon for the profile |
| name | string | The name of the profile |
| markdown_description | string | A markdown-formatted description of the profile |
| starters | IStarter[] (optional) | An optional array of starters associated with this profile |

### `IAudioConfig`
#### Description
Defines the configuration for audio-related features in the chat application.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| enabled | boolean | Indicates if audio features are enabled |
| min_decibels | number | The minimum decibel level for audio detection |
| initial_silence_timeout | number | The timeout for initial silence in milliseconds |
| silence_timeout | number | The timeout for silence during recording in milliseconds |
| chunk_duration | number | The duration of each audio chunk in milliseconds |
| max_duration | number | The maximum duration of audio recording in milliseconds |

### `IAuthConfig`
#### Description
Defines the configuration for authentication in the application.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| requireLogin | boolean | Indicates if login is required |
| passwordAuth | boolean | Indicates if password authentication is enabled |
| headerAuth | boolean | Indicates if header-based authentication is enabled |
| oauthProviders | string[] | An array of OAuth providers available for authentication |

### `IChainlitConfig`
#### Description
Defines the main configuration structure for the Chainlit application, including UI settings, features, and other options.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| markdown | string (optional) | Optional markdown content |
| ui | object | UI-related configuration |
| features | object | Feature-related configuration |
| debugUrl | string (optional) | Optional debug URL |
| userEnv | string[] | User environment variables |
| dataPersistence | boolean | Indicates if data persistence is enabled |
| threadResumable | boolean | Indicates if threads are resumable |
| chatProfiles | ChatProfile[] | An array of available chat profiles |
| starters | IStarter[] (optional) | Optional global starters |
| translation | object | Translation-related configuration |
