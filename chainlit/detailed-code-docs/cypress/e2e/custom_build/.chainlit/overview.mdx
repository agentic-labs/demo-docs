## High-level description
This file is a TOML configuration file for a Chainlit project. It defines various settings and features for a chatbot application, including project settings, user interface customization, and feature toggles.

## Symbols

### [project]
#### Description
Defines general project settings for the Chainlit application.

#### Internal Logic
- Enables telemetry by default
- Sets session timeout to 3600 seconds (1 hour)
- Disables caching
- Allows all origins for CORS
- Does not follow symlinks for asset mounting

### [features]
#### Description
Configures various features of the Chainlit application.

#### Internal Logic
- Enables prompt playground
- Disables unsafe HTML rendering
- Disables LaTeX rendering
- Configures multi-modal file upload settings
- Disables speech-to-text feature

### [UI]
#### Description
Defines user interface settings for the Chainlit application.

#### Internal Logic
- Sets the chatbot name to "Chatbot"
- Enables showing readme as default when the thread is empty
- Configures content collapse and message expansion settings
- Specifies a custom build directory for the frontend

### [UI.theme]
#### Description
Defines theme settings for the user interface.

#### Internal Logic
- Provides commented-out examples for customizing light and dark themes
- Allows overriding default MUI light and dark themes

### [meta]
#### Description
Contains metadata about the configuration file.

#### Internal Logic
- Specifies the version of the tool that generated this configuration file

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| enable_telemetry | boolean | true | Enables or disables telemetry |
| session_timeout | integer | 3600 | Duration in seconds for saving the session when connection is lost |
| cache | boolean | false | Enables or disables third-party caching |
| allow_origins | array | ["*"] | Authorized origins for CORS |
| prompt_playground | boolean | true | Enables or disables the prompt playground feature |
| unsafe_allow_html | boolean | false | Enables or disables processing and displaying HTML in messages |
| latex | boolean | false | Enables or disables processing and displaying mathematical expressions |
| multi_modal.enabled | boolean | true | Enables or disables file upload feature |
| speech_to_text.enabled | boolean | false | Enables or disables speech-to-text feature |
| name | string | "Chatbot" | Name of the app and chatbot |
| show_readme_as_default | boolean | true | Shows readme while the thread is empty |
| default_collapse_content | boolean | true | Collapses large size content by default |
| default_expand_messages | boolean | false | Expands messages by default |
| hide_cot | boolean | false | Hides chain of thought details from the user |
| custom_build | string | "./public/build" | Specifies a custom build directory for the frontend |

Note: Some configuration options are commented out in the file and can be uncommented and customized as needed.
