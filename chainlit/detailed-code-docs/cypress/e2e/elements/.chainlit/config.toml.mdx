## High-level description
This file is a TOML configuration file for a Chainlit project. It defines various settings and features for a chatbot application, including project-wide configurations, UI customizations, and feature toggles.

## Symbols

### [project]
#### Description
Defines project-wide settings for the Chainlit application.

#### Internal Logic
- Enables telemetry by default
- Sets session timeout to 3600 seconds (1 hour)
- Disables caching
- Allows all origins for CORS
- No user environment variables are required

### [features]
#### Description
Configures various features of the Chainlit application.

#### Internal Logic
- Enables prompt playground
- Disables HTML rendering in messages for security
- Disables LaTeX rendering
- Configures multi-modal file upload feature
- Disables speech-to-text feature

### [UI]
#### Description
Defines user interface settings for the Chainlit application.

#### Internal Logic
- Sets the chatbot name to "Chatbot"
- Enables showing readme as default when the thread is empty
- Configures default content collapse and message expansion settings
- Allows customization of theme colors for both light and dark modes

### [meta]
#### Description
Provides metadata about the configuration file.

#### Internal Logic
- Specifies the version of Chainlit that generated this configuration file (1.0.400)

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| enable_telemetry | boolean | true | Enables or disables telemetry |
| session_timeout | integer | 3600 | Duration in seconds for saving the session when connection is lost |
| cache | boolean | false | Enables or disables third-party caching |
| allow_origins | array | ["*"] | Authorized origins for CORS |
| prompt_playground | boolean | true | Enables or disables the prompt playground feature |
| unsafe_allow_html | boolean | false | Enables or disables HTML rendering in messages |
| latex | boolean | false | Enables or disables LaTeX rendering |
| multi_modal.enabled | boolean | true | Enables or disables multi-modal file upload feature |
| speech_to_text.enabled | boolean | false | Enables or disables speech-to-text feature |
| name | string | "Chatbot" | Name of the app and chatbot |
| show_readme_as_default | boolean | true | Shows readme while the thread is empty |
| default_collapse_content | boolean | true | Collapses large size content by default |
| default_expand_messages | boolean | false | Expands messages by default |
| hide_cot | boolean | false | Hides chain of thought details from the user |

## Notes
- The configuration file includes commented-out options for customizing CSS, JavaScript, and fonts.
- Theme customization options are provided for both light and dark modes, but are commented out in this file.
- The file allows for fine-grained control over various aspects of the Chainlit application, enabling developers to tailor the chatbot experience to their specific needs.
