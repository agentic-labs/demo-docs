## High-level description
This file is a TOML configuration file for a Chainlit project. It defines various settings for the project, including user environment variables, session management, UI customization, and feature toggles.

## Symbols

### [project]
#### Description
This section contains general project settings.

#### Internal Logic
- Enables telemetry by default
- Specifies a required user environment variable "TEST_KEY"
- Sets session timeout to 3600 seconds (1 hour)
- Disables caching for third-party services

### [features]
#### Description
This section toggles specific features of the application.

#### Internal Logic
- Enables the prompt playground feature

### [UI]
#### Description
This section customizes the user interface of the application.

#### Internal Logic
- Sets the chatbot name to "Chatbot"
- Configures default content collapse and message expansion settings
- Allows for theme customization (both light and dark themes)

### [meta]
#### Description
This section contains metadata about the configuration file.

#### Internal Logic
- Specifies the version of the tool that generated this configuration file

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| enable_telemetry | boolean | true | Enables or disables telemetry |
| user_env | array | ["TEST_KEY"] | List of required user environment variables |
| session_timeout | integer | 3600 | Duration in seconds for session preservation |
| cache | boolean | false | Enables or disables third-party caching |
| prompt_playground | boolean | true | Toggles the prompt playground feature |
| name | string | "Chatbot" | Name of the app and chatbot |
| default_collapse_content | boolean | true | Default setting for collapsing large content |
| default_expand_messages | boolean | false | Default setting for expanding messages |
| hide_cot | boolean | false | Option to hide chain of thought details |

Note: The configuration file also includes commented-out options for additional customization, such as following symlinks, setting a description, adding a GitHub link, and customizing theme colors.
