## High-level description
This file is a TOML configuration file for a Chainlit project. It defines various settings and features for a chatbot application, including project-wide configurations, UI customizations, and feature toggles.

## Symbols

### [project]
#### Description
Defines project-wide settings for the Chainlit application.

#### Internal Logic
- Enables telemetry by default
- Sets session timeout to 3600 seconds (1 hour)
- Disables caching
- Allows all origins for CORS
- No user environment variables are required

### [features]
#### Description
Configures various features of the Chainlit application.

#### Internal Logic
- Enables prompt playground
- Disables HTML rendering in messages for security
- Disables LaTeX rendering
- Enables multi-modal file uploads with specific constraints
- Disables speech-to-text functionality

### [UI]
#### Description
Defines user interface settings for the Chainlit application.

#### Internal Logic
- Sets the chatbot name to "Chatbot"
- Enables showing readme as default when the thread is empty
- Collapses large content by default
- Disables expanding messages by default
- Allows customization of light and dark themes (currently commented out)

### [meta]
#### Description
Provides metadata about the configuration file.

#### Internal Logic
- Specifies the version of Chainlit that generated this configuration (1.0.400)

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| enable_telemetry | boolean | true | Enables or disables telemetry |
| session_timeout | integer | 3600 | Duration in seconds for saving the session when connection is lost |
| cache | boolean | false | Enables or disables third-party caching |
| allow_origins | array | ["*"] | Authorized origins for CORS |
| prompt_playground | boolean | true | Shows or hides the prompt playground |
| unsafe_allow_html | boolean | false | Enables or disables HTML rendering in messages |
| latex | boolean | false | Enables or disables LaTeX rendering |
| multi_modal.enabled | boolean | true | Enables or disables multi-modal file uploads |
| multi_modal.max_files | integer | 20 | Maximum number of files that can be uploaded |
| multi_modal.max_size_mb | integer | 500 | Maximum total size of uploaded files in MB |
| speech_to_text.enabled | boolean | false | Enables or disables speech-to-text functionality |
| name | string | "Chatbot" | Name of the app and chatbot |
| show_readme_as_default | boolean | true | Shows or hides the readme when the thread is empty |
| default_collapse_content | boolean | true | Collapses or expands large content by default |
| default_expand_messages | boolean | false | Expands or collapses messages by default |
| hide_cot | boolean | false | Hides or shows chain of thought details |

Note: Several configuration options are commented out in the file, indicating they are available for customization but not currently active.
