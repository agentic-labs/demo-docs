## High-level description
This file is a TOML configuration file for a Chainlit project. It defines various settings and features for a chatbot application, including project-wide configurations, UI customization options, and feature toggles.

## Symbols

### [project]
#### Description
This section contains general project settings.

#### Internal Logic
- Enables telemetry by default
- Sets session timeout to 3600 seconds (1 hour)
- Disables caching for third parties

### [features]
#### Description
This section configures various features of the chatbot application.

#### Internal Logic
- Enables prompt playground
- Disables HTML processing in messages for security reasons
- Disables LaTeX processing
- Configures multi-modal file upload settings
- Disables speech-to-text functionality

### [UI]
#### Description
This section defines UI-related settings for the chatbot application.

#### Internal Logic
- Sets the chatbot name to "Chatbot"
- Configures default behavior for content collapse and message expansion
- Allows for custom CSS file specification
- Provides placeholders for overriding default MUI light and dark themes

### [meta]
#### Description
This section contains metadata about the configuration file.

#### Internal Logic
- Specifies the version of the tool that generated this configuration file (1.0.0rc2)

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| enable_telemetry | boolean | true | Enables or disables telemetry |
| session_timeout | integer | 3600 | Duration in seconds for saving the session when connection is lost |
| cache | boolean | false | Enables or disables third-party caching |
| prompt_playground | boolean | true | Shows or hides the prompt playground feature |
| unsafe_allow_html | boolean | false | Enables or disables HTML processing in messages |
| latex | boolean | false | Enables or disables processing of mathematical expressions |
| multi_modal.enabled | boolean | true | Enables or disables file upload feature |
| multi_modal.max_files | integer | 20 | Maximum number of files that can be uploaded |
| multi_modal.max_size_mb | integer | 500 | Maximum total size of uploaded files in MB |
| speech_to_text.enabled | boolean | false | Enables or disables speech-to-text feature |
| name | string | "Chatbot" | Name of the app and chatbot |
| show_readme_as_default | boolean | false | Shows or hides the readme when the thread is empty |
| default_collapse_content | boolean | true | Collapses large content by default |
| default_expand_messages | boolean | false | Expands or collapses messages by default |
| hide_cot | boolean | false | Hides or shows chain of thought details |

Note: Some configuration options are commented out in the file and can be uncommented and customized as needed.
