## High-level description
The `AmazonReviewEmail` component defines the structure and content of an email template designed to solicit product reviews from Amazon customers. It leverages the `@react-email/components` library to construct the email's layout and elements.

## Code Structure
The `AmazonReviewEmail` component is a React functional component that accepts an object of props to customize its content. It utilizes various components from the `@react-email/components` library to structure the email layout, including `Container`, `Section`, `Row`, `Column`, `Text`, `Img`, and `Link`. The component dynamically constructs the email content based on the provided props and utilizes inline styles for visual presentation.

## Symbols

### `AmazonReviewEmail`
#### Description
This functional component renders an HTML email template mimicking an Amazon product review request. It uses various components from the `@react-email/components` library to structure the email's content and layout.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| titleText | string | (Optional) The main title text of the email. Defaults to "Your opinion is important!". |
| reviewText | string | (Optional) The body text prompting the user to review their purchase. Defaults to a generic message. |
| reviwStars | string[] | (Optional) An array of image URLs for the star rating system. Defaults to an array of 5 identical star images. |
| socialMediaIcons | string[] | (Optional) An array of image URLs for social media icons. Defaults to an array of Amazon's Instagram, Facebook, and Twitter icons. |

#### Outputs
React component representing the Amazon review email template.

#### Internal Logic
The component constructs the email template by composing elements from `@react-email/components`. It dynamically generates the star rating images based on the `reviwStars` prop and social media icons from the `socialMediaIcons` prop. The email content includes a header with the Amazon Prime logo, a section prompting for a product review with a sample product image, a call to action to review previous purchases, and a footer with legal information and social media links.
