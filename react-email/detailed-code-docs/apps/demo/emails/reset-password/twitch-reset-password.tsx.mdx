## High-level description
The code defines a React component, `TwitchResetPasswordEmail`, that renders an HTML email for Twitch users who have reset their passwords. It informs the user about the password update and prompts them to take action if they didn't initiate the change. The email also includes links for password security and support.

## Code Structure
The `TwitchResetPasswordEmail` component uses various components from the `@react-email/components` library to structure the email content. It fetches the base URL from environment variables and uses it to construct image URLs. The component formats the provided `updatedDate` and displays it within the email body.

2. **Construct Image URLs:** Uses the `baseUrl` from environment variables to create URLs for Twitch logo and social media icons.
3. **Render Email Content:** Uses components from `@react-email/components` to structure the email with a header, body content, and footer.
4. **Display User Information:** Displays the provided `username` and `formattedDate` within the email body.
5. **Provide Actionable Links:** Includes links for resetting password, learning about strong passwords, and contacting Twitch support.
