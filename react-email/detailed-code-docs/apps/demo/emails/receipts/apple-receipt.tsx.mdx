## High-level description
The `AppleReceiptEmail` component renders an HTML email receipt styled to resemble an Apple purchase receipt. It includes sections for order details, billing information, purchased items, a promotional call to action for Apple Card, and various legal disclaimers and links.

## Code Structure
The code consists of a single functional component `AppleReceiptEmail` that returns the JSX structure of the email. It uses various components from the `@react-email/components` library to construct the email layout and content. The styling is defined inline using JavaScript objects.

## Symbols

### `AppleReceiptEmail`
#### Description
This component renders an HTML email receipt for an Apple purchase. It fetches the base URL from the environment variable `VERCEL_URL` to construct image source URLs.

#### Inputs
This is a functional component, it does not take any inputs.

#### Outputs
Returns a JSX structure representing an HTML email receipt.

#### Internal Logic
The component renders a series of nested `Container`, `Section`, `Row`, and `Column` components to structure the email layout. It uses `Img` components to display images, `Text` components for textual content, and `Link` components for clickable links. The styling is applied inline using JavaScript objects.
