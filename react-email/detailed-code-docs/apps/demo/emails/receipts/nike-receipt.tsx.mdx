## High-level description
The `NikeReceiptEmail` component renders an HTML email receipt styled to resemble a Nike order confirmation. It includes sections for order tracking, shipping information, product details, recommendations, help menu, categories, and footer.

## Code Structure
The code defines a functional React component `NikeReceiptEmail` that returns the JSX structure of the email. It utilizes components from the `@react-email/components` library for building the email layout and content. The styling is defined in several constant objects, each representing a specific section or element of the email.

## Symbols

### `NikeReceiptEmail`
#### Description
This functional component renders the HTML structure of a Nike receipt email. It uses various components from the `@react-email/components` library to construct the email layout and content.

#### Inputs
This is a functional component, it does not take any inputs.

#### Outputs
Returns a JSX structure representing the HTML email receipt.

#### Internal Logic
The component renders a series of sections within a `Container` component. Each section represents a different part of the receipt, such as order tracking, shipping address, product details, order summary, recommendations, help menu, categories, and footer. The styling for each section and element is defined in separate constant objects.
