## High-level description
The `YelpRecentLoginEmail` component renders an email notification informing a Yelp user about a recent login to their account. It displays details about the login such as time, device, and location, and provides a link to the Yelp support page for further assistance.

## Code Structure
The `YelpRecentLoginEmail` component is a React functional component that accepts an object of `YelpRecentLoginEmailProps` as input. It uses the `@react-email/components` library to construct the email layout and content. The component dynamically constructs image URLs based on the environment and formats the login date for display.

## Symbols

### `YelpRecentLoginEmail`
#### Description
This functional component renders an email notification for a recent login to a Yelp account.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| userFirstName | string | The first name of the user. |
| loginDate | Date | The date and time of the login. |
| loginDevice | string | The device used for login. |
| loginLocation | string | The location of the login. |
| loginIp | string | The IP address used for login. |

#### Outputs
React component representing the Yelp recent login email.

#### Internal Logic
1. Formats the `loginDate` into a user-friendly string.
2. Constructs image URLs based on the `baseUrl` (determined from environment variables).
3. Renders an HTML email structure using components from `@react-email/components`.
4. Displays user information, login details, and a message about the recent login.
5. Includes a button linking to the Yelp support page.

#### Performance Considerations
No specific performance considerations are apparent in this code.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| @react-email/components | Provides React components for building email templates. |
| react | Core React library. |

## Configuration
The `baseUrl` is configured using the `VERCEL_URL` environment variable, defaulting to an empty string if not set. This is used to construct image URLs within the email.
