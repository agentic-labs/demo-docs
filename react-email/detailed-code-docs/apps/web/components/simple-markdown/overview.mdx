## High-level description
This code defines a simple React component that renders a sample email template using Markdown. The template includes a heading, a paragraph, and another heading. The component is then wrapped in a layout component for basic styling.

## Code Structure
The code defines two symbols: `component` and a default exported arrow function. The `component` is a React element that renders the Markdown content using the `Markdown` component from the `@react-email/components` library. The default exported function returns the `component` wrapped in a `Layout` component.
