## High-level description
The `CodeBlock` component displays a code snippet with syntax highlighting. It uses the `prism-react-renderer` library to highlight code in different languages and applies a custom theme for styling. The component also adds a gradient overlay to the top-right and bottom-left corners of the code block.

## References
- `Highlight` from `prism-react-renderer`: Used for syntax highlighting.

## Symbols

### `CodeBlock`
#### Description
A React functional component that renders a code block with syntax highlighting and a custom theme.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| children | string | The code snippet to be displayed. |
| className | string (optional) | Additional CSS classes to apply to the code block. |
| language | Language (optional) | The programming language of the code snippet. Defaults to "html". |

#### Outputs
React component that renders a code block with syntax highlighting.

#### Internal Logic
1. **Trims** leading and trailing whitespace from the `children` prop.
2. **Uses `Highlight` component** from `prism-react-renderer` to highlight the code snippet based on the provided `language`.
3. **Applies a custom theme** to the highlighted code for styling.
4. **Renders the highlighted code** within a `&lt;pre&gt;` tag for proper formatting.
5. **Adds two gradient overlays** to the top-right and bottom-left corners of the code block for visual styling.
6. **Handles a special case for bash code snippets** with a single line, adding a "$" prefix.
7. **Identifies and styles a specific code pattern** where "from" is followed by ":" on the next line, applying the "key-white" style.
