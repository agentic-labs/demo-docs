## High-level description
This code exports a React component called `Tooltip` that provides a wrapper around the `@radix-ui/react-tooltip` library. It simplifies the usage of tooltips by exposing the necessary components and props.

## Code Structure
The code imports components from `@radix-ui/react-tooltip` and re-exports them as part of the `Tooltip` object. This structure allows users to access all the necessary tooltip components through a single import.

## References
- `@radix-ui/react-tooltip`: This library provides the underlying tooltip functionality.
- `TooltipContent`: This component, defined in a separate file, is used to render the content of the tooltip.

## Symbols

### `TooltipRoot`
#### Description
This functional component acts as the root element for the tooltip. It takes in `TooltipProps` and renders a `TooltipPrimitive.Root` component with the provided props.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| children | React.ReactNode | The children of the tooltip root element. |
| ...props | TooltipProps | Additional props to be passed to the underlying `TooltipPrimitive.Root` component. |

#### Outputs
ReactNode: The rendered `TooltipPrimitive.Root` component with the provided children and props.

### `Tooltip`
#### Description
This object is the main export of the module and provides a simplified interface for using the tooltip components. It is created by assigning the `TooltipRoot` component and other tooltip-related components from `@radix-ui/react-tooltip` to a new object.

#### Properties
- `Arrow`: The `TooltipPrimitive.TooltipArrow` component.
- `Provider`: The `TooltipPrimitive.TooltipProvider` component.
- `Content`: The `TooltipContent` component.
- `Trigger`: The `TooltipPrimitive.TooltipTrigger` component.
