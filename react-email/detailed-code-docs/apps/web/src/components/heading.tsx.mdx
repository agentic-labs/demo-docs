## High-level description
The `Heading` component is a customizable heading component that allows users to specify the heading level, size, color, and weight. It renders a semantic HTML heading element (`h1` to `h6`) with appropriate CSS classes for styling.

## Code Structure
The `Heading` component is a React functional component that uses the `forwardRef` API to allow access to the underlying HTML heading element. It utilizes several helper functions (`getSizesClassNames`, `getColorClassNames`, `getWeightClassNames`) to generate the appropriate CSS class names based on the provided props.

## References
- `classNames`: A utility function from the `classnames` library for conditionally joining CSS class names.
- `As`: A type utility from `../utils/as` for creating a union type of component props with different `as` prop values.
- `unreachable`: A utility function from `../utils/unreachable` for throwing an error in unreachable code paths.
