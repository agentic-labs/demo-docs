## High-level description
The `Topbar` component renders the top application bar, including the logo and navigation menu. It adapts its layout for different screen sizes, using a drawer for mobile views.

## Code Structure
The `Topbar` component uses `Logo` and `Menu` components to render the logo and navigation menu respectively. It leverages the `classnames` library for conditional CSS class application.

## Symbols
### `Topbar`
#### Description
This functional component renders the top application bar. It displays the application logo on the left and the navigation menu on the right. For smaller screens, the navigation menu is collapsed into a drawer menu.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| className | `string` | Optional. Additional CSS class names to apply to the header element. |
| ...props | `React.ComponentPropsWithoutRef&lt;"header"&gt;` | Any other props passed to the component will be spread to the underlying `header` element. |

#### Outputs
React component representing the top application bar.

#### Internal Logic
1. **Conditional CSS Classes:** Uses `classNames` to apply CSS classes based on screen size and provided `className` prop.
2. **Logo:** Renders the `Logo` component wrapped in a `Link` component that points to the homepage ("/").
3. **Menu:** Renders the `Menu` component.
