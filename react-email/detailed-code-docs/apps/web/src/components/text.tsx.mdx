## High-level description
The `Text` component is a reusable React component that renders text elements with customizable size and color. It allows rendering as different HTML tags like `span`, `div`, or `p` based on the `as` prop.

## Code Structure
The `Text` component uses several helper functions: `getSizesClassNames` and `getColorClassNames` to determine the appropriate CSS classes based on the provided props. It leverages the `classNames` library for dynamic class name generation.

## References
- `classNames`: External library for conditionally joining classNames.
- `As`: Type utility for specifying the rendered HTML tag.
- `unreachable`: Utility function to throw an error in unreachable code branches.

## Symbols

### `Text`
#### Description
Renders a text element with customizable size and color. It uses the `forwardRef` API to allow access to the underlying DOM element.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| as | "span" \| "div" \| "p" | The HTML tag to render the text element as. Defaults to "span". |
| size | TextSize | The size of the text. Defaults to "2". |
| color | TextColor | The color of the text. Defaults to "gray". |
| className | string | Additional CSS class names to apply to the text element. |
| children | React.ReactNode | The content of the text element. |
| ...props | any | Any other props to pass to the underlying HTML tag. |

#### Outputs
React component rendering a text element.

#### Internal Logic
1. Determines the appropriate CSS class names for the given `size` and `color` using the `getSizesClassNames` and `getColorClassNames` functions.
2. Uses the `classNames` library to merge the generated class names with any provided `className`.
3. Renders the text element with the specified `Tag`, class names, and other props.

### `getSizesClassNames`
#### Description
Maps the `size` prop to the corresponding CSS class names.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| size | TextSize | The size of the text. |

#### Outputs
`string`: The CSS class name(s) corresponding to the given size.

#### Internal Logic
Uses a switch statement to map each `size` value to its respective CSS class name. Throws an error if an invalid size is provided.

### `getColorClassNames`
#### Description
Maps the `color` prop to the corresponding CSS class names.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| color | TextColor | The color of the text. |

#### Outputs
`string`: The CSS class name corresponding to the given color.

#### Internal Logic
Uses a switch statement to map each `color` value to its respective CSS class name. Throws an error if an invalid color is provided.
