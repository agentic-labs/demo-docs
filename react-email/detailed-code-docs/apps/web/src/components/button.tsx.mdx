## High-level description
The code defines a React component called `Button` that renders a button element with customizable appearance and size. It leverages Radix UI's `Slot` component for rendering children as a button or any other element.

## Code Structure
The `Button` component uses two helper functions, `getAppearance` and `getSize`, to determine the appropriate CSS classes based on the provided props. These functions use a switch statement to map the appearance and size props to specific class names.

## Symbols

### `Button`
#### Description
A React component that renders a customizable button element. It allows for customization of appearance and size, and can render children as a button or any other element using Radix UI's `Slot` component.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| asChild | boolean | Optional. If true, renders children as a `Slot` component. Defaults to false. |
| appearance | Appearance | Optional. Determines the button's appearance. Can be "white" or "gradient". Defaults to "white". |
| size | Size | Optional. Determines the button's size. Can be "1", "2", "3", or "4". Defaults to "2". |
| ...props | React.ComponentPropsWithoutRef&lt;"button"&gt; | Any other props passed to the button element. |

#### Outputs
JSX.Element representing the rendered button.

#### Internal Logic
1. Determines the component to render based on the `asChild` prop. If true, renders a `Slot` component, otherwise renders a `button` element.
2. Constructs the className string by combining the base button styles with the classes returned by `getAppearance` and `getSize` functions based on the provided props.
3. Renders the chosen component with the calculated className, provided props, and children.

### `getAppearance`
#### Description
A helper function that maps the `appearance` prop to corresponding CSS classes.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| appearance | Appearance | The desired appearance of the button. |

#### Outputs
An array of CSS class names representing the chosen appearance.

#### Internal Logic
Uses a switch statement to determine the appropriate CSS classes based on the `appearance` input. If an invalid appearance is provided, it throws an error using the `unreachable` function.

### `getSize`
#### Description
A helper function that maps the `size` prop to corresponding CSS classes.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| size | Size | The desired size of the button. |

#### Outputs
An array of CSS class names representing the chosen size.

#### Internal Logic
Uses a switch statement to determine the appropriate CSS classes based on the `size` input. If an invalid size is provided, it throws an error using the `unreachable` function.
