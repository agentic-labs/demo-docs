## High-level description
This code defines an asynchronous function `Sitemap` that generates a sitemap for the `react.email` website. The sitemap includes the home page, components page, and examples page.

## Symbols

### `Sitemap`
#### Description
An asynchronous function that generates a sitemap in the form of an array of objects. Each object represents a page on the website and includes the page URL and last modified date.

#### Inputs
This function takes no inputs.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| (return value) | `Promise&lt;Array&lt;{ url: string, lastModified: string }&gt;&gt;` | A promise that resolves to an array of objects. Each object represents a page in the sitemap and includes the page URL (`url`) and the last modified date (`lastModified`) in the format "YYYY-MM-DD". |

#### Internal Logic
1. **Define Routes:** An array of route paths (`""`, `"/components"`, `"/examples"`) is used to generate the sitemap entries.
2. **Map Routes to Sitemap Objects:** The `map` function iterates over the `routes` array and creates a sitemap object for each route. Each object includes:
    - `url`: The full URL of the page, constructed by concatenating "https://react.email" with the route path.
    - `lastModified`: The current date in the format "YYYY-MM-DD", obtained by splitting the ISO string representation of the current date at the "T".
3. **Return Sitemap:** The function returns the generated array of sitemap objects.
