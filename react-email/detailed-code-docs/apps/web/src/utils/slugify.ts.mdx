## High-level description
The `slugify` function transforms a given string into a URL-friendly slug. It accomplishes this by converting the string to lowercase, removing special characters, and replacing spaces with hyphens.

## Symbols

### `slugify`
#### Description
This function takes a string as input and returns a URL-friendly slug. It performs the following operations:
1. Converts the input string to lowercase.
2. Removes leading and trailing whitespace.
3. Replaces consecutive spaces with a single hyphen.
4. Removes characters that are not letters, numbers, hyphens, or underscores.
5. Replaces consecutive hyphens with a single hyphen.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| text | string | The input string to be slugified. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| (return value) | string | The slugified string. |

#### Internal Logic
The function uses a chain of string methods to achieve the desired slug format:
- `toLowerCase()`: Converts the string to lowercase.
- `trim()`: Removes leading and trailing whitespace.
- `replace(/\s+/g, "-")`: Replaces consecutive spaces with a single hyphen.
- `replace(/[^\w-]+/g, "")`: Removes characters that are not letters, numbers, hyphens, or underscores.
- `replace(/--+/g, "-")`: Replaces consecutive hyphens with a single hyphen.
