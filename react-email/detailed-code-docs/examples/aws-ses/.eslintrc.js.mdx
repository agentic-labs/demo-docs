#### Internal Logic
The configuration object contains two properties:
1. `root: true`: This tells ESLint that this is the root configuration file and it should stop looking for other configuration files in parent directories.
2. `extends: ['custom']`: This extends the ESLint configuration from a custom configuration named 'custom'.
