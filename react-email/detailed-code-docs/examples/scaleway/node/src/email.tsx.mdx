## High-level description
This code defines a React component for rendering an email template. It's designed to create a standardized email layout for inviting users to join a team on Scaleway, a cloud computing platform.

## Code Structure
The main component `Email` is defined as a functional component that takes props to customize the email content. It uses various components from the `@react-email/components` library to structure the email layout.

## Symbols

### `Email`
#### Description
A React functional component that renders an email template for team invitations on Scaleway.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| invitedByUsername | string | The username of the person sending the invitation |
| username | string | The username of the person being invited |
| teamName | string | The name of the team the user is being invited to |
| invitedByEmail | string | The email address of the person sending the invitation |

#### Internal Logic
1. Defines a preview text for the email.
2. Renders an HTML structure for the email using components from `@react-email/components`.
3. Includes a Scaleway logo image.
4. Displays a heading with the team name and Scaleway.
5. Provides a personalized greeting and invitation message.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| react | Core React library |
| @react-email/components | Provides pre-built components for email templates |

## Notes
- The `url` prop is defined in the `EmailProps` interface but not used in the component.
- The component uses TypeScript with React, as indicated by the use of `FC` type and prop interface.
- The email layout is responsive and follows a standard structure for email templates.

This component provides a reusable and customizable email template for Scaleway team invitations, ensuring consistent branding and formatting across invitation emails.
