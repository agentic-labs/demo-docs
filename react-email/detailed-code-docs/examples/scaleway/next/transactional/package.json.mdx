## High-level description
This `package.json` file defines the configuration and dependencies for a project named "emails". It's set up to work with React Email, a framework for building and testing email templates using React components.

## Symbols

### Project Metadata
#### Description
The top-level fields in the `package.json` file provide metadata about the project.

| Name | Value | Description |
|:-----|:------|:------------|
| name | "emails" | The name of the project |
| version | "0.0.0" | The current version of the project |
| private | true | Indicates that this project is private and not intended to be published to npm |

### Scripts
#### Description
The `scripts` section defines npm scripts that can be run using `npm run &lt;script-name&gt;`.

| Name | Command | Description |
|:-----|:--------|:------------|
| dev | "email dev" | Starts the development server for React Email |
| export | "email export" | Exports the email templates |

### Dependencies
#### Description
The `dependencies` section lists the project's runtime dependencies.

| Dependency | Version | Description |

| Dependency | Purpose |

This `package.json` file sets up a basic configuration for a project using React Email. It includes scripts for development and exporting email templates, as well as the necessary dependencies to work with React Email components and the framework itself.
