## High-level description
This code sets up a client for the Scaleway API and initializes a Transactional Email API instance. It's designed to facilitate sending transactional emails using Scaleway's services within a Node.js environment.

## Code Structure
The code defines two main symbols: `client` and `scalewayTEM`. The `client` is used to create the `scalewayTEM` instance, which is then exported for use in other parts of the application.

## Symbols

### `client`
#### Description
Creates a Scaleway API client using environment variables for authentication and configuration.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| accessKey | string | Scaleway access key from environment variable |
| secretKey | string | Scaleway secret key from environment variable |
| defaultProjectId | string | Scaleway project ID from environment variable |
| defaultRegion | string | Default region set to "fr-par" |
| defaultZone | string | Default zone set to "fr-par-1" |

#### Internal Logic
The `createClient` function from the Scaleway SDK is called with an object containing the configuration options. Environment variables are used for sensitive information.

### `scalewayTEM`
#### Description
Initializes a Transactional Email API instance using the created client.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| client | Client | The Scaleway API client instance |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| scalewayTEM | TransactionalEmail.v1alpha1.API | Transactional Email API instance |

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| @scaleway/sdk | Provides the Scaleway SDK for interacting with Scaleway services |

## Configuration
The code uses environment variables for configuration:
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| ACCESS_KEY | string | N/A | Scaleway access key |
| SECRET_KEY | string | N/A | Scaleway secret key |
| PROJECT_ID | string | N/A | Scaleway project ID |

## References
The `scalewayTEM` instance is used in the `send.ts` file to create and send emails using the Scaleway Transactional Email service.

## Side Effects
This code does not have any direct side effects, but it sets up a client that can be used to interact with Scaleway services, potentially leading to API calls and email sending operations when used.
