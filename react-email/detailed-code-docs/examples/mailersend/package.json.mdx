## High-level description
This `package.json` file defines the configuration for a React Email project with MailerSend integration. It specifies the project's dependencies, build scripts, and output formats for a TypeScript-based email component library.

|:------|:------|:------------|

| version | "1.0.0" | The current version of the package |
| license | "MIT" | The license under which the package is distributed |
| private | true | Indicates that this package is not intended to be published publicly |

This `package.json` file sets up a project that uses React Email components with MailerSend integration. It uses TypeScript for development and tsup for building the project. The output includes both CommonJS and ES module formats, as well as TypeScript declaration files. The project is set up as a private package, indicating it's not intended for publication to npm.
