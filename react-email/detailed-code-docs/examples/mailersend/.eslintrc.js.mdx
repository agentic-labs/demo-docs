## High-level description
This file is an ESLint configuration file for a project using MailerSend. It sets up the root configuration and extends a custom ESLint configuration.

## Symbols

### `module.exports`
#### Description
This object exports the ESLint configuration for the project.

#### Inputs
N/A

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| module.exports | Object | ESLint configuration object |

#### Internal Logic
The configuration object contains two properties:
1. `root`: Set to `true`, indicating that this is the root ESLint configuration file.
2. `extends`: An array containing a single string `'custom'`, which extends a custom ESLint configuration.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| custom | Custom ESLint configuration that this project extends |

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| root | boolean | true | Indicates that this is the root ESLint configuration file |
| extends | array | ['custom'] | Specifies the configurations to extend |

This ESLint configuration file is minimal and relies on a custom configuration. It sets the root configuration to true, which tells ESLint to stop looking for other configuration files in parent directories. The `extends` array includes a single item, 'custom', which means this project is using a custom ESLint configuration defined elsewhere.
