## High-level description
This code demonstrates how to send an email using the MailerSend API with React Email components. It creates an email with a clickable button, sets up the sender and recipient, and sends the email using the MailerSend client.

## Code Structure
The code imports necessary dependencies, creates a MailerSend client, renders an email component, sets up email parameters, and sends the email.

## References
The code references the `Email` component from "./email".

## Symbols

### `mailerSend`
#### Description
Creates a new instance of the MailerSend client using the API key from the environment variables.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| apiKey | string | The MailerSend API key |

### `emailHtml`
#### Description
Renders the `Email` component to HTML string format.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| url | string | The URL to be used in the email button |

### `sentFrom`
#### Description
Creates a new Sender object representing the email sender.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| email | string | The sender's email address |
| name | string | The sender's name |

### `recipients`
#### Description
Creates an array with a single Recipient object representing the email recipient.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| email | string | The recipient's email address |
| name | string | The recipient's name |

### `emailParams`
#### Description
Creates and configures an EmailParams object with the email details.

#### Internal Logic
1. Sets the sender using `setFrom`
2. Sets the recipients using `setTo`
3. Sets the email subject using `setSubject`
4. Sets the HTML content using `setHtml`

### `mailerSend.email.send(emailParams)`
#### Description
Sends the email using the configured MailerSend client and email parameters.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| emailParams | EmailParams | The configured email parameters |
