## High-level description
This package.json file defines the configuration for a React Email starter project. It specifies the project's dependencies, scripts, and other metadata necessary for setting up and running a React Email application.

## Symbols

### name
#### Description
Specifies the name of the project as "react-email-starter".

### version
#### Description
Indicates the current version of the project, which is "0.0.32".

### private
#### Description
A boolean flag set to true, indicating that this package is not intended to be published to npm.

### scripts
#### Description
Defines npm scripts for building, developing, and exporting the email templates.

#### Internal Logic
- `build`: Runs the email build process
- `dev`: Starts the development server
- `export`: Exports the email templates
