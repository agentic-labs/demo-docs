## High-level description
This configuration file for tsup defines the build settings for a TypeScript project. It specifies two separate build configurations: one for a Node.js environment and another for a browser environment, each with its own entry point and output directory.

## Code Structure
The configuration exports a default array of two configuration objects, each defining the build settings for different environments (Node.js and browser).

## Symbols

### `defineConfig`
#### Description
A function imported from the "tsup" package used to define the configuration for the TypeScript build process.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| configurations | Array&lt;Object&gt; | An array of configuration objects for different build targets |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| config | Object | A tsup configuration object |

### Configuration Objects
#### Description
Two configuration objects are defined, one for Node.js and one for browser environments. Each object specifies the build settings for its respective environment.

#### Properties
| Name | Type | Description |
|:-----|:-----|:------------|
| dts | boolean | Enables generation of declaration files (.d.ts) |
| entry | string[] | Specifies the entry point(s) for the build |
| outDir | string | Defines the output directory for the built files |
| format | string[] | Specifies the output format(s) for the built files |

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| tsup | Provides the `defineConfig` function for creating the build configuration |

This configuration file sets up two separate builds:

1. Node.js build:
   - Entry point: "./src/node/index.ts"
   - Output directory: "./dist/node"
   - Formats: CommonJS and ES Modules

2. Browser build:
   - Entry point: "./src/browser/index.ts"
   - Output directory: "./dist/browser"
   - Formats: CommonJS and ES Modules

Both configurations enable the generation of TypeScript declaration files (.d.ts) and produce outputs in both CommonJS and ES Module formats.
