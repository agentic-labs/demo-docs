## High-level description
This code defines a set of selector definitions used for converting HTML to plain text. It specifically targets certain HTML elements to be skipped or formatted in a particular way during the conversion process.

## Symbols

### `plainTextSelectors`
#### Description
`plainTextSelectors` is an array of `SelectorDefinition` objects that define how specific HTML elements should be handled when converting HTML to plain text.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| N/A | N/A | This is a constant array, not a function |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| plainTextSelectors | SelectorDefinition[] | An array of selector definitions |

#### Internal Logic
The array contains three selector definitions:
1. Skips `&lt;img&gt;` tags
2. Skips elements with the ID `__react-email-preview`
3. Formats `&lt;a&gt;` tags without link brackets

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| html-to-text | Provides the `SelectorDefinition` type used in the code |

## References
This module is imported and used in the following files:
- `packages/render/src/browser/render.ts`
- `packages/render/src/node/render-async.ts`
- Both `packages/render/src/node/index.ts` and `packages/render/src/browser/index.ts` export this module

In these files, `plainTextSelectors` is used as part of the options passed to the `convert` function from the `html-to-text` library when the `plainText` option is set to true. This allows for customized handling of specific HTML elements during the HTML to plain text conversion process.
