## Install

Install component from your command line.

#### With yarn

```sh
yarn add @react-email/section -E
```

#### With npm

```sh
npm install @react-email/section -E
```

## Getting started

Add the component to your email template. Include styles where needed.

```jsx
import { Section } from '@react-email/section';
import { Column } from '@react-email/column';
import { Row } from '@react-email/row';
import { Text } from '@react-email/text';

const Email = () =&gt; {
  return (
    {/* A simple `section` */}
    &lt;Section&gt;
      &lt;Text&gt;Hello World&lt;/Text&gt;
    &lt;/Section&gt;

    {/* Formatted with `rows` and `columns` */}
     &lt;Section&gt;
      &lt;Row&gt;
        &lt;Column&gt;Column 1, Row 1&lt;/Column&gt;
        &lt;Column&gt;Column 2, Row 1&lt;/Column&gt;
      &lt;/Row&gt;
      &lt;Row&gt;
        &lt;Column&gt;Column 1, Row 2&lt;/Column&gt;
        &lt;Column&gt;Column 2, Row 2&lt;/Column&gt;
      &lt;/Row&gt;
    &lt;/Section&gt;
  );
};
```
