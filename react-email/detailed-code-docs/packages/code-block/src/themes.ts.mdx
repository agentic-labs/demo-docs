## High-level description
This file defines various syntax highlighting themes for code blocks. It exports multiple theme objects, each representing a different color scheme and styling for code syntax highlighting. These themes can be used with a code highlighting component to apply consistent styling to code snippets.

## Code Structure
The file exports multiple constant objects, each representing a different syntax highlighting theme. Each theme object contains style definitions for various code elements like comments, keywords, strings, etc. The themes are defined using TypeScript's `as const` assertion to ensure type safety.

## Symbols

### `Theme` (type)
#### Description
Defines the structure of a theme object, mapping syntax token types to React CSS properties.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| string \| "base" \| keyof Prism.Grammar | React.CSSProperties | CSS properties for each token type |

### `xonokai`, `vscDarkPlus`, `duotoneForest`, etc. (constants)
#### Description
These are theme objects that define the styling for different syntax highlighting themes. Each theme object contains CSS properties for various syntax elements.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ThemeObject | Theme | An object containing style definitions for syntax highlighting |
