## High-level description
This code defines an interface `EmailTemplate` and a type guard function `isEmailTemplate` for email templates in a React-based email system. It provides a structure for email template components and a way to check if a value conforms to this structure.

## Symbols

### `EmailTemplate`
#### Description
`EmailTemplate` is an interface that defines the structure of an email template component in React.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| props | Record&lt;string, unknown&gt; \| Record&lt;string, never&gt; | The properties passed to the email template component |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| - | React.ReactNode | The rendered React component |

#### Internal Logic
The interface defines a function that takes props and returns a React node. It also includes an optional `PreviewProps` property of type `Record&lt;string, unknown&gt;`.

### `isEmailTemplate`
#### Description
`isEmailTemplate` is a type guard function that checks if a given value is of type `EmailTemplate`.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| val | unknown | The value to be checked |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| - | boolean | True if the value is an EmailTemplate, false otherwise |

#### Internal Logic
The function checks if the input `val` is of type 'function'. This is because an `EmailTemplate` is essentially a function that returns a React node.
