## High-level description
This code defines a `Topbar` component for a React Email application. It provides a header with controls for toggling the sidebar, displaying the current email, and switching between different views (desktop, mobile, and source code).

## Code Structure
The `Topbar` component is the main export. It uses various sub-components and utilities, including `ToggleGroup`, `motion`, `Tooltip`, and custom icons. The component is styled using Tailwind CSS classes and uses the `cn` utility for conditional class names.

## Symbols

### `Topbar`
#### Description
A React functional component that renders the top bar of the email preview interface.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| currentEmailOpenSlug | string | The slug of the currently open email |
| pathSeparator | string | The separator used in the email path |
| markup | string | (Optional) The HTML markup of the email |
| activeView | string | (Optional) The currently active view (desktop, mobile, or source) |
| setActiveView | function | (Optional) Function to set the active view |
| onToggleSidebar | function | (Optional) Function to toggle the sidebar |

#### Internal Logic
1. Renders a header with a button to toggle the sidebar (hidden on small screens)
2. Displays the current email name
3. Provides a toggle group for switching between desktop, mobile, and source code views
4. Conditionally renders a "Send" button if markup is provided

## Performance Considerations
The component uses `motion` from framer-motion for animations, which may have a slight performance impact. However, the animations are limited to small UI elements and should not significantly affect overall performance.

This component is part of a larger React Email application and is designed to be efficient for frequent re-renders as the user interacts with the email preview interface.
