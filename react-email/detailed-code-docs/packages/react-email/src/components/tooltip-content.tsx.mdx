## High-level description
This code defines a `TooltipContent` component using Radix UI's Tooltip primitive. It's a customized version of the tooltip content that applies specific styling and uses the Inter font. The component is designed to be used within a React application, likely as part of a larger tooltip system.

## Code Structure
The main symbol in this code is the `TooltipContent` component, which is a forwarded ref component built on top of Radix UI's `TooltipPrimitive.Content`. It uses the `cn` utility function for class name concatenation and imports the `inter` font configuration.

## References
- `@radix-ui/react-tooltip`: Used for the base tooltip functionality
- `../utils`: Imports the `cn` utility function
- `../app/inter`: Imports the `inter` font configuration

## Symbols

### TooltipContent
#### Description
A React component that renders the content of a tooltip. It wraps the Radix UI `TooltipPrimitive.Content` component with custom styling and configuration.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| sideOffset | number | The offset from the trigger (default: 6) |
| children | React.ReactNode | The content to be displayed in the tooltip |
| ...props | ContentProps | Additional props to be passed to the underlying Radix UI component |

#### Outputs
Returns a React component that renders the tooltip content.

#### Internal Logic
1. The component is wrapped in a `TooltipPrimitive.Portal` to ensure proper rendering in the DOM.
2. It applies custom className using the `cn` utility function, which includes:
   - Background color, text color, and border styles
   - Padding and border radius
   - Font family (using the Inter font)
3. The `sideOffset` prop is passed to control the positioning of the tooltip relative to its trigger.
4. The component forwards the ref to the underlying Radix UI component.

## API/Interface Reference
The `TooltipContent` component accepts all props that can be passed to `TooltipPrimitive.Content` from Radix UI, with the addition of:

| Prop | Type | Description |
|:-----|:-----|:------------|
| children | React.ReactNode | The content to be displayed in the tooltip |
| sideOffset | number | The offset from the trigger (default: 6) |

Note: This component is designed to be used in conjunction with other Radix UI Tooltip components, likely as part of a larger `Tooltip` component structure (as seen in the related `tooltip.tsx` file).
