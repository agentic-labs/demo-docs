## High-level description
This code defines a React component called `Heading` that renders a customizable heading element with various styling options. It utilizes the `@radix-ui/react-slot` library for flexible rendering and provides props for controlling the heading size, color, and weight.

## Code Structure
The main `Heading` component is defined using `React.forwardRef`. It uses utility functions (`getSizesClassNames`, `getColorClassNames`, `getWeightClassNames`) to generate appropriate CSS classes based on the provided props. The component renders a `SlotPrimitive.Slot` with the generated classes and wraps the content in the specified heading tag.

## Symbols

### Heading
#### Description
A customizable heading component that renders different heading levels (h1-h6) with various styling options.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| as | 'h1' \| 'h2' \| 'h3' \| 'h4' \| 'h5' \| 'h6' | Specifies the heading level |
| size | HeadingSize | Controls the font size |
| color | HeadingColor | Sets the text color |
| weight | HeadingWeight | Determines the font weight |
| className | string | Additional CSS classes |
| children | React.ReactNode | Content of the heading |

#### Outputs
Renders a heading element with the specified properties and styling.

#### Internal Logic
1. Applies default values for `as`, `size`, and `weight` if not provided.
2. Uses utility functions to generate CSS classes based on the provided props.
3. Renders a `SlotPrimitive.Slot` with the generated classes and wraps the content in the specified heading tag.

### getSizesClassNames
#### Description
