## High-level description

This file defines a reusable Button component for React Email. It provides a customizable button element with various styling options and the ability to render as a child component or a native button element.

## Code Structure

The main export is the `Button` component, which is a React forwardRef component. It uses utility functions `getAppearance` and `getSize` to determine the button's appearance and size based on props. The component can render either as a `SlotPrimitive.Slot` or a native `button` element depending on the `asChild` prop.

## Symbols

### Button
#### Description
A customizable button component for email templates.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| asChild | boolean | If true, renders as a child component |
| appearance | 'white' \| 'gradient' | Button appearance style |
| className | string | Additional CSS classes |
| children | React.ReactNode | Button content |
| size | '1' \| '2' \| '3' \| '4' | Button size |
| ...props | RootProps | Additional button props |

#### Internal Logic
1. Combines CSS classes based on size, appearance, and custom classes
2. Renders either a `SlotPrimitive.Slot` or a native `button` based on `asChild` prop
3. Applies combined CSS classes and forwards the ref

## Notes
- The component uses a switch statement to handle different appearance and size options, with a fallback to an `unreachable` function for type safety.
- The component is designed to be flexible, allowing for custom styling through the `className` prop and additional props passed to the underlying element.
