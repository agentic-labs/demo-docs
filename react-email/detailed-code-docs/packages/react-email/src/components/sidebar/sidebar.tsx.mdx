## High-level description
This code defines a Sidebar component for a React application. It renders a collapsible sidebar that displays a directory structure of email templates, allowing users to navigate through different email files and directories.

## Code Structure
The Sidebar component uses the `useEmails` hook to access email directory metadata. It renders a collapsible root containing a `SidebarDirectoryChildren` component, which recursively renders the directory structure.
