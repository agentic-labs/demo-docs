## High-level description
This code defines the home page component for a React Email application. It displays a welcome message, logo, and instructions for creating email templates, utilizing Next.js components and custom UI elements.

## Code Structure
The `Home` component is the main export, which renders the page content within a `Shell` component. It uses various UI components like `Button`, `Heading`, and `Text` to structure the content.

## Symbols

### Home
#### Description
A functional component that renders the home page of the React Email application.

#### Inputs
This component doesn't take any props.

#### Outputs
Returns a JSX element representing the home page.

#### Internal Logic
1. Retrieves the base directory name for email templates using `path.basename()`.
2. Renders a welcome message, logo, and instructions within a `Shell` component.
3. Uses Next.js `Image` component to display the logo with specific styling.
4. Provides a button linking to the documentation.
