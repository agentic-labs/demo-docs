## High-level description
This file configures the tsup bundler for the React Email CLI. It defines the build settings for compiling the CLI source code into both ESM and CommonJS formats, generating type definitions, and specifying the output directory.

## Symbols

### Default Export
#### Description
The default export is a configuration object for tsup, created using the `defineConfig` function. It specifies the build settings for the React Email CLI.

| config | Object | A tsup configuration object |

#### Internal Logic
The configuration object sets the following properties:
1. `dts: true` - Enables generation of TypeScript declaration files (.d.ts).
2. `entry: ['./src/cli/index.ts']` - Specifies the entry point for the CLI.
3. `format: ['esm', 'cjs']` - Sets the output formats to ECMAScript Modules and CommonJS.
4. `outDir: 'dist/cli'` - Defines the output directory for the compiled files.

## Dependencies
| Dependency | Purpose |

The file imports the `defineConfig` function from the 'tsup' package, which is used to create a type-safe configuration object for tsup.

## Performance Considerations
By generating both ESM and CJS formats, the package supports a wide range of JavaScript environments, enhancing compatibility and performance across different systems.

The use of TypeScript declaration files (enabled by `dts: true`) allows for better type checking and IDE support when using the compiled CLI in TypeScript projects.
