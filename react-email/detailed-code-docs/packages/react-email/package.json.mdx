## High-level description

This `package.json` file defines the configuration and dependencies for the `react-email` package, which is a tool for creating and previewing email templates using React components. It includes scripts for building, testing, and linting the project, as well as a comprehensive list of dependencies and development dependencies.

## Code Structure

The main symbols in this code are the various npm scripts defined in the `scripts` section and the dependencies listed in the `dependencies` and `devDependencies` sections. These are used to build, test, and run the react-email package.

## Symbols

### `scripts`

#### Description
Defines various npm scripts for building, testing, and maintaining the project.

#### Internal Logic
- `build`: Runs the build process using tsup-node and a custom build script
- `dev`: Runs tsup-node in watch mode for development
- `test`: Runs vitest for unit testing
- `test:watch`: Runs vitest in watch mode
- `clean`: Removes the dist directory
- `lint`: Runs eslint and TypeScript type checking

### `dependencies`

#### Description
Lists the production dependencies required for the package to function.

#### Internal Logic
Includes various libraries for parsing, building, and rendering React components, as well as utilities for CLI interactions and file system operations.

### `devDependencies`

#### Description
Lists the development dependencies used for building, testing, and maintaining the package.

#### Internal Logic
Includes libraries for UI components, testing, building, and code quality tools.
