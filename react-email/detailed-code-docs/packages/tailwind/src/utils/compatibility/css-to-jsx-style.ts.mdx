## High-level description
This code provides utility functions to convert CSS text to a JavaScript object representation of styles, which can be used in JSX. It includes functions to parse CSS declarations and convert CSS property names to their camelCase JavaScript equivalents.

## Code Structure
The main functions in this file are `getCssDeclarations` and `cssToJsxStyle`. These functions work together to parse CSS text and convert it into a format suitable for use in React components. The `convertPropertyName` and `camelCase` functions are helper functions used in this process.

## Symbols

### `camelCase`
#### Description
Converts a hyphenated string to camelCase.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| string | string | The hyphenated string to convert |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| (unnamed) | string | The camelCased version of the input string |

#### Internal Logic
Uses a regular expression to replace hyphens followed by a character with the uppercase version of that character.

### `convertPropertyName`
#### Description
Converts CSS property names to their JavaScript equivalents.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| prop | string | The CSS property name to convert |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| (unnamed) | string | The converted property name |

#### Internal Logic
1. Converts the property to lowercase.
2. Handles special cases like "float" and CSS custom properties.
3. Removes the "ms" prefix for Microsoft-specific properties.
4. Calls `camelCase` for the final conversion.

### `getCssDeclarations`
#### Description
Parses CSS text and extracts individual CSS declarations.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| cssText | string | The CSS text to parse |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| (unnamed) | Array&lt;{property: string, value: string}&gt; | An array of objects containing CSS property-value pairs |

#### Internal Logic
Uses a regular expression to match CSS declarations and extracts the property and value. It then cleans up the value by removing extra whitespace.

### `cssToJsxStyle`
#### Description
Converts CSS text to a JavaScript object suitable for use as inline styles in JSX.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| cssText | string | The CSS text to convert |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| style | Record&lt;string, string&gt; | An object with camelCased CSS properties as keys and their corresponding values |

#### Internal Logic
1. Calls `getCssDeclarations` to parse the CSS text.
2. Iterates through the declarations, converting each property name using `convertPropertyName`.
3. Adds each converted property-value pair to the `style` object.

This file does not have any external dependencies.

## Performance Considerations
The use of regular expressions for parsing CSS might not be the most efficient for very large CSS strings. However, for typical use cases, the performance should be adequate.

## Error Handling
This code does not implement explicit error handling. It assumes well-formed CSS input and may produce unexpected results with malformed input.
