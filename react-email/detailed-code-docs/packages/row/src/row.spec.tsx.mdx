## Symbols
### `&lt;Row&gt; component`
#### Description
A React component that renders an HTML table row (`&lt;tr&gt;`) wrapped in a full-width table structure (`&lt;table&gt;`). It accepts children and additional props for styling and other attributes.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| children | React.ReactNode | The content to be rendered within the table row. |
| style | React.CSSProperties | Inline styles to be applied to the table row. |
| ...props | React.TableHTMLAttributes&lt;HTMLTableElement&gt; | Any other HTML attributes to be applied to the table element. |

#### Outputs
Renders an HTML table row (`&lt;tr&gt;`) within a full-width table structure (`&lt;table&gt;`).

#### Internal Logic
The component uses `React.forwardRef` to allow access to the underlying `&lt;table&gt;` element. It renders a `&lt;table&gt;` element with specific attributes for email compatibility:
- `align="center"`: Centers the table horizontally.
- `width="100%"`: Makes the table span the full width of its container.
- `border="0"`: Removes the table border.
- `cellPadding="0"`: Removes cell padding.
- `cellSpacing="0"`: Removes cell spacing.
- `role="presentation"`: Indicates that the table is for presentational purposes.

It then renders a `&lt;tbody&gt;` with a single `&lt;tr&gt;` containing the provided `children`.

#### Performance Considerations
The component is lightweight and should not have any significant performance implications.
