## High-level description
This file is a TypeScript configuration file (`tsconfig.json`) specifically tailored for React library projects. It extends a base configuration and sets up compiler options optimized for React development with support for modern JavaScript features and testing with Vitest.

### `compilerOptions`
#### Description
Defines specific compiler options for the TypeScript compiler, tailored for React library development.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| jsx | string | Specifies the JSX factory to use |
| lib | string[] | Specifies the library files to include in the compilation |
| module | string | Specifies the module code generation method |
| target | string | Specifies the ECMAScript target version |
| types | string[] | Specifies type declaration files to be included in compilation |

#### Internal Logic
- Sets `jsx` to "react-jsx" for React JSX support without importing React.
- Includes "ESNext", "DOM", and "DOM.Iterable" in the `lib` array for modern JavaScript and DOM support.
- Sets `module` to "ESNext" for the latest ECMAScript module syntax.
- Sets `target` to "es6" for ECMAScript 2015 (ES6) compatibility.
- Includes "vitest/globals" in the `types` array for Vitest testing support.
