## High-level description
The `pxToPt` function converts a pixel value to a point value, which is a unit of measurement commonly used in typography and printing. This conversion is useful for maintaining consistent font sizing across different email clients, as some clients may render font sizes differently depending on the unit used.

## Symbols

### `pxToPt`
#### Description
This function converts a pixel value to a point value using the conversion factor of 3/4 (1px = 0.75pt). It first checks if the input is a valid number. If it is, it performs the conversion and returns the result. Otherwise, it returns `null`.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| px | number | The pixel value to convert. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
|  | number \| null | The converted point value, or `null` if the input is invalid. |

#### Internal Logic
1. Check if the input `px` is a number and if it's not NaN.
2. If both conditions are true, calculate the point value by multiplying `px` by 3/4 and return the result.
3. If either condition is false, return `null`.

## References
- This function is used in the `Button` component to convert padding values from pixels to points for consistent rendering in email clients.

## Error Handling
The function handles invalid input by returning `null`. This allows the calling code to handle the error gracefully, for example, by using a default value or displaying an error message.
