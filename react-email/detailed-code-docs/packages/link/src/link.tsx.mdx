## High-level description
The `Link` component is a custom implementation of an anchor tag (`&lt;a&gt;`) that provides default styling and behavior for links within the application. It sets the link color to blue, removes text decoration, and defaults to opening links in a new tab.

## Code Structure
The `Link` component is a React functional component that uses `React.forwardRef` to allow access to the underlying anchor element. It accepts all standard anchor tag props and applies default styles.

## Symbols

### `Link`
#### Description
This component renders an anchor tag (`&lt;a&gt;`) with default styling and behavior for links.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ...props | `LinkProps` | All standard anchor tag props are spread onto the rendered anchor element. |
| target | `string` | Specifies where to open the linked document. Defaults to `"_blank"` (new tab/window). |
| style | `React.CSSProperties` |  Inline styles to apply to the link. These styles are merged with the default styles. |

#### Outputs
Returns a styled anchor tag (`&lt;a&gt;`) element.

#### Internal Logic
1. The component uses `React.forwardRef` to obtain a ref to the underlying anchor element.
2. It renders an `&lt;a&gt;` tag, spreading all provided `props` onto it.
3. Default styles are applied, including `color: "#067df7"`, `textDecoration: "none"`, and any additional styles provided via the `style` prop.
4. The `target` prop is set to `"_blank"` by default, ensuring links open in a new tab/window.
5. The children of the `Link` component are rendered within the anchor tag.

#### Performance Considerations
This is a simple component with minimal performance overhead.
