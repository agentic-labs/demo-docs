## High-level description
This code defines a React component called `Html` that represents the HTML root element. It provides a reusable and customizable way to create the `&lt;html&gt;` tag in React applications, with default language and text direction settings.

## Symbols

### `HtmlProps`
#### Description
`HtmlProps` is a type alias that represents the props for the `Html` component. It extends React's built-in props for the `&lt;html&gt;` element, excluding the ref prop.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| N/A | React.ComponentPropsWithoutRef&lt;"html"&gt; | All props available for the HTML `&lt;html&gt;` element, excluding the ref prop |

### `Html`
#### Description
`Html` is a React component that renders an `&lt;html&gt;` element. It uses React's `forwardRef` to allow ref forwarding and provides default values for `lang` and `dir` attributes.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| children | React.ReactNode | Child elements to be rendered inside the `&lt;html&gt;` tag |
| lang | string | Language attribute for the `&lt;html&gt;` tag (default: "en") |
| dir | string | Text direction attribute for the `&lt;html&gt;` tag (default: "ltr") |
| ...props | HtmlProps | Any additional props to be spread onto the `&lt;html&gt;` element |

#### Outputs
Returns a React element representing the `&lt;html&gt;` tag with the specified props and children.

#### Internal Logic
1. The component destructures the input props, extracting `children`, `lang`, and `dir`.
2. It sets default values for `lang` ("en") and `dir` ("ltr") if not provided.
3. It renders an `&lt;html&gt;` element with the spread props, specified `dir` and `lang` attributes, and the forwarded ref.
4. The `children` are rendered inside the `&lt;html&gt;` element.

## Notes
- The `Html` component is given a `displayName` of "Html", which can be useful for debugging purposes.
- The component uses the `Readonly` type to ensure that the props object is immutable.
- The `index.ts` file in the same directory exports all contents from the `html.tsx` file, making the `Html` component and `HtmlProps` type available when importing from the package.
