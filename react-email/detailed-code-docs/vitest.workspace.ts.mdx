## High-level description
This file defines the Vitest workspace configuration. It exports an array containing a single glob pattern that specifies which directories should be included in the Vitest test suite.

## Symbols

### Default export
#### Description
The default export is an array containing a single string element: `"packages/*"`. This glob pattern indicates that Vitest should include all directories within the `packages` folder for testing.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| default export | Array&lt;string&gt; | An array containing a glob pattern for Vitest workspace configuration |

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|

## Configuration
This file serves as a configuration for Vitest, specifying which directories should be included in the test suite. The glob pattern `"packages/*"` indicates that all subdirectories within the `packages` directory should be considered part of the Vitest workspace.
