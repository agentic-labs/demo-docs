## High-level description
This script generates an HTML report from a JSON file. It combines a predefined HTML template with the JSON data to create a standalone HTML report file.

## Symbols

### `gen_report`
#### Description
This function generates an HTML report by combining a template with JSON data.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| json_output_path | str | Path to the JSON file containing the report data |

#### Internal Logic
1. Reads the JSON data from the specified file.
2. Locates and reads the HTML report template.
3. Combines the template and JSON data into a single HTML string.
4. Writes the combined HTML to a new file named "manually_generated_report.html".

### `main`
#### Description
The main entry point of the script that handles command-line arguments and calls `gen_report`.

#### Internal Logic
1. Checks if a command-line argument (JSON file path) is provided.
2. Raises a `ValueError` if no argument is given.
3. Calls `gen_report` with the provided JSON file path.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| pathlib | For file and path operations |
| sys | For accessing command-line arguments |

## Error Handling
The script implements basic error handling by raising a `ValueError` if the required command-line argument is missing.
