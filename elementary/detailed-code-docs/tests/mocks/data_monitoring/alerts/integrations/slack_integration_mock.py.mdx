## High-level description
This code defines a `SlackIntegrationMock` class that extends the `SlackIntegration` class. It is used for testing purposes to mock the behavior of the Slack integration in Elementary's data monitoring system. The class allows for customization of Slack-related configuration options and initializes a `Config` object with these options.

## Code Structure
The `SlackIntegrationMock` class inherits from `SlackIntegration` and overrides its `__init__` method. It creates a `Config` object with the provided Slack-related parameters and passes it to the parent class constructor.
