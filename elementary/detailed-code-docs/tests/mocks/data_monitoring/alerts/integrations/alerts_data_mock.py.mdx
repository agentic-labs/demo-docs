## High-level description
This file contains mock data for various types of alerts used in testing the Elementary data monitoring system. It defines several alert models and a grouped alerts class, which are used to simulate different scenarios in unit tests for the Elementary framework.

## Code Structure
The file defines several constants representing different types of alert models (DBT_TEST_ALERT_MOCK, ELEMENTARY_TEST_ALERT_MOCK, MODEL_ALERT_MOCK, SNAPSHOT_ALERT_MOCK, SOURCE_FRESHNESS_ALERT_MOCK) and a grouped alerts mock (GROUPED_NY_TABLE_ALERTS_MOCK). These are then used in the AlertsDataMock class to provide easy access to the mock data.

## Symbols

### `DBT_TEST_ALERT_MOCK`
#### Description
A mock instance of TestAlertModel representing a dbt test alert.

### `ELEMENTARY_TEST_ALERT_MOCK`
#### Description
A mock instance of TestAlertModel representing an Elementary test alert.

### `MODEL_ALERT_MOCK`
#### Description
A mock instance of ModelAlertModel representing a model alert.

### `SNAPSHOT_ALERT_MOCK`
#### Description
A mock instance of ModelAlertModel representing a snapshot alert.

### `SOURCE_FRESHNESS_ALERT_MOCK`
#### Description
A mock instance of SourceFreshnessAlertModel representing a source freshness alert.

### `GROUPED_NY_TABLE_ALERTS_MOCK`
#### Description
A mock instance of GroupedByTableAlerts containing a list of mock alerts.

### `AlertsDataMock`
#### Description
A class that provides easy access to all the mock alert data.

#### Inputs
This class doesn't take any inputs.

#### Internal Logic
The class initializes attributes for each type of mock alert and the grouped alerts mock.
