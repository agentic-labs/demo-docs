## High-level description
This YAML configuration file serves as a template for setting up monitoring and notification settings for dbt (data build tool) projects. It defines the destination for alerts and specifies the dbt projects to be monitored.

## Symbols

### `slack_notification_webhook`
#### Description
This configuration option specifies the Slack webhook URL where alerts and notifications will be sent.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| slack_notification_webhook | string | The Slack webhook URL for sending notifications |

### `dbt_projects`
#### Description
This configuration option defines a list of dbt project paths that should be monitored.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| dbt_projects | list of strings | Paths to the dbt projects to be monitored |

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| slack_notification_webhook | string | `&lt;your_slack_webhook_url&gt;` | The Slack webhook URL for sending notifications |
| dbt_projects | list of strings | `[&lt;your_dbt_project_path&gt;]` | Paths to the dbt projects to be monitored |
