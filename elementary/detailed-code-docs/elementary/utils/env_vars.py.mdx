## High-level description
This code defines a simple utility function `is_debug` to check if the application is running in debug mode. It achieves this by checking for the presence of a specific environment variable, "EDR_DEBUG".

## Symbols

### `is_debug`
#### Description
This function checks if the application is running in debug mode.

#### Inputs
This function takes no inputs.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| return value | bool | True if "EDR_DEBUG" environment variable is present, False otherwise. |

#### Internal Logic
1.  The function uses `os.environ` to access the environment variables.
2.  It checks if the key "EDR_DEBUG" exists in the environment variables.
3.  If the key exists, it returns `True`, indicating debug mode is on. Otherwise, it returns `False`.
