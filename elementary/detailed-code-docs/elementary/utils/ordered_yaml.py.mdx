## High-level description
The `OrderedYaml` class provides a wrapper around the `ruamel.yaml` library to load and dump YAML data while preserving the order of keys in dictionaries. It also sets specific indentation and quote preservation settings for consistent formatting.

## Code Structure
The `OrderedYaml` class encapsulates a `ruamel.yaml.YAML` object and provides methods for loading and dumping YAML data. The `load` and `dump` methods handle file I/O, while the `loads` method works directly with YAML strings.

#### Internal Logic

### `load`
#### Description
Loads YAML data from a file while preserving order.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| file_path | str | The path to the YAML file. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| data | dict | A dictionary representing the loaded YAML data. |

#### Internal Logic
- Opens the specified file in read mode with UTF-8 encoding.
- Uses the `ruamel.yaml.YAML` object to load the YAML data from the file object.
- Returns the loaded data as a dictionary.
