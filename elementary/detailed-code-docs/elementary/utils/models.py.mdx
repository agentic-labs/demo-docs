## High-level description
This code defines a utility function `get_shortened_model_name` that extracts the class name from a fully qualified Python class name represented as a string. 

## Symbols

### `get_shortened_model_name`
#### Description
This function takes a fully qualified Python class name as a string and returns the class name, which is the part after the last dot. If the input is None, it returns None.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| model | str | A fully qualified Python class name (e.g., "my_package.my_module.MyClass") or None. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
|  | str | The class name extracted from the input string (e.g., "MyClass") or None if the input was None. |

#### Internal Logic
1. Checks if the input `model` is None. If it is, returns None.
2. Splits the `model` string by ".".
3. Returns the last element of the split list, which represents the class name.
