## High-level description
This file is a dbt project configuration file (`dbt_project.yml`) for the "elementary_cli" project. It defines various settings and configurations for the dbt project, including project name, version, file paths, and custom configurations for the Elementary CLI tool.

## Symbols

### Project Configuration
#### Description
The file defines the main configuration for the dbt project named "elementary_cli".

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| name | string | The name of the dbt project |
| version | string | The version of the project |
| config-version | integer | The version of the configuration format |
| profile | string | The dbt profile to use for this project |

#### Internal Logic
- Sets the project name to "elementary_cli"
- Sets the project version to "1.0.0"
- Uses configuration version 2
- Specifies the "elementary" profile for this project

### File Paths
#### Description
Defines the directory structure for various components of the dbt project.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| model-paths | list | Directories containing dbt models |
| analysis-paths | list | Directories containing analysis files |
| test-paths | list | Directories containing test files |
| seed-paths | list | Directories containing seed data files |
| macro-paths | list | Directories containing macro files |
| snapshot-paths | list | Directories containing snapshot files |

#### Internal Logic
- Sets default paths for models, analyses, tests, seeds, macros, and snapshots
- Configures the packages installation path using an environment variable with a default fallback
- Defines the target path for compiled SQL files
- Specifies directories to be cleaned by `dbt clean` command

### Variables
#### Description
Defines custom variables for the project.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| edr_cli_run | boolean | Indicates if the project is running in Elementary CLI mode |

#### Internal Logic
Sets `edr_cli_run` to `true`, which likely affects the behavior of the Elementary CLI tool when running dbt commands.

### Quoting Configuration
#### Description
Configures the quoting behavior for database objects.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| database | string | Quoting configuration for database names |
| schema | string | Quoting configuration for schema names |
| identifier | string | Quoting configuration for identifier names |

#### Internal Logic
- Uses environment variables to determine the quoting behavior for database, schema, and identifier names
- Applies the `as_native` filter to convert the environment variable values to their native Python types
