## High-level description
This JSON file, `mint.json`, is a configuration file for a documentation website for Elementary, a data observability platform. It defines the structure, appearance, and content organization of the documentation site, including navigation, theming, integrations, and various features.

## Code Structure
The JSON object contains multiple top-level keys that configure different aspects of the documentation site. These include site metadata, visual styling, navigation structure, integrations, and analytics settings.

## Symbols

### `name`
#### Description
Specifies the name of the documentation site.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| name | string | The name of the site |

### `logo`
#### Description
Defines the logo images for light and dark modes.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| light | string | Path to the light mode logo |
| dark | string | Path to the dark mode logo |

### `modeToggle`
#### Description
Configures the default color mode for the site.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| default | string | The default color mode ("dark" in this case) |

### `colors`
#### Description
Defines the color scheme for the site, including primary colors, background colors, and anchor colors.

### `feedback`
#### Description
Configures feedback options for the documentation.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| suggestEdit | boolean | Enables/disables the suggest edit feature |
| raiseIssue | boolean | Enables/disables the raise issue feature |

### `search`
#### Description
Configures the search functionality.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| prompt | string | The placeholder text for the search input |

### `tabs`
#### Description
Defines the main navigation tabs for the site.

### `topbarCtaButton`
#### Description
Configures the call-to-action button in the top bar.

### `anchors`
#### Description
Defines additional navigation links or buttons.

### `navigation`
#### Description
Defines the main navigation structure of the documentation, organizing content into groups and pages.

### `footerSocials`
#### Description
Configures social media links in the footer.

### `analytics`
#### Description
Configures various analytics integrations (GA4, PostHog, GTM).

### `redirects`
#### Description
Defines URL redirects for the site.
