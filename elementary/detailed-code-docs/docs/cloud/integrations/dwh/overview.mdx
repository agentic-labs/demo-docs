## High-level description
This directory contains documentation for connecting Elementary Cloud to various data warehouses (DWH). Each file provides instructions for setting up a connection to a specific DWH, including any required configurations or prerequisites.

## What does it do?
This documentation guides users on how to connect their Elementary Cloud account to their data warehouse. This connection allows Elementary to synchronize its schema with the data warehouse, enabling users to leverage Elementary's data modeling and transformation capabilities on their DWH data. The documentation covers various DWH options, including Athena, BigQuery, ClickHouse, Databricks, Postgres, Redshift, Snowflake, and Trino. Each DWH has its own specific instructions and requirements for setting up the connection.
