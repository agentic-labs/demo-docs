## High-level description

This directory contains documentation for integrating Elementary with various Business Intelligence (BI) tools. Each file provides instructions and explanations for connecting a specific BI tool to Elementary, enabling automated lineage tracking to the dashboard level.

## What does it do?

This documentation guides users on how to connect their BI tools (like Tableau, Looker, Power BI) to Elementary. By establishing this connection, Elementary can automatically track how data flows from its origin all the way to the dashboards and visualizations created in the BI tool. This provides a complete picture of data lineage, allowing users to understand data dependencies, analyze the impact of data issues or changes, and identify unused datasets.

## Configuration

The integration process typically involves the following steps:

1. Creating necessary permissions and roles in the BI tool
2. Generating API keys or access tokens
3. Providing connection details in Elementary's environment settings

Specific configuration steps vary by BI tool and are detailed in each tool's respective documentation file.
