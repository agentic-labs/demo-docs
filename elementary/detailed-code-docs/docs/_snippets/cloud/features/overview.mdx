## High-level description
This directory contains documentation snippets related to features offered by a cloud-based data pipeline monitoring system. The snippets cover topics such as alerting mechanisms and automated anomaly detection.

## What does it do?
This documentation provides users with information on how to configure alerts for various events within their data pipelines, including model run failures, dbt test results, and anomaly detection warnings. It also explains the functionality of automated anomaly detection features, which help identify data freshness and volume issues.
