## High-level description
The `docs/_snippets/guides` directory contains code snippets and explanations for various aspects of using Elementary Data, a data observability platform. The snippets cover topics such as configuring alerts, collecting job data, and integrating with dbt source freshness.

## What does it do?
This directory houses documentation snippets that provide users with practical examples and instructions on how to leverage Elementary Data's features. These snippets demonstrate how to configure alerts for different scenarios, collect job metadata from various orchestrators, and integrate Elementary with dbt source freshness checks. The goal is to guide users on how to effectively use Elementary Data to monitor their data pipelines and receive timely notifications about potential issues.
