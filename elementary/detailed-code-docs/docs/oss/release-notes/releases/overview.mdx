## High-level description
The `/docs/oss/release-notes/releases` directory contains markdown files documenting the changes introduced in each release of Elementary Data, an open-source data observability platform. Each file provides a summary of new features, improvements, bug fixes, and contributions for a specific version.

## What does it do?
This directory acts as a historical record of Elementary Data's evolution, allowing users to track the progress of the platform across different versions. Each release note file highlights the key changes and additions, making it easy for users to stay informed about new functionalities, resolved issues, and contributions from the community. This information helps users understand the evolution of the platform and leverage the latest features and improvements.
