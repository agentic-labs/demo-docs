## High-level description
The `docs/oss/quickstart` directory contains documentation for quickly setting up and using the open-source Elementary CLI and dbt package. It guides users through the installation process and provides instructions for basic usage.

## What does it do?
This documentation guides users on how to install and run the Elementary CLI and dbt package. It explains how to set up a connection profile for the CLI to interact with the Elementary database and how to install the CLI itself. The documentation also includes frequently asked questions about the connection profile and its required permissions. Finally, it points users to further resources for using the CLI and integrating Elementary into their data workflow.
