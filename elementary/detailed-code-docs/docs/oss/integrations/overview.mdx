## High-level description
This directory contains documentation for integrating Elementary with various data warehouses and data transformation tools. It provides instructions and examples for using Elementary with BigQuery, Databricks, dbt, Postgres, Redshift, and Snowflake.

## What does it do?
This documentation explains how to use Elementary with different data systems. It covers both Elementary Cloud and Elementary CLI, providing specific instructions for each integration. The documentation helps users understand how to connect Elementary to their data warehouse, configure the integration, and start monitoring their data.
