## High-level description
This code defines a TypeScript class `Trieve` that serves as a client for interacting with a RESTful API. It provides methods for making HTTP requests, handling authentication, and processing responses. The class is designed to work with a generated OpenAPI schema, allowing for type-safe API calls.

## Code Structure
The `Trieve` class is the main export of this module. It contains a `fetch` method that handles the API requests. The code also defines several type aliases and utility functions to support the main functionality.
