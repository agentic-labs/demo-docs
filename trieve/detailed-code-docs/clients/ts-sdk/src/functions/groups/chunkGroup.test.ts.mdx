## High-level description
This file contains unit tests for the chunk group methods of the TrieveSDK. It tests various functionalities related to creating, searching, updating, and retrieving chunk groups and their associated data.

## Code Structure
The file defines a test suite using Vitest, with each test focusing on a specific method of the TrieveSDK related to chunk groups. The tests use a shared TrieveSDK instance initialized in the `beforeAll` hook.
