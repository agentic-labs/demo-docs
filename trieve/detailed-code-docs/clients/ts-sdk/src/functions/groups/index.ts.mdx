## High-level description
This file contains functions for interacting with the groups API of the Trieve SDK. It provides methods for creating, updating, deleting, and searching chunk groups, as well as managing chunks within groups.

## Code Structure
The file exports multiple async functions, each corresponding to a specific operation on chunk groups. These functions are methods of the TrieveSDK class, utilizing its `trieve.fetch` method to make API calls.
