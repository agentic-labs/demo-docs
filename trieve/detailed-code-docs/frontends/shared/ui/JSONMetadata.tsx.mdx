## High-level description
This code defines a React component called `JSONMetadata` that renders a collapsible JSON structure with customizable styling and copy functionality. It allows for displaying nested JSON objects and arrays in a user-friendly format.

## Code Structure
The main `JSONMetadata` component uses several helper components and functions to render the JSON structure. The `JSONMetadaRow` component is used to render individual rows of the JSON data, handling different data types and nested structures.

## Symbols

### JSONMetadata
#### Description
The main component that renders the entire JSON structure. It handles the overall layout, copy functionality, and delegates rendering of individual rows to `JSONMetadaRow`.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| isChild | boolean | Indicates if this is a child component |
| data | object | The JSON data to be displayed |
| isAlternate | boolean | Used for alternating row colors |
| copyJSONButton | boolean | Whether to show a copy button |
| closedByDefault | boolean | Whether rows should be collapsed by default |
| class | string | Additional CSS classes |
| monospace | boolean | Whether to use monospace font |

#### Internal Logic
- Creates memoized rows from the input data
- Handles copying of JSON data to clipboard
- Renders the copy button and rows

### JSONMetadaRow
#### Description
A component that renders a single row of JSON data, handling different data types and nested structures.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| key | string | The key of the JSON property |
| value | unknown | The value of the JSON property |
| isAlternate | boolean | Used for alternating row colors |
| isChild | boolean | Indicates if this is a child row |
| closedByDefault | boolean | Whether the row should be collapsed by default |

#### Internal Logic
- Determines the type of value and how to display it
- Handles collapsible sections for objects and arrays
- Recursively renders nested JSON structures
