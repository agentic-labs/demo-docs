## High-level description
The `TinyEditor` component is a wrapper for the TinyMCE rich text editor, providing a customizable and feature-rich text editing experience within a React application. It allows for HTML content editing with various formatting options and custom shortcuts.

## Code Structure
The main `TinyEditor` component is defined, which initializes and manages the TinyMCE editor instance. It uses Solid.js's `createEffect` and `onCleanup` for lifecycle management.

### TinyEditor
#### Description
A functional component that renders a TinyMCE editor instance with customized options and event handlers.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|

#### Internal Logic
1. Defines editor initialization options, including plugins, toolbar configuration, and custom shortcuts.

3. Sets up event listeners for content changes.
4. Implements custom shortcuts for font size changes and text highlighting.
5. Handles editor cleanup on component unmount.

The component uses lazy initialization of the TinyMCE editor to improve initial load time.
