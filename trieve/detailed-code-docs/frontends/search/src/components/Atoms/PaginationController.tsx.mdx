## High-level description
The `PaginationController` component is a reusable UI element for pagination in a search interface. It displays page numbers and navigation buttons, allowing users to move between pages of search results or other paginated content.

## Code Structure
The file contains two main exports: a utility function `createArrayWithCenteredRange` and the `PaginationController` component. The component uses the utility function to generate the array of page numbers to display.

## Symbols

### `createArrayWithCenteredRange`
#### Description
This function generates an array of numbers centered around a given value, with a specified range.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| center | number | The central value of the range |
| range | number | The total number of elements in the array |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| array | number[] | An array of numbers representing the page numbers to display |

#### Internal Logic
1. If the center is at the beginning of the range, it generates a sequential array from 1 to `range`.
2. Otherwise, it creates an array centered on the given value, ensuring it doesn't go below 1.

### `PaginationController`
#### Description
A functional component that renders pagination controls, including previous/next buttons and page numbers.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| props | PaginationControllerProps | Object containing setPage function, current page, and total pages |

#### Internal Logic
1. Renders a "previous" button if not on the first page.
2. Generates and displays an array of page numbers using `createArrayWithCenteredRange`.
3. Renders a "next" button if not on the last page.
4. Applies appropriate styling to the current page and buttons.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| solid-js | For reactive programming and component rendering |
| solid-icons/bi | For chevron icons in navigation buttons |

## Performance Considerations
The component uses `createArrayWithCenteredRange` to efficiently generate the array of page numbers, avoiding unnecessary re-renders by computing this array only when needed.
