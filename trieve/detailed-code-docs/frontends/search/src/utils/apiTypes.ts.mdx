## High-level description
This file defines TypeScript interfaces and types for various data structures used in the Trieve search application. It includes definitions for chunk metadata, search results, filters, and other related entities. The file also contains utility functions for type checking and data manipulation.

## Code Structure
The file primarily consists of interface and type definitions, with some utility functions interspersed. The interfaces and types are related to each other, often extending or referencing one another to create a cohesive type system for the application.
