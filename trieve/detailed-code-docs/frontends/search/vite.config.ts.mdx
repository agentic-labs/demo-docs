## High-level description
This file configures Vite for a Solid.js project with runtime environment variable support. It uses the `defineConfig` function from Vite to set up the build process with two plugins: `solid` for Solid.js support and `runtimeEnv` for runtime environment variable handling.

## Symbols

### `defineConfig`
#### Description
`defineConfig` is a function imported from the Vite package. It is used to define the configuration object for Vite, providing type checking and autocompletion for the configuration options.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| config | Object | An object containing Vite configuration options |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| config | Object | A typed Vite configuration object |

### Configuration Object
#### Description
The configuration object passed to `defineConfig` specifies the plugins to be used in the Vite build process.

#### Internal Logic
The `plugins` array in the configuration object includes two plugin functions:
1. `solid()`: Enables Solid.js support in the Vite build process.
2. `runtimeEnv()`: Adds runtime environment variable support to the project.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| vite | The build tool and dev server for modern web projects |
| vite-plugin-solid | Vite plugin for Solid.js support |
| vite-plugin-runtime-env | Vite plugin for runtime environment variable support |

This configuration file sets up a minimal Vite build process tailored for a Solid.js project with runtime environment variable support. It doesn't include any custom configuration options, relying on the default behavior of the included plugins.
