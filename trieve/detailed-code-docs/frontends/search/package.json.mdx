## High-level description
This package.json file defines the configuration and dependencies for the "search" frontend application of the Trieve Search project. It includes scripts for development, building, and linting, as well as a list of dependencies and development dependencies required for the project.

## Dependencies
The project relies on several external libraries and modules:

| Dependency | Purpose |
|:-----------|:--------|
| @nozbe/microfuzz | Fuzzy search functionality |
| @sentry/browser | Error tracking and monitoring |
| @solidjs/router | Routing for SolidJS applications |
| sanitize-html | HTML sanitization |
| shared | Shared components or utilities |
| solid-headless | Unstyled UI components for SolidJS |
| solid-icons | Icon library for SolidJS |
| solid-js | Core SolidJS library |
| trieve-ts-sdk | TypeScript SDK for Trieve |

## Development Dependencies
The project uses various development tools and libraries:

| Dependency | Purpose |
|:-----------|:--------|
| config | Configuration management |
| @tailwindcss/forms | Form styles for Tailwind CSS |
| @typescript-eslint/* | TypeScript ESLint plugins |
| autoprefixer | CSS autoprefixer |
| eslint | JavaScript linter |
| postcss | CSS post-processing tool |
| prettier | Code formatter |
| tailwindcss | Utility-first CSS framework |
| typescript | TypeScript language support |
| vite | Build tool and development server |

## Scripts
The package.json defines several npm scripts:

| Script | Description |
|:-------|:------------|
| dev | Starts the development server on port 5174 |
| build | Compiles TypeScript and builds the project |
| preview | Previews the built project |
| lint | Runs Prettier and ESLint to format and lint the code |
| lint:only | Runs ESLint only |
| lint:CI | Runs ESLint and outputs results to a JSON file |

## Configuration
The project is configured as an ES module with the "type" field set to "module". It also specifies the version and name of the package.

This package.json file sets up a comprehensive development environment for a SolidJS-based search frontend, including tools for styling, linting, and building the application.
