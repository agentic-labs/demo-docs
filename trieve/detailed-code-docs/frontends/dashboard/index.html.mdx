## High-level description
This HTML file serves as the main entry point for the Trieve Dashboard web application. It sets up the basic structure of the web page, includes necessary scripts and stylesheets, and configures the Mintlify widget for in-app documentation and support.

## Symbols

### `window.mintlifyWidgetSettings`
#### Description
This object configures the Mintlify widget, which provides in-app documentation and support for the Trieve Dashboard.

#### Internal Logic
The configuration includes:
- API key for connection
- Display settings (dark mode, colors, example queries)
- Trigger settings (button icon)

### Anonymous function for Mintlify widget initialization
#### Description
This self-executing function checks if the Mintlify widget script is already loaded, and if not, it dynamically adds the script to the document and initializes the widget.

#### Internal Logic
1. Checks if the script with id 'mintlify-widget' exists
2. If not, creates a new script element
3. Sets the script source to the Mintlify widget CDN
4. Adds an onload event to initialize the widget
5. Appends the script to the document head

This HTML file sets up the basic structure and resources needed for the Trieve Dashboard web application. It includes meta tags for SEO and social media sharing, configures the Mintlify widget for in-app support, and loads necessary scripts and stylesheets. The actual content of the dashboard will be rendered within the `&lt;div id="root"&gt;&lt;/div&gt;` element by the JavaScript application loaded from `/src/index.tsx`.
