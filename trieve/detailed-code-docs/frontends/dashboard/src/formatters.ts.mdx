### `formatBytesDecimal`
#### Description
Formats a number of bytes into a human-readable string representation in MB or GB.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| bytes | number | The number of bytes to format |
| dm | number | The number of decimal places (default: 2) |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| formattedSize | string | A string representing the size in MB or GB |

#### Internal Logic
- If bytes is 0, return "0 Bytes"
- If bytes is greater than 100MB, convert to GB
- Otherwise, convert to MB
- Round to the specified number of decimal places
