## High-level description
This code defines utility functions for formatting various types of data, including currency (USD), file sizes, numbers, and dates. These formatters are likely used throughout the application to ensure consistent and localized data presentation.

## Symbols

### `usdFormatter`
#### Description
A formatter for US dollar currency values.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| usdFormatter | Intl.NumberFormat | A formatter for USD currency |

#### Internal Logic
Uses the `Intl.NumberFormat` API with "en-US" locale and "USD" currency settings.

### `formatBytesDecimal`
#### Description
A function to format byte sizes into human-readable formats (MB or GB).

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| bytes | number | The number of bytes to format |
| dm | number | The number of decimal places (default: 2) |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| formattedSize | string | A string representing the formatted byte size |

#### Internal Logic
- If bytes is 0, returns "0 Bytes"
- If bytes &gt; 100MB, converts to GB
- Otherwise, converts to MB
- Uses the specified number of decimal places
