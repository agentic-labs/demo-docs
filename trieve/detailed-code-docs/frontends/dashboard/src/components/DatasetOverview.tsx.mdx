## High-level description
This code defines a `DatasetOverview` component in React that displays a list of datasets for a selected organization. It allows users to view, search, and manage datasets, including creating new datasets, deleting existing ones, and accessing various tools associated with each dataset.

## Code Structure
The main component `DatasetOverview` uses several hooks and sub-components to manage its state and render the UI. It interacts with an API to fetch and manipulate dataset information, and uses various UI components to display this information and provide user interactions.

## Symbols

### `DatasetOverview`
#### Description
The main component that renders the dataset overview UI.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| setOpenNewDatasetModal | Setter&lt;boolean&gt; | Function to open/close the new dataset modal |
| selectedOrganization | Accessor&lt;Organization \| undefined&gt; | The currently selected organization |

#### Internal Logic
1. Initializes state variables and fetches dataset information using the `useDatasetPages` hook.
2. Handles dataset search, pagination, and various dataset operations (delete, clear, reload chunk count).
3. Renders the dataset list with associated actions and tools.

### `PaginationArrows`
#### Description
A sub-component that renders pagination controls for the dataset list.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| page | Accessor&lt;number&gt; | Current page number |
| setPage | Setter&lt;number&gt; | Function to set the current page |
| maxPageDiscovered | Accessor&lt;number \| null&gt; | Maximum page number discovered |

## Side Effects
- Fetches dataset usage information from the API.
- Deletes datasets via API calls.
- Clears datasets via API calls.
- Reloads chunk counts for datasets via API calls.

## Dependencies

## Error Handling
The component uses `createToast` to display error messages when API operations fail, such as deleting or clearing datasets, or reloading chunk counts.

## API/Interface Reference
| Endpoint | Method | Description |
|:---------|:-------|:------------|
| /dataset/${datasetId} | DELETE | Deletes a dataset |
| /dataset/clear/${datasetId} | PUT | Clears a dataset |
| /dataset/usage/${datasetId} | GET | Fetches usage information for a dataset |
