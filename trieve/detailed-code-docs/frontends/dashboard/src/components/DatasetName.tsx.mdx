## High-level description
The `DatasetName` component is a React component that displays the name of the current dataset. It utilizes the `DatasetContext` to access the dataset information and renders the dataset name within an `h3` element.

## Symbols

### DatasetName
#### Description
`DatasetName` is a functional component that retrieves the current dataset name from the `DatasetContext` and renders it as part of a heading.

#### Inputs
This component doesn't accept any props.

#### Outputs
Returns a JSX element: an `h3` tag containing the dataset name followed by the word "Dataset".

#### Internal Logic
1. The component uses the `useContext` hook to access the `DatasetContext`.
2. It creates a memoized value `curDatasetName` using `createMemo`:
   - This memo retrieves the current dataset from the context.
   - If the dataset exists, it returns the dataset name; otherwise, it returns null.
3. The component renders an `h3` element with specific CSS classes.
4. Inside the `h3`, it displays the result of calling `curDatasetName()` followed by the word "Dataset".

## Related Code Context
The `DatasetName` component is used within the `DatasetLayout` component (from `DatasetLayout.tsx`). It's part of the header section of the dataset view, appearing above the `DatasetTabs` component.

The `DatasetContext` (from `DatasetContext.tsx`) provides the dataset information used by this component. It fetches the dataset data based on the current user and selected organization.
