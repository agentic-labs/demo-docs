## High-level description
The `BuildingSomething` component is a React functional component that renders an informational message encouraging users to share their projects in Discord or Matrix communities. It's designed to be a reusable UI element that can be easily integrated into various parts of the dashboard.

## Symbols

### `BuildingSomething`
#### Description
This is a functional component that creates a styled div containing an information icon and a message with links to Discord and Matrix communities.

#### Inputs
This component doesn't accept any props.

#### Outputs
Returns a JSX element representing the styled informational message.

#### Internal Logic
1. The component uses the `BiRegularInfoCircle` icon from the `solid-icons/bi` package.
2. It creates a flex container with specific styling classes for layout and appearance.
3. Inside the container, it renders the info icon and a paragraph with the message.
4. The message includes two hyperlinks (Discord and Matrix) with underline styling.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| solid-icons/bi | Provides the `BiRegularInfoCircle` icon component |

## References
This component is used in the following files:
1. `frontends/dashboard/src/pages/Dashboard/Overview.tsx`
2. `frontends/dashboard/src/pages/Dashboard/Dataset/DatasetStart.tsx`

In both cases, it's rendered as part of larger dashboard views to encourage user engagement and community participation.

## Side Effects
This component doesn't have any side effects. It's a pure presentational component that renders static content.

## Performance Considerations
As a simple, stateless component with no complex logic or data fetching, this component should have minimal performance impact. It can be safely used in various parts of the application without significant performance concerns.
