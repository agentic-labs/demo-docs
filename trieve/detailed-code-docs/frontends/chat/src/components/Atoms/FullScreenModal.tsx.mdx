## High-level description
The `FullScreenModal` component is a reusable React component that creates a full-screen modal dialog. It uses the `Dialog` component from the `terracotta` library and provides a customizable container for displaying content in a modal overlay.

## Symbols

### FullScreenModalProps
#### Description
An interface that defines the props for the `FullScreenModal` component.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| children | JSX.Element | The content to be displayed inside the modal |
| isOpen | Accessor&lt;boolean&gt; | A function that returns a boolean indicating whether the modal is open |
| setIsOpen | Setter&lt;boolean&gt; | A function to set the open state of the modal |

### FullScreenModal
#### Description
A functional component that renders a full-screen modal dialog.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| props | FullScreenModalProps | The props for the FullScreenModal component |

#### Internal Logic
1. The component uses a `Show` component to conditionally render the modal based on the `isOpen` prop.
2. It renders a `Dialog` component from the `terracotta` library, passing the `isOpen` state and an `onClose` handler.
3. The modal content is wrapped in a flex container for centering.
4. A `DialogOverlay` is used to create a semi-transparent background.
5. The main content is wrapped in a `DialogPanel` with styling for appearance and positioning.
6. The `children` prop is rendered inside the `DialogPanel`.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| terracotta | Provides the Dialog, DialogOverlay, and DialogPanel components |
| solid-js | Provides the Accessor, JSX, Setter, and Show components/types |
