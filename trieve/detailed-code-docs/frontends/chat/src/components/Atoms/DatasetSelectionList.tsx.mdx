## High-level description
The `DatasetSelectionList` component is a reusable React component that renders a list of selectable datasets. It allows users to choose a dataset from the provided list and updates the current dataset in the user context when selected.

## Code Structure
The main component `DatasetSelectionList` uses the `UserContext` to access and update the current dataset. It renders a list of `MenuItem` components for each dataset, with visual indicators for the currently selected dataset.

## Symbols

### DatasetSelectionList
#### Description
A functional component that renders a list of selectable datasets.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| onSelect | () =&gt; void | Callback function to be called when a dataset is selected |
| datasets | DatasetAndUsageDTO[] | Array of dataset objects to be displayed in the list |

#### Internal Logic
1. Retrieves the current dataset from the `UserContext`.
2. Maps through the first 500 datasets in the `datasets` prop.
3. For each dataset, renders a `MenuItem` component with:
   - Styling based on whether it's the currently selected dataset.
   - Click handler to update the current dataset and call the `onSelect` callback.
   - Dataset name display.
   - A checkmark icon for the currently selected dataset.
