## High-level description
This TypeScript file defines interfaces and utility functions related to an Actix-based API, focusing on user plans, Stripe checkout sessions, and referral token management. It provides type definitions and type guards for API responses and user data structures.

## Symbols

### `ActixApiDefaultError`
#### Description
An interface representing the default error structure for the Actix API.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| message | string | The error message |

### `StripeCheckoutSessionResponse`
#### Description
An interface representing the response structure for a Stripe checkout session.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| checkout_session_url | string | The URL for the Stripe checkout session |

### `detectReferralToken`
#### Description
A function that detects and stores referral tokens in the browser's local storage.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| queryParamT | string \| undefined | The referral token from query parameters |

#### Internal Logic
1. Checks if a referral token is provided
2. Retrieves previously stored tokens from local storage
3. Adds the new token if it's not already present
4. Updates the local storage with the new token list

### `getReferralTokenArray`
#### Description

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|

## Dependencies
This file doesn't explicitly import any external libraries, but it relies on the browser's `window` object and `localStorage` API for managing referral tokens.

## Side Effects
The `detectReferralToken` function modifies the browser's local storage by storing and updating referral tokens.
