## High-level description
This TypeScript file defines utility functions and formatters for handling currency, number, and data size formatting. It provides a USD currency formatter, a function to format bytes into human-readable sizes, and a general number formatter.

## Symbols

### `usdFormatter`
#### Description
A constant that creates an instance of `Intl.NumberFormat` configured to format numbers as US dollar currency.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| usdFormatter | Intl.NumberFormat | An object that can be used to format numbers as USD currency |

#### Internal Logic
Uses the `Intl.NumberFormat` constructor with "en-US" locale and options specifying the style as "currency" and the currency as "USD".

___

### `formatBytesDecimal`
#### Description
A function that converts a number of bytes into a human-readable string representation in either megabytes (MB) or gigabytes (GB).

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| bytes | number | The number of bytes to format |
| dm | number | The number of decimal places to show (default is 2) |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| formattedSize | string | A string representing the size in MB or GB with the specified decimal places |

#### Internal Logic
1. If the input is 0, it returns "0 Bytes".
2. If the input is greater than 100 MB (1e8 bytes), it converts to GB by dividing by 1e9.
3. Otherwise, it converts to MB by dividing by 1e6.
4. The result is rounded to the specified number of decimal places using `toFixed()`.
5. The appropriate unit (MB or GB) is appended to the formatted number.

___

### `numberFormatter`
#### Description
A constant that creates an instance of `Intl.NumberFormat` configured to format numbers using US English conventions.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| numberFormatter | Intl.NumberFormat | An object that can be used to format numbers according to US English conventions |

#### Internal Logic
Uses the `Intl.NumberFormat` constructor with "en-US" locale, using default options for number formatting.
