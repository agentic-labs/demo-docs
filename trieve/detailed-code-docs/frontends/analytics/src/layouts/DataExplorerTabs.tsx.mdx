## High-level description
The `DataExplorerTabs` component renders a set of navigation tabs that allow users to switch between different data exploration views within the application. It dynamically highlights the currently active tab based on the current URL pathname.

## Code Structure
The `DataExplorerTabs` component uses the `pages` array to define the available tabs. It utilizes the `useBetterNav` hook for navigation and the `usePathname` hook to determine the active tab. The `For` component iterates over the `pages` array to render each tab as an anchor tag.

## References
- `useBetterNav`: A custom hook for navigation, preserving query parameters.
- `usePathname`: A custom hook to get the current URL pathname.

#### Outputs

#### Internal Logic
1. **Retrieves navigation and pathname hooks:** Uses `useBetterNav` for navigation and `usePathname` to get the current URL pathname.
2. **Defines tab data:** Uses the `pages` array to store tab names and corresponding paths.
3. **Renders tabs:** Iterates through the `pages` array using the `For` component.
4. **Dynamically applies styles:** Applies the `text-fuchsia-800` class to the active tab's anchor tag based on the current pathname.
5. **Handles tab clicks:** Uses the `handleLinkClick` function to navigate to the clicked tab's path while preserving query parameters using the `betterNav` hook.
6. **Renders child components:** Renders the `children` prop within a div, representing the content area of the active tab.
