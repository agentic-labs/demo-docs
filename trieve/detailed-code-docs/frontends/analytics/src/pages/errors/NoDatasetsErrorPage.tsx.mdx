## High-level description
The `NoDatasetsErrorPage` component displays an error message to the user when there are no datasets found for the selected organization. It provides a link to the dashboard where the user can create a new dataset.

## References
- `OrgContext`: Used to access the selected organization's ID.

## Symbols
### `NoDatasetsErrorPage`
#### Description
This component renders an error page indicating that no datasets exist for the current organization. It provides a link to the dashboard for creating a new dataset.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| props | `NoDatasetsErrorPageProps` | An object containing the `orgId` property. |

#### Internal Logic
1. Retrieves the dashboard URL from environment variables.
2. Constructs a link to the dashboard's dataset creation page using the organization ID.
3. Renders a styled div containing:
    - An error message informing the user about the lack of datasets.
    - A link to the dashboard for creating a new dataset.

## Dependencies
- `solid-icons/fi`: Used for the external link icon.
- `@solidjs/router`: Used for redirecting to the dashboard.

## Configuration
- `VITE_DASHBOARD_URL`: Environment variable storing the base URL of the dashboard.
