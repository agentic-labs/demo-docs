## High-level description
The `Sidebar` component is a navigation sidebar for an analytics dashboard. It displays the user's organization and dataset selection, navigation links, and user information. The component is designed to be used within a larger application layout.

## Code Structure
The `Sidebar` component is the main export of this file. It uses various context providers (`OrgContext` and `UserContext`) to access user and organization data. The component renders a series of navigation links, dataset and organization selectors, and user information.

## Symbols

### Sidebar
#### Description
The `Sidebar` component is a functional component that renders the sidebar navigation for the analytics dashboard.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| datasetOptions | DatasetAndUsage[] | Array of available datasets |
| selectedDataset | DatasetAndUsage \| null | Currently selected dataset |
| setSelectedDataset | (dataset: DatasetAndUsage) =&gt; void | Function to set the selected dataset |

#### Internal Logic
1. Retrieves user and organization context using `useContext`
2. Uses `usePathname` and `useBetterNav` hooks for navigation
3. Implements a `logOut` function to handle user logout
4. Creates a `createMemo` for constructing dataset parameters
5. Renders the sidebar structure with logo, organization and dataset selectors, navigation links, and user information

## Side Effects
- The component makes a fetch request to log out the user when the logout button is clicked
- It uses the browser's `window.location` to redirect the user after logout
