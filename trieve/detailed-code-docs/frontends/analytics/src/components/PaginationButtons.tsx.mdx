## High-level description
The `PaginationButtons` component renders a simple pagination control with buttons to navigate to the previous and next pages. It displays the current page number and disables the navigation buttons when at the first or last page.

## Code Structure
The `PaginationButtons` component uses the `usePagination` hook to manage the pagination state. It receives the pagination state and an optional size prop as input. The component renders three elements: a button for the previous page, a div displaying the current page number, and a button for the next page. The buttons are disabled based on the current page and maximum discovered page.

## Symbols
### `PaginationButtons`
#### Description
This component renders pagination buttons for navigating through multiple pages of data. It displays the current page number and provides buttons to move to the previous and next pages. The buttons are disabled if the user is on the first or last page, respectively.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| pages | `ReturnType&lt;typeof usePagination&gt;` | An object containing the pagination state and functions from the `usePagination` hook. |
| size | `number` | (Optional) The size of the pagination buttons. Defaults to 24. |

#### Outputs
JSX representing the pagination buttons.

#### Internal Logic
The component uses the `pages` object to determine the current page, the maximum discovered page, and whether to disable the navigation buttons. It renders a left arrow icon for the previous page button and a rotated left arrow icon for the next page button. The current page number is displayed between the buttons.
