## High-level description
The `PaginationButtons` component is a reusable UI element for pagination controls in a React application. It provides buttons for navigating between pages and displays the current page number.

## Symbols

### PaginationButtons
#### Description
This is a functional component that renders pagination controls, including previous and next page buttons, and the current page number.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| props | PaginationButtonsProps | Contains the pagination state and optional size for icons |

#### Internal Logic
1. Renders a container div with flex layout.
2. Renders a "previous page" button, disabled when on the first page.
3. Displays the current page number.
4. Renders a "next page" button, disabled when on the last discovered page.
5. Uses the `AiFillCaretLeft` icon for both buttons, rotating it 180 degrees for the "next page" button.

### PaginationButtonsProps
#### Description
An interface defining the props for the PaginationButtons component.

#### Properties
| Name | Type | Description |
|:-----|:-----|:------------|
| pages | ReturnType&lt;typeof usePagination&gt; | Pagination state and methods |
| size? | number | Optional size for the pagination icons |
