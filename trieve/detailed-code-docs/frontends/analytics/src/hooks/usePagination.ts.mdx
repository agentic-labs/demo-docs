## High-level description
The `usePagination` hook provides functionality for managing pagination state and actions within a SolidJS component. It allows tracking the current page, navigating to the next/previous pages, and determining if further navigation is possible.

## Code Structure
The `usePagination` hook utilizes SolidJS's `createSignal` and `createMemo` primitives to manage its internal state and derived values. The `page` signal represents the current page number, while `maxPageDiscovered` tracks the highest page fetched so far. The `canGoNext` memo determines if navigation to the next page is allowed based on the `maxPageDiscovered`.

## Symbols
### `usePagination`
#### Description
This hook manages pagination state and provides functions for navigation and state updates.

#### Inputs
None

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| page | Accessor&lt;number&gt; | Signal representing the current page number. |
| nextPage | () =&gt; void | Function to navigate to the next page. |
| prevPage | () =&gt; void | Function to navigate to the previous page. |
| canGoNext | Accessor&lt;boolean&gt; | Memo indicating if navigation to the next page is possible. |
| setMaxPageDiscovered | Setter&lt;number \| null&gt; | Function to update the highest page discovered. |
| resetMaxPageDiscovered | () =&gt; void | Function to reset the highest page discovered. |
| maxPageDiscovered | Accessor&lt;number \| null&gt; | Signal representing the highest page fetched. |

#### Internal Logic
- `nextPage`: Increments the current page number.
- `prevPage`: Decrements the current page number if it's greater than 1.
- `canGoNext`: Determines if the next page exists based on the `maxPageDiscovered`.
- `setMaxPageDiscovered`: Updates the `maxPageDiscovered` signal with the provided page number.
- `resetMaxPageDiscovered`: Resets the `maxPageDiscovered` signal to `null`.

## Side Effects
The hook itself doesn't have any direct side effects. However, the returned functions (`nextPage`, `prevPage`, `setMaxPageDiscovered`, `resetMaxPageDiscovered`) modify the internal state of the hook, which can trigger re-renders in components using this hook.
