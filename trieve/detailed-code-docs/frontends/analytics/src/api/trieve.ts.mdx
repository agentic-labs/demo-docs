## High-level description

## Symbols
### `trieve`
#### Description
This symbol is an instance of the `Trieve` class from the `trieve-fetch-client` library, configured to interact with the Trieve API. It is exported for use in other parts of the application.

#### Internal Logic
The code initializes a new `Trieve` object with an object containing a single property:
- `baseUrl`: This property is set to the value of the `VITE_API_HOST` environment variable, cast to a string. This variable is assumed to contain the base URL of the Trieve API.
