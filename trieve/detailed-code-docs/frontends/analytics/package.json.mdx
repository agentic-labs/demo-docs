## Scripts
The package.json file defines several npm scripts:

### `dev`
#### Description
Starts the development server using Vite on port 5176.

#### Internal Logic
Runs the command `vite --port 5176`.

### `build`
#### Description
Builds the project for production.

#### Internal Logic
Runs TypeScript compilation (`tsc`) and then builds the project using Vite (`vite build`).

### `preview`
#### Description
Previews the built project.

#### Internal Logic
Runs the command `vite preview`.

### `lint`
#### Description
Lints and formats the project files.

#### Internal Logic
1. Runs Prettier to format various file types.
2. Runs ESLint to fix issues in JavaScript, TypeScript, and JSX files.

### `lint:only`
#### Description
Runs ESLint without Prettier formatting.

#### Internal Logic
Runs ESLint to fix issues in JavaScript, TypeScript, and JSX files.

### `lint:CI`
#### Description
Runs ESLint and generates a JSON report, likely for use in continuous integration environments.

#### Internal Logic
Runs ESLint and outputs the results to a JSON file named `eslint_report.json`.
