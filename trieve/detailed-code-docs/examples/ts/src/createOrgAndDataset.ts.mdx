## High-level description
This script creates a new organization and a dataset within that organization using the Trieve API. It then logs the created dataset information to the console.

## References
- `trieve`: This symbol is imported from `./trieve.ts` and represents an instance of the `Trieve` client used to interact with the Trieve API.

## Symbols
### `main`
#### Description
This asynchronous function orchestrates the creation of an organization and a dataset using the Trieve API.

#### Inputs
This function has no inputs.

#### Outputs
This function has no explicit outputs. It logs the created dataset information to the console.

#### Internal Logic
1. **Create Organization:** Sends a POST request to the `/api/organization` endpoint with the organization name to create a new organization.
2. **Create Dataset:** Sends a POST request to the `/api/dataset` endpoint with the dataset name and the ID of the newly created organization.
3. **Log Dataset:** Logs the created dataset information to the console.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| trieve-fetch-client | Provides the `Trieve` class for interacting with the Trieve API. |

## Logging
The script logs the created dataset information to the console using `console.log`.
