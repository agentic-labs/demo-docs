## High-level description
This SQL script adds a new column named `oc_file_path` to the `card_metadata` table. This column is intended to store text-based file paths and is nullable, meaning it can be empty. 

## Symbols

### `ALTER TABLE card_metadata`
#### Description
This SQL statement modifies the existing `card_metadata` table.

#### Internal Logic
It uses the `ADD COLUMN` clause to append a new column with the following specifications:

*   **`oc_file_path`**: The name of the new column.
*   **`TEXT`**:  The data type of the column, indicating it will store text strings.
*   **`NULL`**: This specifies that the column allows null values, meaning it can be empty.
