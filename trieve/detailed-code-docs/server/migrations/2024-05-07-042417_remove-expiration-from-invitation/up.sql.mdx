## High-level description
This SQL script defines a database migration that removes the `expires_at` column from the `invitations` table. 

## Symbols

### `ALTER TABLE invitations DROP COLUMN expires_at;`
#### Description
This SQL statement modifies the existing `invitations` table by removing the `expires_at` column. 

#### Internal Logic
* **`ALTER TABLE invitations`**: This clause specifies that the `invitations` table is being modified.
* **`DROP COLUMN expires_at`**: This clause instructs the database to remove the column named `expires_at` from the table. 

#### Side Effects
This migration will permanently delete the `expires_at` column and all its data from the `invitations` table.
