## High-level description
This SQL script adds a new column named `card_html` to the `card_metadata` table. This column will store text and is nullable, meaning it can be empty.

## Symbols

### `ALTER TABLE card_metadata`
#### Description
This SQL statement modifies the existing `card_metadata` table.

#### Internal Logic
It adds a new column using the `ADD COLUMN` clause.

### `ADD COLUMN card_html TEXT NULL;`
#### Description
This clause adds a new column with the following specifications:

* **Column Name:** `card_html`
* **Data Type:** `TEXT` - This indicates that the column will store textual data.
* **Nullability:** `NULL` - This means the column can contain NULL values, indicating the absence of data.
