## High-level description
This SQL script adds a new column named `card_html` to the `card_metadata` table. This column will store text and has a `TEXT` data type, allowing for large amounts of data. It is nullable, meaning it can be empty.

## Symbols

### `ALTER TABLE card_metadata`
#### Description
This SQL statement modifies the existing `card_metadata` table.

#### Internal Logic
It adds a new column to the table using the `ADD COLUMN` clause.

### `ADD COLUMN card_html TEXT NULL;`
#### Description
This clause adds a new column with the following properties:

* **Name:** `card_html`
* **Data Type:** `TEXT` -  capable of storing large string values.
* **Nullability:** `NULL` - the column can contain null values.
