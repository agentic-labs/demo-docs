## High-level description
This SQL script defines a database migration that reverts a change made to the `card_metadata` table. Specifically, it drops the `card_html` column. This script is intended to be run as part of a database migration rollback.

## Symbols

### `ALTER TABLE card_metadata DROP COLUMN card_html;`
#### Description
This SQL statement removes the `card_html` column from the `card_metadata` table.

#### Internal Logic
- **`ALTER TABLE card_metadata`**: This clause specifies that the `card_metadata` table is being modified.
- **`DROP COLUMN card_html`**: This clause specifies that the `card_html` column should be removed from the table.

#### Side Effects
This statement modifies the database schema by removing a column from the `card_metadata` table. Any data previously stored in the `card_html` column will be permanently deleted.
