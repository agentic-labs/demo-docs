## High-level description
This SQL script reverts a database migration by adding back the `client_configuration` column to the `datasets` table. This action presumably undoes the change introduced in the corresponding `up.sql` migration file.

## Symbols

### `ALTER TABLE datasets ADD COLUMN client_configuration jsonb;`
#### Description
This SQL statement adds a column named `client_configuration` to the `datasets` table. The column's data type is `jsonb`, which is used to store JSON documents in PostgreSQL.

#### Internal Logic
1. **`ALTER TABLE datasets`**: This clause specifies that the `datasets` table is being modified.
2. **`ADD COLUMN client_configuration`**: This clause adds a new column named `client_configuration` to the table.
3. **`jsonb`**: This specifies the data type of the new column as `jsonb`, allowing it to store JSON data.

#### Side Effects
- The `datasets` table will be modified to include the `client_configuration` column.
- Existing data in the `datasets` table will remain untouched, and the new column will initially contain `NULL` values for all existing rows.
