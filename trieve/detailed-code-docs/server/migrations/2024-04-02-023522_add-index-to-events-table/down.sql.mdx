## High-level description
This SQL script defines a database migration that reverts changes made by the corresponding `up.sql` migration. Specifically, it drops two indexes, `idx_events_dataset_id` and `idx_events_event_type`, from the `events` table.

## Symbols

### `DROP INDEX IF EXISTS`
#### Description
This SQL statement attempts to drop an index if it exists. This ensures that the script can be run multiple times without causing errors if the index is already dropped.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| idx_events_dataset_id | string | The name of the index to drop. |
| idx_events_event_type | string | The name of the index to drop. |

#### Outputs
None. This statement modifies the database schema by dropping the specified indexes.
