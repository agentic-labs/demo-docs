## High-level description
This SQL script reverts the database migration defined in the corresponding `up.sql` file. It specifically removes a GiST index named `idx_gist` and the `pg_trgm` extension if they exist.

## Symbols

### `DROP INDEX IF EXISTS idx_gist;`
#### Description
This statement attempts to drop the index named `idx_gist`. The `IF EXISTS` clause ensures that the statement executes successfully even if the index does not exist.

#### Internal Logic
- Checks if an index named `idx_gist` exists in the database.
- If the index exists, it is dropped.
- If the index does not exist, no action is taken.

### `DROP EXTENSION IF EXISTS pg_trgm;`
#### Description
This statement attempts to drop the PostgreSQL extension `pg_trgm`. The `IF EXISTS` clause ensures that the statement executes successfully even if the extension is not installed.

#### Internal Logic
- Checks if the `pg_trgm` extension is installed in the database.
- If the extension is installed, it is dropped.
- If the extension is not installed, no action is taken.
