## High-level description
This SQL script adds a new column named `oc_file_path` to the `files` table in a database. The new column is of type `TEXT` and allows null values. 

## Symbols

### `ALTER TABLE files`
#### Description
This SQL statement modifies the existing `files` table.

#### Internal Logic
It adds a new column using the `ADD COLUMN` clause.

### `ADD COLUMN oc_file_path TEXT NULL;`
#### Description
This clause defines the new column being added to the `files` table.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| oc_file_path | TEXT | The name of the new column. |
| TEXT | Data type | Specifies the column will store text data. |
| NULL | Constraint | Allows the column to contain null values. |
