## High-level description
This SQL script creates a new table named `password_resets` in the database. This table is designed to store information required for a password reset functionality, such as user email, reset token expiration, and timestamps for creation and updates.

## Symbols

### `password_resets` table
#### Description
This table stores data related to password reset requests. Each row represents a single password reset request.

#### Columns
| Name | Type | Description |
|:-----|:-----|:------------|
| `id` | UUID NOT NULL UNIQUE PRIMARY KEY | Unique identifier for each reset request. |
| `email` | VARCHAR(100) NOT NULL | Email address of the user requesting a password reset. |
| `expires_at` | TIMESTAMP NOT NULL | Timestamp indicating when the password reset request expires. |
| `created_at` | TIMESTAMP NOT NULL | Timestamp indicating when the password reset request was created. |
| `updated_at` | TIMESTAMP NOT NULL | Timestamp indicating when the password reset request was last updated. |
