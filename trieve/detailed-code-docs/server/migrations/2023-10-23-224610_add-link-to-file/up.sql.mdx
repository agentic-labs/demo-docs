## High-level description
This SQL script adds a new column named `link` to the `files` table in a database. The `link` column is of type `text`, allows null values, and defaults to null.

## Symbols

### `ALTER TABLE files`
#### Description
This SQL statement modifies the existing `files` table.

#### Internal Logic
It uses the `ADD COLUMN` clause to append a new column with the following specifications:

*   **`link`**: The name of the new column.
*   **`text`**: The data type of the new column, capable of storing long strings.
*   **`NULL`**: Allows the column to contain null values.
*   **`DEFAULT NULL`**: Sets the default value of the column to `NULL` for new rows inserted into the table.

This means that existing rows in the `files` table will have a `NULL` value in the newly added `link` column by default.
