## High-level description
This SQL script is a database migration script that reverts a change made to the `files` table. Specifically, it drops the `link` column from the table. This script is intended to be run as part of a database migration rollback.

## Symbols

### `ALTER TABLE files DROP COLUMN link;`
#### Description
This SQL statement removes the `link` column from the `files` table.

#### Internal Logic
* **`ALTER TABLE files`**: This part of the statement specifies that the `files` table is being modified.
* **`DROP COLUMN link`**: This part instructs the database to remove the column named `link`.
* **`;`**: This semicolon marks the end of the SQL statement.
