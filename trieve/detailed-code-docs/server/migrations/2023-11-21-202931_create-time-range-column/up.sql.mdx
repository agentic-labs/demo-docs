## High-level description
This SQL script adds a new column named `time_stamp` of type `TIMESTAMP` with a default value of `NULL` to two existing tables: `card_metadata` and `files`. 

## Symbols

### `ALTER TABLE card_metadata ADD COLUMN time_stamp TIMESTAMP NULL DEFAULT NULL;`
#### Description
This SQL statement adds a new column named `time_stamp` to the `card_metadata` table.

#### Internal Logic
1. **`ALTER TABLE card_metadata`**: This clause specifies that the `card_metadata` table is being modified.
2. **`ADD COLUMN time_stamp`**: This clause adds a new column named `time_stamp` to the table.
3. **`TIMESTAMP`**: This specifies the data type of the new column as `TIMESTAMP`, which stores both date and time information.
4. **`NULL DEFAULT NULL`**: This sets the default value of the `time_stamp` column to `NULL`, meaning that new rows inserted into the table will have a `NULL` value for this column unless explicitly specified otherwise.

### `ALTER TABLE files ADD COLUMN time_stamp TIMESTAMP NULL DEFAULT NULL;`
#### Description
This SQL statement is identical to the previous one, except it adds the `time_stamp` column to the `files` table instead of the `card_metadata` table.
