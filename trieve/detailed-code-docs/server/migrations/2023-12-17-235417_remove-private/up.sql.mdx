## High-level description
This SQL script defines a database migration that removes the `private` column from both the `card_metadata` and `files` tables. 

## Symbols

### `ALTER TABLE card_metadata DROP COLUMN private;`
#### Description
This statement modifies the `card_metadata` table by removing the `private` column.

#### Internal Logic
- `ALTER TABLE card_metadata`: This part specifies that the `card_metadata` table is being modified.
- `DROP COLUMN private`: This part instructs the database to remove the column named `private`.

### `ALTER TABLE files DROP COLUMN private;`
#### Description
This statement modifies the `files` table by removing the `private` column.

#### Internal Logic
- `ALTER TABLE files`: This part specifies that the `files` table is being modified.
- `DROP COLUMN private`: This part instructs the database to remove the column named `private`.
