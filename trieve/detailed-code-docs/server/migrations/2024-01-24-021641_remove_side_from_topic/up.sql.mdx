## High-level description
This SQL script defines a database migration that removes the `side` and `normal_chat` columns from the `topics` table. 

## Symbols

### `ALTER TABLE topics DROP COLUMN side;`
#### Description
This SQL statement removes the `side` column from the `topics` table.

#### Internal Logic
This statement uses the `ALTER TABLE` command to modify the structure of the `topics` table. The `DROP COLUMN` clause specifies that the `side` column should be removed.

### `ALTER TABLE topics DROP COLUMN normal_chat;`
#### Description
This SQL statement removes the `normal_chat` column from the `topics` table.

#### Internal Logic
Similar to the previous statement, this statement uses `ALTER TABLE` and `DROP COLUMN` to remove the `normal_chat` column from the `topics` table.
