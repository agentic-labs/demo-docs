## High-level description
This SQL script adds a new column named `normal_chat` to the `topics` table in a database. The column is defined as a boolean with a default value of `false` and is set as `NOT NULL`.

## Symbols

### `ALTER TABLE topics ADD COLUMN normal_chat boolean NOT NULL DEFAULT false;`
#### Description
This SQL statement adds a new column to the existing `topics` table.

#### Internal Logic
1. **`ALTER TABLE topics`**: This part specifies that the `topics` table is being modified.
2. **`ADD COLUMN normal_chat`**: This adds a new column named `normal_chat` to the table.
3. **`boolean`**: This defines the data type of the new column as boolean (true/false).
4. **`NOT NULL`**: This constraint ensures that the `normal_chat` column cannot have null values.
5. **`DEFAULT false`**: This sets the default value of the `normal_chat` column to `false` for any new rows inserted into the table.
