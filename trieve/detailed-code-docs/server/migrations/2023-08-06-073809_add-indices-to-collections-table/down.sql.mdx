## High-level description
This SQL script reverts database schema changes related to indexes on the `card_collection` and `card_collection_bookmarks` tables. It's designed to be executed as a down migration in a database migration tool.

## Symbols
### `down.sql`
#### Description
This SQL script removes indexes previously created on the `card_collection` and `card_collection_bookmarks` tables. This is typically used to revert database schema changes during a down migration.

#### Internal Logic
The script uses `DROP INDEX IF EXISTS` statements to safely remove indexes. This ensures that the script can be run multiple times without causing errors if the indexes don't exist.

The script removes the following indexes:

**From `card_collection` table:**
* `idx_card_collection_author_id`
* `idx_card_collection_is_public`
* `idx_card_collection_created_at`
* `idx_card_collection_updated_at`

**From `card_collection_bookmarks` table:**
* `idx_card_collection_bookmarks_collection_id`
* `idx_card_collection_bookmarks_card_metadata_id`
* `idx_card_collection_bookmarks_created_at`
* `idx_card_collection_bookmarks_updated_at`
