## High-level description
This SQL script defines a database migration that adds indices to the `card_collection` and `card_collection_bookmarks` tables. These indices are intended to improve the performance of queries that filter or sort by the indexed columns. 

## Symbols

### `up.sql`
#### Description
This SQL script adds indices to specific columns in the `card_collection` and `card_collection_bookmarks` tables. 

#### Internal Logic
The script defines two sets of index creations:

* **`card_collection` table:**
    * `idx_card_collection_author_id`: Creates an index on the `author_id` column.
    * `idx_card_collection_is_public`: Creates an index on the `is_public` column.
    * `idx_card_collection_created_at`: Creates an index on the `created_at` column.
    * `idx_card_collection_updated_at`: Creates an index on the `updated_at` column.

* **`card_collection_bookmarks` table:**
    * `idx_card_collection_bookmarks_collection_id`: Creates an index on the `collection_id` column.
    * `idx_card_collection_bookmarks_card_metadata_id`: Creates an index on the `card_metadata_id` column.
    * `idx_card_collection_bookmarks_created_at`: Creates an index on the `created_at` column.
    * `idx_card_collection_bookmarks_updated_at`: Creates an index on the `updated_at` column.

#### Side Effects
This script modifies the database schema by adding the defined indices to the specified tables.
