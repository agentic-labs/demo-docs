## High-level description
This SQL script adds a unique constraint to the `chunk_group_bookmarks` table. This constraint ensures that for a given `group_id`, each `chunk_metadata_id` is unique. 

## Symbols

### `ALTER TABLE chunk_group_bookmarks`
#### Description
This statement modifies the existing `chunk_group_bookmarks` table.

#### Internal Logic
This statement adds a new constraint named `group_id_chunk_metadata_id_key` to the table. This constraint is a UNIQUE constraint, meaning that the combination of `group_id` and `chunk_metadata_id` must be unique for each row in the table.
