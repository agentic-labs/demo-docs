## High-level description
This SQL script reverts the database changes introduced by the migration `2023-05-27-192000_user_info`. It specifically removes the columns `username`, `website`, and `visible_email` from the `users` table.

## Symbols

### `ALTER TABLE users ...`
#### Description
This SQL statement modifies the existing `users` table by dropping specific columns.

#### Internal Logic
1. **`ALTER TABLE users`**: This clause initiates the modification of the `users` table.
2. **`DROP COLUMN username, DROP COLUMN website, DROP COLUMN visible_email;`**: These clauses individually drop the specified columns (`username`, `website`, and `visible_email`) from the `users` table. Each `DROP COLUMN` clause removes a column and any data it contains.

#### Side Effects
- **Data Loss**: This operation permanently removes the `username`, `website`, and `visible_email` columns and all associated data from the `users` table.
