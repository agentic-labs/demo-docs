## High-level description
This YAML file defines a Helm chart for deploying the "Trieve" application. It specifies the application version, dependencies on other charts (Redis, PostgreSQL, Qdrant), and their respective versions. The dependencies can be conditionally enabled based on configuration settings.

## Code Structure
This code defines a Helm chart with metadata and dependencies. The dependencies are conditional and rely on configuration settings.

#### `dependencies.name`
##### Description
The name of the dependent Helm chart.

#### `dependencies.repository`
##### Description
The repository URL where the dependent chart can be found.

#### `dependencies.version`
##### Description
The specific version of the dependent chart to use.

#### `dependencies.condition`
##### Description
A configuration expression that determines whether the dependency is included. 
* `config.redis.useSubchart`: Include the Redis chart if this configuration option is true.
* `postgres.useSubchart`: Include the PostgreSQL chart if this configuration option is true.
* `config.qdrant.useSubchart`: Include the Qdrant chart if this configuration option is true.
