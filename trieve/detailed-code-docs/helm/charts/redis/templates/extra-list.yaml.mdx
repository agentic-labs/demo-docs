## High-level description
This Kubernetes Helm template file iterates through a list of extra deployments specified in the `extraDeploy` value and renders each deployment using a common template. This allows for flexible customization of the Redis deployment by adding extra resources.

## Code Structure
The code uses a single Helm range to iterate over the `extraDeploy` list. For each item in the list, it calls the `common.tplvalues.render` template, passing the current item and the global context.

## References
- `common.tplvalues.render`: This template is assumed to be defined elsewhere and is responsible for rendering the actual deployment resources based on the provided values.

## Symbols
### `range .Values.extraDeploy`
#### Description
This is a Helm template directive that iterates over the `extraDeploy` list defined in the `Values` object. The `Values` object typically holds configuration values passed to the Helm chart.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| `.Values.extraDeploy` | list | A list of extra deployment configurations. Each item in the list is expected to be a dictionary or object that can be used by the `common.tplvalues.render` template. |

#### Outputs
This directive doesn't have a direct output. Instead, it iterates over the input list and executes the code within the block (in this case, the `include` directive) for each item.

### `include "common.tplvalues.render" (dict "value" . "context" $)`
#### Description
This directive includes and renders another Helm template named `common.tplvalues.render`. It passes two arguments to the template:

- `value`: The current item from the `extraDeploy` list being iterated over.
- `context`: The global context of the Helm chart, represented by the `$` variable.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| `"common.tplvalues.render"` | string | The name of the template to include and render. |
| `(dict "value" . "context" $)` | dict | A dictionary containing two keys: "value" set to the current item from the `extraDeploy` list and "context" set to the global Helm context. |

#### Outputs
The output of this directive is the rendered output of the `common.tplvalues.render` template, which is expected to define the actual Kubernetes resources for the extra deployment.
