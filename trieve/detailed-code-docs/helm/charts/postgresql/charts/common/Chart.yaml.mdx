## High-level description
This file, `Chart.yaml`, is a Helm chart descriptor for a library chart named "common" within the `postgresql` chart. This library chart is intended to be a repository for shared logic and functionalities used across various Bitnami charts, and is not meant to be deployed independently.

## Symbols
### `annotations`
#### Description
Defines annotations providing metadata about the chart.

#### Internal Logic
- `category: Infrastructure`: Categorizes the chart as infrastructure-related.
- `licenses: Apache-2.0`: Specifies the chart's licensing as Apache-2.0.

### `apiVersion`
#### Description
Specifies the Helm chart API version, which is `v2` in this case.

### `appVersion`
#### Description
Indicates the application version associated with the chart, which is `2.19.1`. This likely refers to the version of the common logic and functionalities provided by the library chart.

### `description`
#### Description
Provides a brief description of the chart's purpose. It clarifies that this is a library chart containing common logic for Bitnami charts and is not meant for standalone deployment.

### `home`
#### Description
Specifies the homepage URL associated with the chart, pointing to `https://bitnami.com`.

### `icon`
#### Description
Provides a URL to an icon representing the chart, linking to the Bitnami logo.

### `keywords`
#### Description
Lists keywords relevant to the chart, aiding in discoverability and categorization. The keywords highlight the chart's nature as a library providing helper functions, templates, and common logic.

### `maintainers`
#### Description
Lists maintainers of the chart, including their names and contact information. In this case, it indicates that the chart is maintained by VMware, Inc., with a link to their GitHub repository.

### `name`
#### Description
Specifies the name of the chart, which is "common".

### `sources`
#### Description
Provides a list of source code repositories for the chart. Here, it points to the Bitnami charts GitHub repository.

### `type`
#### Description
Indicates the type of the chart, which is "library". This signifies that the chart is not a deployable application but a collection of reusable components.

### `version`
#### Description
Specifies the version of the Helm chart itself, which is `2.19.1`. This versioning is independent of the application version and helps track changes specific to the chart's structure and definitions.
