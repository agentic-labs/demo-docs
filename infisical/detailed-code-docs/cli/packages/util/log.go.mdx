## High-level description
This Go package, `util`, provides utility functions for error handling, logging, and printing formatted messages to the console. It includes functions for handling errors, printing warnings and success messages, and exiting the program with specific error codes.

## Symbols

### `HandleError`
#### Description
This function handles an error by printing it and any additional messages, then exits the program with a status code of 1.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| err | error | The error to be handled |
| messages | ...string | Optional additional messages to be printed |

#### Internal Logic
It calls `PrintErrorAndExit` with an exit code of 1, passing the error and messages.

### `PrintErrorAndExit`
#### Description
This function prints an error, optional messages, a support message, and then exits the program with the specified exit code.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| exitCode | int | The exit code to use when terminating the program |
| err | error | The error to be printed |
| messages | ...string | Optional additional messages to be printed |

#### Internal Logic
1. Prints the error using `printError`
2. Prints any additional messages if provided
3. Prints a support message with a link to Infisical's Slack
4. Exits the program with the specified exit code

### `PrintWarning`
#### Description
Prints a warning message to stderr in yellow color.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| message | string | The warning message to be printed |

### `PrintSuccessMessage`
#### Description
Prints a success message to stdout in green color.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| message | string | The success message to be printed |

### `PrintErrorMessageAndExit`
#### Description
Prints error messages to stderr and exits the program with a status code of 1.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| messages | ...string | Error messages to be printed |

#### Internal Logic
1. Prints all provided messages to stderr
2. Exits the program with a status code of 1

### `printError`
#### Description
A helper function that prints an error message to stderr in red color.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| e | error | The error to be printed |

## Error Handling
This package is primarily focused on error handling and reporting. It provides various functions to print errors, warnings, and success messages with appropriate formatting and colors. The main error handling mechanisms include:

1. Printing errors in red color to stderr
2. Optionally printing additional messages along with errors
3. Exiting the program with specific exit codes after reporting errors
4. Providing a support message with a link to Infisical's Slack for persistent issues

## Logging
While this package doesn't implement a full-fledged logging system, it provides basic logging functionality through colored console output:

- Errors are printed in red to stderr
- Warnings are printed in yellow to stderr
- Success messages are printed in green to stdout

These functions can be used throughout the application to provide consistent and visually distinct log messages for different types of information.
