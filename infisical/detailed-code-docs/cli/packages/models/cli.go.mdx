## High-level description
This file defines various data structures and types used in the Infisical CLI (Command Line Interface) application. It includes models for user credentials, configuration files, environment variables, workspaces, and various parameters for different operations related to secret management and folder operations.

## Symbols

### UserCredentials
#### Description
Represents the credentials of a user, including email, private key, JWT token, and refresh token.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| Email | string | User's email address |
| PrivateKey | string | User's private key |
| JTWToken | string | JSON Web Token for authentication |
| RefreshToken | string | Token used to refresh the JWT |
