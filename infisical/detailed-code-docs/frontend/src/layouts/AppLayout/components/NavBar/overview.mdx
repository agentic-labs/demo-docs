## High-level description
The `Navbar` component is a React functional component that renders the navigation bar for the main application. It includes support options, user menu (billing, logout, org/user settings), and organization-related functionality. The component uses various hooks and context providers to manage state and perform actions.

## Code Structure
The `Navbar` component is the main export of this file. It uses several hooks and context providers to access user data, organization data, and perform actions like logout. The component renders a complex UI with dropdown menus for support options and user actions.

## Symbols

### `Navbar`
#### Description
The main component that renders the navigation bar for the application.

#### Inputs
This component doesn't accept any props directly.

#### Internal Logic
1. Uses various hooks to access user data, organization data, and subscription information.
2. Renders the Infisical logo and brand name.
3. Displays support options in a dropdown menu.
4. Shows user information and organization details in another dropdown menu.
5. Provides options for user settings, organization settings, and logout.
6. Displays a banner for users on the Starter plan to upgrade to Pro Free Trial.

#### Side Effects
- Interacts with local storage to remove user-related data on logout.
- Navigates to different pages using Next.js router.

### `supportOptions`
#### Description
A function that returns an array of support options with icons, text, and URLs.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| t | TFunction | Translation function from i18next |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| supportOptions | Array | Array of support options with icons, text, and URLs |
