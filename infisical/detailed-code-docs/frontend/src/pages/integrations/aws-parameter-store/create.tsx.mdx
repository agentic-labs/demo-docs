## High-level description
This code defines a React component for creating an AWS Parameter Store integration in an Infisical project. It allows users to configure connection details, options, and create the integration.

## Code Structure
The main component `AWSParameterStoreCreateIntegrationPage` uses various hooks and components to render a form for creating the integration. It manages state for form inputs, handles form submission, and displays different sections of the form using tabs.

## Symbols

### AWSParameterStoreCreateIntegrationPage
#### Description
This is the main functional component that renders the AWS Parameter Store integration creation page.

#### Internal Logic
1. Initializes state variables for form inputs and loading states.
2. Fetches necessary data using custom hooks (e.g., `useGetWorkspaceById`, `useGetIntegrationAuthById`).
3. Renders a form with two tabs: Connection and Options.
4. Handles form submission and validation.
5. Creates the integration using the `useCreateIntegration` hook when the form is submitted.

### TabSections (enum)
#### Description
An enum defining the tab sections of the form.

| Name | Type | Description |
|:-----|:-----|:------------|
| Connection | string | Represents the connection tab |
| Options | string | Represents the options tab |

### awsRegions (constant)
#### Description
An array of objects containing AWS region information.

### handleButtonClick (function)
#### Description
Handles the form submission when the "Create Integration" button is clicked.

#### Internal Logic
1. Validates the input path.
2. Prepares the integration data.
3. Calls the `mutateAsync` function to create the integration.
4. Redirects to the integrations page on success.

### isValidAWSParameterStorePath (function)
#### Description
Validates if the given path is a valid AWS Parameter Store path.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| awsStorePath | string | The path to validate |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| isValid | boolean | Whether the path is valid |
