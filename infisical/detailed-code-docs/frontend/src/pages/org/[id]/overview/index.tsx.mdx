## High-level description
This file contains the React component for the organization overview page. It displays projects, onboarding guides, and various features of Infisical. The component handles project creation, user actions, and provides a comprehensive view of the organization's projects and resources.

## Code Structure
The main component `OrganizationPage` uses various hooks and context providers to fetch and manage data. It renders different sections like projects, onboarding guide, and features. The component also includes modals for adding new projects and upgrading plans.

## Symbols

### `OrganizationPage`
#### Description
The main component that renders the organization overview page.

#### Internal Logic
1. Fetches organization and workspace data
2. Renders projects in grid or list view
3. Displays onboarding guide and features
4. Handles project creation and user actions

### `LearningItem` and `LearningItemSquare`
#### Description
Components that render individual learning items in the onboarding guide.

#### Props
| Name | Type | Description |
|:-----|:-----|:------------|
| text | string | Main text of the learning item |
| subText | string | Subtext of the learning item |
| complete | boolean | Whether the item is completed |
| icon | IconProp | Icon to display |
| time | string | Estimated time for the item |
| userAction | string | Action to register when clicked |
| link | string | Link to navigate to when clicked |

### `TabsObject`
#### Description
Component that renders a tabbed interface for different platform instructions.

### `CodeItem`
#### Description
Component that renders a code snippet with a copy button.

#### Props
| Name | Type | Description |
|:-----|:-----|:------------|
| isCopied | boolean | Whether the code has been copied |
| setIsCopied | function | Function to set the copied state |
| textExplanation | string | Explanation text for the code |
| code | string | The code snippet |
| id | string | Unique identifier for the code item |

### `copyToClipboard`
#### Description
Function to copy text to clipboard.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | ID of the element to copy from |
| setState | function | Function to update the copied state |

## Dependencies
