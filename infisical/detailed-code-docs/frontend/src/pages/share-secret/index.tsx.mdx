## High-level description
This code defines a Next.js page component for sharing secrets securely using Infisical. It sets up the page's metadata, including the title and favicon, and renders the `ShareSecretPublicPage` component within a dark-themed container.

## Symbols

### ShareNewPublicSecretPage
#### Description
This is the main functional component that represents the page for sharing new public secrets. It uses Next.js's `Head` component to set metadata and renders the `ShareSecretPublicPage` component.

#### Inputs
This component doesn't take any inputs (props).

#### Outputs
Returns a React element that includes the page's head metadata and the `ShareSecretPublicPage` component.

#### Internal Logic
1. Sets up the page's metadata using Next.js `Head` component:
   - Sets the page title
   - Sets the favicon
   - Sets Open Graph (og) metadata for social media sharing
2. Renders the `ShareSecretPublicPage` component within a dark-themed div

## References

- `ShareSecretPublicPage` from "@app/views/ShareSecretPublicPage"

The component sets a custom property:

| Option | Type | Default | Description |
