## High-level description
The `useToggle` hook is a custom React hook that provides a boolean state and a set of functions to manipulate that state. It offers a convenient way to toggle boolean values and perform timed toggles in React components.

## Code Structure
The `useToggle` hook is defined as a function that returns a tuple containing the current boolean state and an object with functions to manipulate that state. It uses React's `useState` and `useCallback` hooks internally to manage the state and memoize the functions.

## Symbols

### useToggle
#### Description
A custom React hook that manages a boolean state and provides functions to manipulate it.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| initialState | boolean | The initial state of the toggle (default: false) |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| value | boolean | The current state of the toggle |
| actions | object | An object containing functions to manipulate the toggle state |

#### Internal Logic
1. Creates a state variable `value` using `useState` with the provided `initialState`.
2. Defines four memoized functions using `useCallback`:
   - `on`: Sets the state to true
   - `off`: Sets the state to false
   - `toggle`: Toggles the state or sets it to a specific boolean value
   - `timedToggle`: Toggles the state and then sets it back to false after a specified timeout

#### Performance Considerations
The use of `useCallback` for the action functions ensures that they are not recreated on every render, which can help optimize performance in some scenarios.
