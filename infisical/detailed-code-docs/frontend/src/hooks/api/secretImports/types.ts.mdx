## High-level description
This TypeScript file defines various types and interfaces related to secret imports in a project management system. It includes types for secret import objects, API request parameters, and data transfer objects (DTOs) for different operations on secret imports.

## Code Structure
The file consists of multiple type definitions, each representing a specific aspect of secret import functionality. These types are likely used across the application for type checking and ensuring consistency in data structures.

## Symbols

### `TSecretImport`
#### Description
Defines the structure of a secret import object.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the secret import |
| folderId | string | Identifier of the folder containing the secret |
| importPath | string | Path of the imported secret |
| importEnv | WorkspaceEnv | Environment of the imported secret |
| position | string | Position of the secret import |
| createdAt | string | Timestamp of creation |
| updatedAt | string | Timestamp of last update |
| isReserved | boolean (optional) | Indicates if the secret is reserved |
| isReplication | boolean (optional) | Indicates if the secret is a replication |
| isReplicationSuccess | boolean (optional) | Indicates if replication was successful |
| replicationStatus | string (optional) | Status of replication |
| lastReplicated | string (optional) | Timestamp of last replication |
