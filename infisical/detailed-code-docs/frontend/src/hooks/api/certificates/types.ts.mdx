## High-level description
This TypeScript file defines types and interfaces related to certificates and certificate operations within the application. It includes types for certificate data structures and data transfer objects (DTOs) for delete and revoke operations.

## Symbols

### `TCertificate`
#### Description
Defines the structure of a certificate object.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the certificate |
| caId | string | Identifier for the Certificate Authority |
| certificateTemplateId | string (optional) | Identifier for the certificate template |
| status | CertStatus | Current status of the certificate |
| friendlyName | string | Human-readable name for the certificate |
| commonName | string | Common Name (CN) of the certificate |
| altNames | string | Alternative names for the certificate |
| serialNumber | string | Unique serial number of the certificate |
| notBefore | string | Start date of the certificate's validity period |
| notAfter | string | End date of the certificate's validity period |

### `TDeleteCertDTO`
#### Description
Data Transfer Object for deleting a certificate.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| projectSlug | string | Identifier for the project |
| serialNumber | string | Serial number of the certificate to be deleted |

### `TRevokeCertDTO`
#### Description
Data Transfer Object for revoking a certificate.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| projectSlug | string | Identifier for the project |
| serialNumber | string | Serial number of the certificate to be revoked |
| revocationReason | string | Reason for revoking the certificate |

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| CertStatus | Enum imported from "./enums" to define certificate status |
