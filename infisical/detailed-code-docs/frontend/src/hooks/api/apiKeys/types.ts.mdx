## High-level description
This TypeScript file defines types and interfaces related to API key management in a frontend application. It includes types for API key data, creation, updating, and deletion operations, providing a structured way to handle API key-related data and operations.

## Symbols

### `APIKeyDataV2`
#### Description
Defines the structure of an API key data object.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the API key |
| name | string | Name of the API key |
| user | string | User associated with the API key |
| lastUsed | string (optional) | Timestamp of last usage |
| usageCount | number | Number of times the API key has been used |
| createdAt | string | Timestamp of creation |
| updatedAt | string | Timestamp of last update |

### `CreateAPIKeyDataV2DTO`
#### Description
Defines the data transfer object for creating a new API key.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| name | string | Name of the new API key |

### `CreateServiceTokenDataV3Res`
#### Description
Defines the response structure for creating a new API key (service token).

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| apiKeyData | APIKeyDataV2 | Data of the created API key |
| apiKey | string | The actual API key string |

### `UpdateAPIKeyDataV2DTO`
#### Description
Defines the data transfer object for updating an existing API key.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| apiKeyDataId | string | ID of the API key to update |
| name | string | New name for the API key |

### `DeleteAPIKeyDataV2DTO`
#### Description
Defines the data transfer object for deleting an API key.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| apiKeyDataId | string | ID of the API key to delete |

## References
These types are used in the related file `frontend/src/hooks/api/apiKeys/queries.tsx`, which implements API operations for creating, updating, and deleting API keys using these type definitions.

## Dependencies
This file doesn't have any external dependencies. It only uses TypeScript's built-in types.
