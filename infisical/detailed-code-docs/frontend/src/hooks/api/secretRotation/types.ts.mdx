## High-level description
This TypeScript file defines types and enums related to secret rotation functionality in a frontend application. It includes definitions for provider functions, templates, and data transfer objects (DTOs) used in API interactions for managing secret rotations.

## Code Structure
The file defines several interconnected types and enums that are used to represent different aspects of secret rotation. The main types include `TSecretRotationProviderTemplate`, `THttpProviderTemplate`, `TDbProviderTemplate`, and `TSecretRotation`. These types are used in various DTOs for API interactions.
