## High-level description
This TypeScript file defines types and enums related to secret approval requests in a frontend application. It includes definitions for approval statuses, commit types, secret changes, and various data structures used in the secret approval process.

## Symbols

### `ApprovalStatus` (enum)
#### Description
Defines the possible statuses for a secret approval request.

#### Values
- `PENDING`: Represents a pending approval status
- `APPROVED`: Represents an approved status
- `REJECTED`: Represents a rejected status

### `CommitType` (enum)
#### Description
Defines the types of changes that can be made to a secret.

#### Values
- `DELETE`: Represents a deletion operation
- `UPDATE`: Represents an update operation
- `CREATE`: Represents a creation operation

### `TSecretApprovalSecChangeData` (type)
#### Description
Defines the structure for secret change data in an approval request.

#### Properties
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the secret |
| secretKey | string | Key of the secret |
| secretValue | string (optional) | Value of the secret |
| secretComment | string (optional) | Comment associated with the secret |
| skipMultilineEncoding | boolean (optional) | Flag to skip multiline encoding |
| algorithm | "aes-256-gcm" | Encryption algorithm used |
| keyEncoding | "utf8" \| "base64" | Encoding used for the key |
| tags | WsTag[] (optional) | Array of tags associated with the secret |
