## High-level description
This TypeScript file defines types and interfaces related to secret snapshots in a workspace environment. It provides structures for representing secret snapshots, snapshot data, and various data transfer objects (DTOs) used in API requests and responses.

## Symbols

### `TSecretSnapshot`
#### Description
Defines the structure of a secret snapshot.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the snapshot |
| workspace | string | Identifier for the workspace |
| secretVersions | string[] | Array of secret version identifiers |
| createdAt | string | Timestamp of creation |
| updatedAt | string | Timestamp of last update |

### `TSnapshotData`
#### Description
Extends `TSecretSnapshot` with additional data, replacing `secretVersions` with more detailed information.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the snapshot |
| secretVersions | SecretVersions[] | Array of detailed secret versions |
| folderVersion | Array&lt;{ name: string; id: string }&gt; | Array of folder information |
| environment | WorkspaceEnv | Environment information |

### `TSnapshotDataProps`
#### Description
Defines properties for retrieving snapshot data.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| snapshotId | string | Identifier for the snapshot |
| env | string | Environment identifier |

### `TGetSecretSnapshotsDTO`
#### Description
Data transfer object for retrieving secret snapshots.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| workspaceId | string | Identifier for the workspace |
| limit | number | Number of snapshots to retrieve |
| environment | string | Environment identifier |
| directory? | string | Optional directory path |

### `TSecretRollbackDTO`
#### Description
Data transfer object for rolling back to a specific snapshot.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| snapshotId | string | Identifier for the snapshot to rollback to |
| workspaceId | string | Identifier for the workspace |
| environment | string | Environment identifier |
| directory? | string | Optional directory path |

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| SecretVersions | Imported from "../secrets/types" for use in TSnapshotData |
| WorkspaceEnv | Imported from "../types" for use in TSnapshotData |

## References
This file is referenced by and used in conjunction with `frontend/src/hooks/api/secretSnapshots/queries.tsx`, which implements API queries and mutations using the types defined here.

The types defined in this file are crucial for maintaining type safety and structure when working with secret snapshots throughout the application, especially in API interactions and state management.
