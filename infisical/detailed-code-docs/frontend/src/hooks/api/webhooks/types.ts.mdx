## High-level description
This TypeScript file defines types and interfaces related to webhooks in the application. It includes an enum for webhook types, a main webhook type, and several Data Transfer Objects (DTOs) for creating, updating, deleting, and testing webhooks.

## Symbols

### `WebhookType`
#### Description
An enum that defines the types of webhooks supported by the application.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| GENERAL | string | Represents a general webhook type |
| SLACK | string | Represents a Slack-specific webhook type |

___

### `TWebhook`
#### Description
Defines the structure of a webhook object in the application.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the webhook |
| type | WebhookType | Type of the webhook (GENERAL or SLACK) |
| projectId | string | ID of the project associated with the webhook |
| environment | object | Details of the environment associated with the webhook |
| envId | string | ID of the environment |
| secretPath | string | Path to the secret associated with the webhook |
| url | string | URL of the webhook |
| lastStatus | string | Status of the last webhook execution ("success" or "failed") |
| lastRunErrorMessage | string (optional) | Error message from the last failed run |
| isDisabled | boolean | Indicates if the webhook is disabled |
| createdAt | string | Timestamp of webhook creation |
| updatedAt | string | Timestamp of last webhook update |

___

### `TCreateWebhookDto`
#### Description
Data Transfer Object for creating a new webhook.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| workspaceId | string | ID of the workspace where the webhook will be created |
| environment | string | Environment for the webhook |
| webhookUrl | string | URL of the webhook |
| webhookSecretKey | string (optional) | Secret key for the webhook |
| secretPath | string | Path to the secret associated with the webhook |
| type | WebhookType | Type of the webhook (GENERAL or SLACK) |

___

### `TUpdateWebhookDto`
#### Description
Data Transfer Object for updating an existing webhook.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| webhookId | string | ID of the webhook to be updated |
| workspaceId | string | ID of the workspace containing the webhook |
| isDisabled | boolean (optional) | New disabled status for the webhook |

___

### `TDeleteWebhookDto`
#### Description
Data Transfer Object for deleting a webhook.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| webhookId | string | ID of the webhook to be deleted |
| workspaceId | string | ID of the workspace containing the webhook |

___

### `TTestWebhookDTO`
#### Description
Data Transfer Object for testing a webhook.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| webhookId | string | ID of the webhook to be tested |
| workspaceId | string | ID of the workspace containing the webhook |
