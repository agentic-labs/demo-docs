## High-level description
This code defines types and enums related to dynamic secrets in a TypeScript-based frontend application. It includes definitions for dynamic secret statuses, providers, and various data transfer objects (DTOs) used for creating, updating, listing, and deleting dynamic secrets.

## Symbols

### `DynamicSecretStatus` (enum)
#### Description
Defines the possible statuses of a dynamic secret.

#### Enum Values
| Name | Value |
|:-----|:------|
| Deleting | "Revocation in process" |
| FailedDeletion | "Failed to delete" |

### `TDynamicSecret` (type)
#### Description
Represents the structure of a dynamic secret.

#### Properties
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the dynamic secret |
| name | string | Name of the dynamic secret |
| type | DynamicSecretProviders | Type of dynamic secret provider |
| createdAt | string | Creation timestamp |
| updatedAt | string | Last update timestamp |
| defaultTTL | string | Default Time-To-Live for the secret |
| status | DynamicSecretStatus (optional) | Current status of the dynamic secret |
| statusDetails | string (optional) | Additional details about the status |
| maxTTL | string | Maximum Time-To-Live for the secret |

### `DynamicSecretProviders` (enum)
#### Description
Defines the types of dynamic secret providers supported.

#### Enum Values
| Name | Value |
|:-----|:------|
| SqlDatabase | "sql-database" |
| Cassandra | "cassandra" |
| AwsIam | "aws-iam" |

### `SqlProviders` (enum)
#### Description
Defines the types of SQL database providers supported.

#### Enum Values
| Name | Value |
|:-----|:------|
| Postgres | "postgres" |
| MySql | "mysql2" |
| Oracle | "oracledb" |
| MsSQL | "mssql" |

### `TDynamicSecretProvider` (type)
#### Description
Represents the configuration for different types of dynamic secret providers.

#### Variants
1. SQL Database Provider
2. Cassandra Provider
3. AWS IAM Provider

Each variant has its own set of input properties specific to the provider type.

### `TCreateDynamicSecretDTO` (type)
#### Description
Data transfer object for creating a new dynamic secret.

### `TUpdateDynamicSecretDTO` (type)
#### Description
Data transfer object for updating an existing dynamic secret.

### `TListDynamicSecretDTO` (type)
#### Description
Data transfer object for listing dynamic secrets.

### `TDeleteDynamicSecretDTO` (type)
#### Description
Data transfer object for deleting a dynamic secret.

### `TDetailsDynamicSecretDTO` (type)
#### Description
Data transfer object for retrieving details of a specific dynamic secret.

### `TGetDynamicSecretsByEnvsDTO` (type)
#### Description
Data transfer object for retrieving dynamic secrets across multiple environments.
