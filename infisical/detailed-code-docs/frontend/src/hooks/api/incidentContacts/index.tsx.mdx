## High-level description
This code exports three custom hooks for managing incident contacts in an organization. These hooks provide functionality for adding, deleting, and retrieving incident contacts using React Query for efficient data fetching and state management.

## Code Structure
The file acts as a central export point for the custom hooks defined in the "queries.tsx" file. It re-exports the hooks, making them easily accessible to other parts of the application.

## References
This code references the following symbols from the "queries.tsx" file:
- `useAddIncidentContact`
- `useDeleteIncidentContact`
- `useGetOrgIncidentContact`

## Symbols

### `useAddIncidentContact`
#### Description
A custom hook that provides functionality to add a new incident contact to an organization.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| orgId | string | The organization ID |
| email | string | The email of the contact to be added |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| mutate | function | A function to trigger the mutation |
| isLoading | boolean | Indicates if the mutation is in progress |
| isError | boolean | Indicates if an error occurred |
| error | Error | The error object if an error occurred |

#### Internal Logic
Uses `useMutation` from React Query to handle the API request for adding a new incident contact. On success, it invalidates the related query to refetch the updated list of contacts.

### `useDeleteIncidentContact`
#### Description
A custom hook that provides functionality to delete an existing incident contact from an organization.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| orgId | string | The organization ID |
| incidentContactId | string | The ID of the contact to be deleted |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| mutate | function | A function to trigger the mutation |
| isLoading | boolean | Indicates if the mutation is in progress |
| isError | boolean | Indicates if an error occurred |
| error | Error | The error object if an error occurred |

#### Internal Logic
Uses `useMutation` from React Query to handle the API request for deleting an incident contact. On success, it invalidates the related query to refetch the updated list of contacts.

### `useGetOrgIncidentContact`
#### Description
A custom hook that fetches and provides the list of incident contacts for a given organization.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| orgId | string | The organization ID |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| data | IncidentContact[] | The list of incident contacts |
| isLoading | boolean | Indicates if the data is being fetched |
| isError | boolean | Indicates if an error occurred |
| error | Error | The error object if an error occurred |

#### Internal Logic
Uses `useQuery` from React Query to fetch the list of incident contacts for the given organization. The query is only enabled when `orgId` is provided.
