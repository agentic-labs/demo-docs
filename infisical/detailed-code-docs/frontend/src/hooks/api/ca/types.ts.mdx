## High-level description
This TypeScript file defines various types and interfaces related to Certificate Authorities (CAs) and certificate management. It provides type definitions for CA-related data structures, request payloads, and response formats used in the application's API interactions.

## Symbols

### `TCertificateAuthority`
#### Description
Defines the structure of a Certificate Authority object.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the CA |
| parentCaId | string (optional) | ID of the parent CA, if applicable |
| projectId | string | ID of the project the CA belongs to |
| type | CaType | Type of the CA |
| status | CaStatus | Current status of the CA |
| friendlyName | string | Human-readable name for the CA |
| organization | string | Organization name |
| ou | string | Organizational Unit |
| country | string | Country code |
| province | string | Province or state |
| locality | string | City or locality |
| commonName | string | Common Name (CN) for the CA |
| dn | string | Distinguished Name |
| maxPathLength | number (optional) | Maximum path length for certificate chain |
| notAfter | string (optional) | Expiration date of the CA certificate |
| notBefore | string (optional) | Start date of the CA certificate |
| keyAlgorithm | CertKeyAlgorithm | Algorithm used for the CA's key |
| activeCaCertId | string (optional) | ID of the active CA certificate |
| createdAt | string | Creation timestamp |
| updatedAt | string | Last update timestamp |
