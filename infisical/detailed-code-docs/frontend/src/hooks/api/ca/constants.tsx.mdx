This file defines constants and utility functions related to Certificate Authority (CA) types and statuses. It provides mappings for enum values to human-readable names and a function to determine the appropriate badge variant based on CA status.

### `getCaStatusBadgeVariant`
#### Description
A function that determines the appropriate badge variant based on the given CA status.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| status | CaStatus | The status of the CA |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| variant | string | The badge variant corresponding to the CA status |

#### Internal Logic
The function uses a switch statement to determine the badge variant:
- Returns "success" for `CaStatus.ACTIVE`
- Returns "danger" for `CaStatus.DISABLED`
- Returns "primary" for any other status (default case)
