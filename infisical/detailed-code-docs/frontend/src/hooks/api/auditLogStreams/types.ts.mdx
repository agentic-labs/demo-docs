## High-level description
This TypeScript file defines types related to audit log streams. It includes types for log stream headers, audit log streams, and data transfer objects (DTOs) for creating, updating, and deleting audit log streams.

## Symbols

### `LogStreamHeaders`
#### Description
Defines the structure for headers associated with log streams.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| key | string | The key of the header |
| value | string | The value of the header |

### `TAuditLogStream`
#### Description
Represents the structure of an audit log stream.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the log stream |
| url | string | URL associated with the log stream |
| headers | LogStreamHeaders[] | Optional array of headers for the log stream |

### `TCreateAuditLogStreamDTO`
#### Description
Data Transfer Object for creating a new audit log stream.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| url | string | URL for the new log stream |
| headers | LogStreamHeaders[] | Optional array of headers for the log stream |
| orgId | string | Identifier for the organization |

### `TUpdateAuditLogStreamDTO`
#### Description
Data Transfer Object for updating an existing audit log stream.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier of the log stream to update |
| url | string | Optional new URL for the log stream |
| headers | LogStreamHeaders[] | Optional new array of headers for the log stream |
| orgId | string | Identifier for the organization |

### `TDeleteAuditLogStreamDTO`
#### Description
Data Transfer Object for deleting an audit log stream.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier of the log stream to delete |
| orgId | string | Identifier for the organization |

## Dependencies
This file does not have any external dependencies. It only uses TypeScript's built-in type system.
