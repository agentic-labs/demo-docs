## High-level description
This code defines a set of React Query hooks for managing identity-related API requests in a frontend application. It provides functions for fetching, creating, updating, and deleting various types of identity authentication methods, such as Universal Auth, GCP Auth, AWS Auth, Azure Auth, Kubernetes Auth, OIDC Auth, and Token Auth.

## Code Structure
The code is organized into a series of custom hooks, each corresponding to a specific API operation. These hooks use the `useMutation` and `useQuery` functions from React Query to manage API requests and cache invalidation. The `identitiesKeys` object defines query keys for different identity-related queries.
