## High-level description
This TypeScript file defines types and enums related to dynamic secret leases in a frontend application. It provides type definitions for various data transfer objects (DTOs) used in API operations such as creating, renewing, listing, and revoking dynamic secret leases.

## Symbols

### `DynamicSecretLeaseStatus`
#### Description
An enum that defines possible statuses for a dynamic secret lease.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| FailedDeletion | string | Represents a status where the lease failed to delete |

### `TDynamicSecretLease`
#### Description
A type definition for a dynamic secret lease object.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the lease |
| version | number | Version of the lease |
| expireAt | string | Expiration date/time of the lease |
| dynamicSecretId | string | Identifier of the associated dynamic secret |
| status | DynamicSecretLeaseStatus (optional) | Current status of the lease |
| statusDetails | string (optional) | Additional details about the lease status |
| createdAt | string | Creation date/time of the lease |
| updatedAt | string | Last update date/time of the lease |

### `TCreateDynamicSecretLeaseDTO`
#### Description
A type definition for the data transfer object used when creating a new dynamic secret lease.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| dynamicSecretName | string | Name of the dynamic secret |
| projectSlug | string | Slug of the project |
| ttl | string (optional) | Time-to-live for the lease |
| path | string | Path associated with the lease |
| environmentSlug | string | Slug of the environment |

### `TRenewDynamicSecretLeaseDTO`
#### Description
A type definition for the data transfer object used when renewing an existing dynamic secret lease.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| leaseId | string | Identifier of the lease to renew |
| dynamicSecretName | string | Name of the dynamic secret |
| ttl | string (optional) | New time-to-live for the lease |
| projectSlug | string | Slug of the project |
| path | string | Path associated with the lease |
| environmentSlug | string | Slug of the environment |

### `TListDynamicSecretLeaseDTO`
#### Description
A type definition for the data transfer object used when listing dynamic secret leases.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| dynamicSecretName | string | Name of the dynamic secret |
| projectSlug | string | Slug of the project |
| path | string | Path associated with the lease |
| environmentSlug | string | Slug of the environment |
| enabled | boolean (optional) | Flag to filter enabled/disabled leases |

### `TRevokeDynamicSecretLeaseDTO`
#### Description
A type definition for the data transfer object used when revoking a dynamic secret lease.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| leaseId | string | Identifier of the lease to revoke |
| dynamicSecretName | string | Name of the dynamic secret |
| projectSlug | string | Slug of the project |
| path | string | Path associated with the lease |
| environmentSlug | string | Slug of the environment |
| isForced | boolean (optional) | Flag to indicate if the revocation should be forced |

## References
The types defined in this file are used in the related file `frontend/src/hooks/api/dynamicSecretLease/mutation.ts`, which implements API mutation hooks for creating, renewing, and revoking dynamic secret leases.
