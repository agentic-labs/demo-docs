## High-level description
This TypeScript file defines various types and interfaces related to organization management, billing, and product plans. It provides a structured representation of data used in the frontend for handling organization-related operations, billing information, and product comparisons.

## Symbols

### `Organization`
#### Description
Defines the structure of an organization entity.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the organization |
| name | string | Name of the organization |
| createAt | string | Timestamp of when the organization was created |
| updatedAt | string | Timestamp of the last update to the organization |
| authEnforced | boolean | Indicates if authentication is enforced |
| scimEnabled | boolean | Indicates if SCIM (System for Cross-domain Identity Management) is enabled |
| slug | string | URL-friendly version of the organization name |

### `UpdateOrgDTO`
#### Description

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| orgId | string | Unique identifier of the organization to update |
