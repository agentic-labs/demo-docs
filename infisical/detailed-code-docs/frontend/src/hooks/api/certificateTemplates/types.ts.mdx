## High-level description
This TypeScript file defines various types and interfaces related to certificate templates and EST (Enrollment over Secure Transport) configurations. These types are likely used for managing and interacting with certificate templates and EST configurations in a PKI (Public Key Infrastructure) system.

## Symbols

### `TCertificateTemplate`
#### Description
Defines the structure of a certificate template.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the certificate template |
| caId | string | ID of the Certificate Authority |
| caName | string | Name of the Certificate Authority |
| projectId | string | ID of the project |
| pkiCollectionId | string (optional) | ID of the PKI collection |
| name | string | Name of the certificate template |
| commonName | string | Common Name for the certificate |
| subjectAlternativeName | string | Subject Alternative Name for the certificate |
| ttl | string | Time to Live for the certificate |
