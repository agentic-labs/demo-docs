|:-----|:-----|:------------|
| projectMembershipId | string | ID of the project membership |

| createdAt | Date | Creation timestamp |
| updatedAt | Date | Last update timestamp |
| permissions | TProjectPermission[] (optional) | Array of project permissions |
| isTemporary | boolean | Flag indicating if the privilege is temporary |
| temporaryMode | string or null | Mode of temporary privilege (if applicable) |
| temporaryRange | string or null | Range of temporary privilege (if applicable) |
| temporaryAccessStartTime | string or null | Start time of temporary access (if applicable) |
