## High-level description
This file serves as an export module for SSO (Single Sign-On) configuration related hooks. It re-exports three custom hooks: `useCreateSSOConfig`, `useGetSSOConfig`, and `useUpdateSSOConfig` from a "./queries" module.

## Code Structure
