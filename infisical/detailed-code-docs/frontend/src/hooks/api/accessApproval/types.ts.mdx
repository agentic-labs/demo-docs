## High-level description
This file defines TypeScript types and interfaces for access approval policies and requests in a project management system. It includes types for access approval policies, requests, user privileges, and various DTOs (Data Transfer Objects) used in API calls related to access management.

## Symbols

### `TAccessApprovalPolicy`
#### Description
Represents an access approval policy for a project.

#### Properties
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the policy |
| name | string | Name of the policy |
| approvals | number | Number of approvals required |
| secretPath | string | Path to the secret |
| envId | string | Environment ID |
| workspace | string | Workspace identifier |
| environment | WorkspaceEnv | Environment details |
| projectId | string | Project ID |
| approvers | string[] | List of approver IDs |
| policyType | PolicyType | Type of policy |
| approversRequired | boolean | Whether approvers are required |
| enforcementLevel | EnforcementLevel | Level of enforcement |
| updatedAt | Date | Last update timestamp |
| userApprovers? | { userId: string }[] | Optional list of user approvers |
