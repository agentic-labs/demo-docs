## High-level description
This file defines TypeScript types and enums for cloud integrations and integration data used in the frontend of the application. It provides structured data types for representing cloud integration options, integration details, and synchronization behaviors.

## Symbols

### `TCloudIntegration`
#### Description
Defines the structure for cloud integration options available to users.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| name | string | Name of the cloud integration |
| slug | string | Unique identifier for the integration |
| image | string | Path to the integration's image |
| isAvailable | boolean | Indicates if the integration is available |
| type | string | Type of the integration |
| clientId | string | Client ID for authentication |
| docsLink | string | Link to the integration's documentation |
| clientSlug | string | Slug used for client identification |

### `TIntegration`
#### Description
Represents the details of an active integration instance.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the integration instance |
| isActive | boolean | Indicates if the integration is active |
| url | string (optional) | URL associated with the integration |
| app | string (optional) | Name of the app using the integration |
| appId | string (optional) | ID of the app using the integration |
| targetEnvironment | string (optional) | Target environment for the integration |
| targetEnvironmentId | string (optional) | ID of the target environment |
| targetService | string (optional) | Target service for the integration |
| targetServiceId | string (optional) | ID of the target service |
| owner | string (optional) | Owner of the integration |
| path | string (optional) | Path associated with the integration |
| region | string (optional) | Region for the integration |
| scope | string (optional) | Scope of the integration |
| integration | string | Type of integration |
| integrationAuthId | string | ID for integration authentication |
| envId | string | Environment ID |
| secretPath | string | Path for secrets |
| createdAt | string | Creation timestamp |
| updatedAt | string | Last update timestamp |
| lastUsed | string (optional) | Last usage timestamp |
| isSynced | boolean (optional) | Indicates if the integration is synced |
| syncMessage | string (optional) | Message related to synchronization |
| __v | number | Version number |
| metadata | object (optional) | Additional metadata for the integration |

### `IntegrationSyncBehavior`
#### Description
Enum defining different synchronization behaviors for integrations.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| OVERWRITE_TARGET | string | Overwrites the target with source data |
| PREFER_TARGET | string | Prefers target data over source data |
| PREFER_SOURCE | string | Prefers source data over target data |

### `IntegrationMappingBehavior`
#### Description
Enum defining different mapping behaviors for integrations.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ONE_TO_ONE | string | One-to-one mapping between source and target |
| MANY_TO_ONE | string | Many-to-one mapping from source to target |
