## High-level description
This code exports three custom hooks for managing workspace tags in a React application. These hooks provide functionality for fetching, creating, and deleting workspace tags using API requests and React Query for state management.

## Code Structure
The file `frontend/src/hooks/api/tags/index.tsx` serves as an entry point, re-exporting the hooks defined in the `queries.tsx` file. This structure allows for easier imports and better organization of the codebase.

## References
The code references the following external modules and functions:
- `@tanstack/react-query`: For managing server state in React applications
- `@app/config/request`: For making API requests
- Custom types from `./types`: `CreateTagDTO`, `DeleteTagDTO`, `UserWsTags`, `WsTag`

## Symbols

### `useGetWsTags`
#### Description
A custom hook that fetches workspace tags for a given workspace ID.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| workspaceID | string | The ID of the workspace to fetch tags for |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | UseQueryResult&lt;UserWsTags&gt; | The query result containing workspace tags data |

#### Internal Logic
- Uses `useQuery` from React Query to fetch workspace tags
- Calls `fetchWsTag` function to make the API request
- Enabled only when `workspaceID` is truthy

### `useCreateWsTag`
#### Description
A custom hook that creates a new workspace tag.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| mutationFn input | CreateTagDTO | Object containing workspaceID, tagColor, and tagSlug |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | UseMutationResult&lt;WsTag, {}, CreateTagDTO&gt; | The mutation result |

#### Internal Logic
- Uses `useMutation` from React Query to create a new tag
- Makes a POST request to the API with the provided tag data
- Invalidates the workspace tags query on success to trigger a refetch

### `useDeleteWsTag`
#### Description
A custom hook that deletes a workspace tag.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| mutationFn input | DeleteTagDTO | Object containing tagID and projectId |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | UseMutationResult&lt;WsTag, {}, DeleteTagDTO&gt; | The mutation result |

#### Internal Logic
- Uses `useMutation` from React Query to delete a tag
- Makes a DELETE request to the API with the provided tag and project IDs
- Invalidates the workspace tags query on success to trigger a refetch

## API/Interface Reference
| Endpoint | Method | Request | Response | Description |
|:---------|:-------|:--------|:---------|:------------|
| /api/v1/workspace/:workspaceID/tags | GET | - | { workspaceTags: UserWsTags } | Fetch workspace tags |
| /api/v1/workspace/:workspaceID/tags | POST | { color: string, slug: string } | { workspaceTag: WsTag } | Create a new workspace tag |
| /api/v1/workspace/:projectId/tags/:tagID | DELETE | - | { workspaceTag: WsTag } | Delete a workspace tag |
