## High-level description
This TypeScript file defines types and interfaces related to project user privileges in a frontend application. It includes definitions for project user privileges, their temporary modes, and data transfer objects (DTOs) for creating, updating, deleting, and retrieving privilege information.

## Symbols

### `ProjectUserAdditionalPrivilegeTemporaryMode`
#### Description
An enumeration defining the temporary mode for project user additional privileges.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| Relative | string | The only defined mode, with value "relative" |

### `TProjectUserPrivilege`
#### Description
A type defining the structure of a project user privilege, including both permanent and temporary privilege information.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| projectMembershipId | string | Identifier for the project membership |
| slug | string | A slug associated with the privilege |
| id | string | Unique identifier for the privilege |
| createdAt | Date | Creation timestamp of the privilege |
| updatedAt | Date | Last update timestamp of the privilege |
| permissions | TProjectPermission[] (optional) | Array of project permissions |
| isTemporary | boolean | Flag indicating if the privilege is temporary |
| temporaryMode | string (conditional) | Mode of temporary privilege (if isTemporary is true) |
| temporaryRange | string (conditional) | Range of temporary privilege (if isTemporary is true) |
| temporaryAccessStartTime | string (conditional) | Start time of temporary access (if isTemporary is true) |
| temporaryAccessEndTime | string (optional, conditional) | End time of temporary access (if isTemporary is true) |

### `TCreateProjectUserPrivilegeDTO`
#### Description
A type defining the data transfer object for creating a project user privilege.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| projectMembershipId | string | Identifier for the project membership |
| slug | string (optional) | A slug associated with the privilege |
| isTemporary | boolean (optional) | Flag indicating if the privilege is temporary |
| temporaryMode | ProjectUserAdditionalPrivilegeTemporaryMode (optional) | Mode of temporary privilege |
| temporaryRange | string (optional) | Range of temporary privilege |
| temporaryAccessStartTime | string (optional) | Start time of temporary access |
| permissions | TProjectPermission[] | Array of project permissions |

### `TUpdateProjectUserPrivlegeDTO`

#### Outputs
