## High-level description
This code defines types and enums related to roles and permissions in a project management system. It includes definitions for project membership roles, data transfer objects (DTOs) for various API operations, and structures for representing roles, permissions, and related entities.

## Symbols

### ProjectMembershipRole
#### Description
An enum that defines the possible roles a member can have in a project.

#### Inputs
None (enum definition)

#### Outputs
