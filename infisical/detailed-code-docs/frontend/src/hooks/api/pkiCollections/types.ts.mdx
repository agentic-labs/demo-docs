## High-level description
This file defines TypeScript types for PKI (Public Key Infrastructure) collections and related operations in the frontend of an application. It provides type definitions for PKI collections, their items, and various data transfer objects (DTOs) used for creating, updating, and deleting collections and items.

## Symbols

### `TPkiCollection`
#### Description
Defines the structure of a PKI collection.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the collection |
| name | string | Name of the collection |
| description | string | Description of the collection |
| projectId | string | ID of the project the collection belongs to |
| createdAt | string | Timestamp of when the collection was created |
| updatedAt | string | Timestamp of when the collection was last updated |

### `TCreatePkiCollectionDTO`
#### Description
Data transfer object for creating a new PKI collection.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| projectId | string | ID of the project to create the collection in |
| name | string | Name of the new collection |
| description | string | Description of the new collection |

### `TUpdatePkiCollectionTO`
#### Description
Data transfer object for updating an existing PKI collection.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| collectionId | string | ID of the collection to update |
| projectId | string | ID of the project the collection belongs to |
| name? | string | Optional new name for the collection |
| description? | string | Optional new description for the collection |

### `TDeletePkiCollectionDTO`
#### Description
Data transfer object for deleting a PKI collection.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| collectionId | string | ID of the collection to delete |
| projectId | string | ID of the project the collection belongs to |

### `TPkiCollectionItem`
#### Description
Defines the structure of an item within a PKI collection.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the item |
| collectionId | string | ID of the collection the item belongs to |
| type | string | Type of the item |
| itemId | string | ID of the referenced item |
| createdAt | string | Timestamp of when the item was added to the collection |
| updatedAt | string | Timestamp of when the item was last updated |

### `TAddItemToPkiCollectionDTO`
#### Description
Data transfer object for adding an item to a PKI collection.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| collectionId | string | ID of the collection to add the item to |
| type | string | Type of the item being added |
| itemId | string | ID of the item to add |

### `TRemoveItemFromPkiCollectionDTO`
#### Description
Data transfer object for removing an item from a PKI collection.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| collectionId | string | ID of the collection to remove the item from |
| itemId | string | ID of the item to remove |
