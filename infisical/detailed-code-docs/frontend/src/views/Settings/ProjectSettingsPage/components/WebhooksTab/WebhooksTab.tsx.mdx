## High-level description
This code defines a `WebhooksTab` component for managing webhooks in a project settings page. It provides functionality to view, create, test, enable/disable, and delete webhooks, with appropriate permissions checks.

## Code Structure
The `WebhooksTab` component is the main export, wrapped with project permissions. It uses various hooks for state management and API calls, and renders a table of webhooks with action buttons. It also includes modals for adding and deleting webhooks.

## Symbols

### WebhooksTab
#### Description
A React component that displays and manages webhooks for a project.

#### Internal Logic
1. Fetches webhooks for the current workspace
2. Renders a table of webhooks with their details
3. Provides buttons for creating, testing, enabling/disabling, and deleting webhooks
4. Handles webhook operations (create, test, update, delete) with appropriate API calls and notifications
5. Implements permission checks for various actions

### handleWebhookCreate
#### Description
Asynchronous function to create a new webhook.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| data | TFormSchema | Webhook data from the form |

#### Internal Logic
1. Calls the `createWebhook` API with the form data and workspace ID
2. Closes the add webhook popup on success
3. Displays a success or error notification

### handleWebhookDisable
#### Description
Asynchronous function to enable or disable a webhook.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| webhookId | string | ID of the webhook to update |
| isDisabled | boolean | New disabled state of the webhook |

#### Internal Logic
1. Calls the `updateWebhook` API with the webhook ID, workspace ID, and new disabled state
2. Displays a success or error notification

### handleWebhookDelete
#### Description
Asynchronous function to delete a webhook.

#### Internal Logic
1. Retrieves the webhook ID from the popup state
2. Calls the `deleteWebhook` API with the webhook ID and workspace ID
3. Closes the delete webhook popup on success
4. Displays a success or error notification

### handleWebhookTest
#### Description
Asynchronous function to test a webhook.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| webhookId | string | ID of the webhook to test |

#### Internal Logic
1. Calls the `testWebhook` API with the webhook ID and workspace ID
2. Displays a success or error notification
