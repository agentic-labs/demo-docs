## High-level description
This code defines a React component called `SSOModal` that manages SAML SSO configuration for an organization. It provides a form for adding or updating SSO settings, including the authentication provider, entry point, issuer, and certificate. The component also includes functionality for deleting the SSO configuration.

## Code Structure
The main component `SSOModal` uses React Hook Form for form management and Zod for schema validation. It interacts with API hooks for creating, updating, and fetching SSO configurations. The component renders different form fields based on the selected authentication provider and handles form submission and deletion actions.

## Symbols

### `SSOModal`

A React component that renders a modal for managing SAML SSO configuration.

| handlePopUpToggle | Function | Function to toggle the modal's visibility |
| hideDelete | boolean | Optional flag to hide the delete button |

#### Internal Logic
1. Fetches the current SSO configuration using `useGetSSOConfig` hook.
2. Uses React Hook Form to manage form state and validation.
3. Renders different form fields based on the selected authentication provider.
4. Handles form submission to create or update SSO configuration.
5. Provides functionality to delete the SSO configuration.

- Updates the SSO configuration in the backend.
- Invalidates related queries on successful operations.
