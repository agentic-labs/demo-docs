## High-level description
This code defines a React component called `OrgRoleTable` that displays a table of organization roles. It allows users to view, add, edit, and delete roles based on their permissions. The component uses various UI components and hooks to manage the table's state and interactions.

## Code Structure
The `OrgRoleTable` component is the main symbol in this code. It uses several hooks and components to manage its state and render the UI. The component renders a table of roles, handles role deletion, and manages modals for adding and deleting roles.
