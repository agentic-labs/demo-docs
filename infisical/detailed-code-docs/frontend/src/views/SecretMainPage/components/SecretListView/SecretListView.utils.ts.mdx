## High-level description
This file, `SecretListView.utils.ts`, contains utility functions, enums, and schemas related to managing secrets in a frontend application. It defines the structure for secret actions, form validation, and icon mappings for the secret list view.

## Symbols

### `SecretActionType`
#### Description
An enum that defines the types of actions that can be performed on a secret.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| Created | enum | Represents the creation of a secret |
| Modified | enum | Represents the modification of a secret |
| Deleted | enum | Represents the deletion of a secret |

### `formSchema`
#### Description
A Zod schema that defines the structure and validation rules for the secret form.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| formSchema | z.ZodObject | A Zod schema object defining the form fields and their validation rules |

#### Internal Logic
- Defines various fields such as `key`, `value`, `idOverride`, `valueOverride`, `overrideAction`, `comment`, `skipMultilineEncoding`, `reminderRepeatDays`, `reminderNote`, and `tags`.
- Applies validation rules and transformations to these fields.

### `TFormSchema`
#### Description
A type definition inferred from the `formSchema` Zod object.

### `FontAwesomeSpriteName`
#### Description
An enum that maps human-readable names to icon identifiers for use with Font Awesome icons.

### `FontAwesomeSpriteSymbols`
#### Description
An array of objects that maps Font Awesome icons to their corresponding `FontAwesomeSpriteName` enum values.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| FontAwesomeSpriteSymbols | Array&lt;{icon: IconDefinition, symbol: FontAwesomeSpriteName}&gt; | An array of objects mapping Font Awesome icons to sprite names |
