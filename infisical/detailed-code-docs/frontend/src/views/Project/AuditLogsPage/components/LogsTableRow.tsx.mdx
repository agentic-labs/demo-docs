## High-level description
This code defines a React component called `LogsTableRow` that renders a single row in an audit logs table. It displays information about various audit events, including the event type, actor, user agent, and event-specific metadata.

## Code Structure
The main component `LogsTableRow` uses helper functions `renderActor` and `renderMetadata` to generate the appropriate table cells based on the type of actor and event. The `formatDate` function is used to format the timestamp of the audit log entry.

#### Internal Logic
