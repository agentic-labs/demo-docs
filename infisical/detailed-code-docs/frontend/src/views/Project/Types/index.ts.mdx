## High-level description
This code serves as an export module for the `TabSections` enum and the `isTabSection` function. It centralizes the export of these types and functions related to tab sections in a project view, making them easily accessible for other parts of the application.

## Code Structure
The code imports and re-exports two symbols from the "./TabSections" module:
1. `isTabSection`: A function to validate if a given string is a valid tab section.
2. `TabSections`: An enum defining the available tab sections.

## References
This code references the "./TabSections" module, which contains the actual implementations of the exported symbols.

## Symbols

### `isTabSection`
#### Description
A function that checks if a given string value is a valid tab section.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| value | string | The string to be checked against the TabSections enum |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | boolean | True if the input is a valid tab section, false otherwise |

### `TabSections`
#### Description
An enum that defines the available tab sections in the project view.

#### Enum Values
| Name | Value | Description |
|:-----|:------|:------------|
| Member | "members" | Represents the members tab |
| Roles | "roles" | Represents the roles tab |
| Groups | "groups" | Represents the groups tab |
| Identities | "identities" | Represents the identities tab |
| ServiceTokens | "service-tokens" | Represents the service tokens tab |
