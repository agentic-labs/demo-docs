## High-level description
This file defines TypeScript types and enums related to organization permissions in the Infisical application. It includes enums for permission actions and subjects, as well as types for permission sets and the overall organization permission structure.

## Symbols

### OrgPermissionActions
#### Description
An enum that defines the possible actions for organization permissions.

#### Inputs
None

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| Read | string | Permission to read |
| Create | string | Permission to create |
| Edit | string | Permission to edit |
| Delete | string | Permission to delete |

### OrgPermissionSubjects
#### Description
An enum that defines the possible subjects for organization permissions.

#### Inputs
None

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| Workspace | string | Permission subject for workspace |
| Role | string | Permission subject for role |
| Member | string | Permission subject for member |
| Settings | string | Permission subject for settings |
| IncidentAccount | string | Permission subject for incident contact |
| Scim | string | Permission subject for SCIM |
| Sso | string | Permission subject for SSO |
| Ldap | string | Permission subject for LDAP |
| Groups | string | Permission subject for groups |
| Billing | string | Permission subject for billing |
| SecretScanning | string | Permission subject for secret scanning |
| Identity | string | Permission subject for identity |
| Kms | string | Permission subject for KMS |
| AdminConsole | string | Permission subject for organization admin console |

### OrgPermissionAdminConsoleAction
#### Description
An enum that defines the possible actions for the admin console.

#### Inputs
None

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| AccessAllProjects | string | Permission to access all projects |

### OrgPermissionSet
#### Description
A type that defines the possible combinations of actions and subjects for organization permissions.

#### Inputs
None

#### Outputs
A union type of tuples combining OrgPermissionActions and OrgPermissionSubjects.

### TOrgPermission
#### Description
A type alias for the MongoAbility type using OrgPermissionSet.

#### Inputs
None

#### Outputs
MongoAbility&lt;OrgPermissionSet&gt;

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| @casl/ability | Provides the MongoAbility type used in TOrgPermission |

This file serves as a central definition for organization permissions in the Infisical application, providing a structured way to define and manage access control across different parts of the organization.
