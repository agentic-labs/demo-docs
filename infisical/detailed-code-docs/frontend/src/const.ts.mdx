## High-level description
This file, `const.ts`, defines various constants and configurations used throughout the frontend application. It includes public paths, language mappings, plan configurations, session storage keys, and color schemes for secret tags.

## Symbols

### `publicPaths`
#### Description
An array of strings representing public paths in the application that do not require authentication.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| publicPaths | string[] | Array of public path strings |

### `languageMap`
#### Description
An object mapping language codes to their full names.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| languageMap | object | Key-value pairs of language codes and names |

### `plansDev` and `plansProd`
#### Description
Objects containing plan configurations for development and production environments.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| plansDev, plansProd | Mapping | Objects with plan names as keys and product IDs as values |

### `plans`
#### Description
Exports the production plans configuration (`plansProd`).

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| plans | Mapping | Object with plan configurations |

### `leaveConfirmDefaultMessage`
#### Description
A string constant for the default message shown when a user attempts to leave a page with unsaved changes.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| leaveConfirmDefaultMessage | string | Default confirmation message |

### `SessionStorageKeys`
#### Description
An enum defining keys used for session storage.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| SessionStorageKeys | enum | Enum with session storage key definitions |

### `secretTagsColors`
#### Description
An array of objects defining color schemes for secret tags.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| secretTagsColors | object[] | Array of color configuration objects |
