## High-level description
This code defines a reusable Button component in React using TypeScript. It provides a customizable button with various styling options, including color schemes, variants, sizes, and icons. The component is designed to be flexible and can be used in different contexts within the application.

## Code Structure
The main symbols in the code are:

1. `buttonVariants`: A function that generates CSS classes for the button based on provided variants.
2. `Button`: The main React component that renders the button.
3. `ButtonProps`: The type definition for the component's props.

The `buttonVariants` function is used by the `Button` component to determine the appropriate CSS classes based on the props passed to it.

## Symbols

### buttonVariants
#### Description
A function that generates CSS classes for the button using the `cva` (class variance authority) library.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| variants | Object | An object containing various style options for the button |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| classNames | String | A string of CSS class names |

#### Internal Logic
- Defines base classes for the button
- Specifies variants for color scheme, variant type, disabled state, full width, rounded corners, and size
- Defines compound variants for specific combinations of props

### Button
#### Description
A React component that renders a customizable button.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| props | ButtonProps | Props for customizing the button's appearance and behavior |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| JSX.Element | React component | The rendered button element |

#### Internal Logic
- Uses the `forwardRef` to allow ref forwarding
- Applies the `buttonVariants` function to generate appropriate CSS classes
- Renders the button with left and right icons, loading state, and children content

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| react | Core React library |
| cva | Class variance authority for dynamic class generation |
| tailwind-merge | Utility for merging Tailwind CSS classes |

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| colorSchema | string | "primary" | Color scheme of the button |
| variant | string | "solid" | Visual variant of the button |
| size | string | "sm" | Size of the button |
| isFullWidth | boolean | false | Whether the button should take full width |
| isRounded | boolean | true | Whether the button should have rounded corners |
| isDisabled | boolean | false | Whether the button is disabled |
| isLoading | boolean | false | Whether the button is in a loading state |

The Button component is highly customizable and can be used in various parts of the application where a button element is needed. It provides a consistent look and feel while allowing for flexibility in styling and behavior.
