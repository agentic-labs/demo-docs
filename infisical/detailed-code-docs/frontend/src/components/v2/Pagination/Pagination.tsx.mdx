## High-level description
This code defines a reusable Pagination component in React for handling pagination in user interfaces. It provides controls for navigating between pages and selecting the number of items per page, with customizable options and styling.

## Code Structure
The main component `Pagination` is exported along with its props type `PaginationProps`. It uses several sub-components from other files, such as `DropdownMenu` and `IconButton`, to create the pagination interface.

## Symbols

### `PaginationProps`
#### Description
This type defines the props for the Pagination component.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| count | number | Total number of items |
| page | number | Current page number |
| perPage | number (optional) | Number of items per page |
| onChangePage | function | Callback for page change |
| onChangePerPage | function | Callback for items per page change |
| className | string (optional) | Additional CSS classes |
| perPageList | number[] (optional) | List of options for items per page |

### `Pagination`
#### Description
A React component that renders a pagination interface with controls for navigating pages and selecting items per page.

#### Inputs
Uses the `PaginationProps` type for its props.

#### Internal Logic
1. Calculates previous and next page numbers, and whether navigation to these pages is possible.
2. Renders a container with:
   - Current range of items being displayed
   - Dropdown menu for selecting items per page
   - Navigation buttons for previous and next pages

## Performance Considerations
The component uses memoization for the dropdown menu items, which can help optimize performance when rendering large lists of per-page options.

## API/Interface Reference
| Prop | Type | Default | Description |
|:-----|:-----|:--------|:------------|
| count | number | - | Total number of items |
| page | number | 1 | Current page number |
| perPage | number | 20 | Number of items per page |
| onChangePage | function | - | Callback for page change |
| onChangePerPage | function | - | Callback for items per page change |
| perPageList | number[] | [10, 20, 50, 100] | List of options for items per page |
| className | string | - | Additional CSS classes |

This Pagination component is designed to be flexible and reusable across different parts of the application. It's used in the related `PkiCollectionItemsTable` component to handle pagination for a table of PKI collection items.
