## High-level description
This code defines a set of reusable React components for creating tables with various features such as sorting, pagination, and empty states. It includes components for the table container, table header, body, footer, and individual rows and cells. The code also provides utility components like a table skeleton for loading states and an empty state component.

## Code Structure
The main symbols in this code are:
- `TableContainer`: Wraps the entire table
- `Table`: The main table component
- `THead`, `TBody`, `TFoot`: Components for table head, body, and foot
- `Tr`, `Th`, `Td`: Components for table rows and cells
- `TableSkeleton`: A loading placeholder for tables

These components are designed to work together to create a complete table structure.

## Symbols

### `TableContainer`
#### Description
A wrapper component for the table that provides styling and layout.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| children | ReactNode | The table content |
| isRounded | boolean | Whether the container should have rounded corners |
| className | string | Additional CSS classes |

#### Internal Logic
Applies conditional CSS classes based on the `isRounded` prop and merges custom classes.

#### Internal Logic
