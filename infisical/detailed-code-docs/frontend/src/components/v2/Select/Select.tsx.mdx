## High-level description
The `Select` component is a customizable dropdown select input built using Radix UI's `SelectPrimitive`. It provides a flexible and accessible way to create select inputs with various styling options and features.

## Code Structure
The main symbols in this code are:
1. `Select`: The main component that wraps the Radix UI Select components.
2. `SelectItem`: A component for individual select options.
3. `SelectClear`: A component for clearing the selected value.

These components are interconnected, with `SelectItem` and `SelectClear` being used within the `Select` component to create a complete select input.

## Symbols

### Select
#### Description
The main select component that combines various Radix UI Select primitives to create a customizable select input.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| children | ReactNode | The content of the select, typically SelectItem components |
| placeholder | string | Placeholder text for the select |
| className | string | Additional CSS classes for styling |
| dropdownContainerClassName | string | CSS classes for the dropdown container |
| isLoading | boolean | Whether the select is in a loading state |
| position | "item-aligned" \| "popper" | Position of the dropdown |
| isDisabled | boolean | Whether the select is disabled |
| icon | IconProp | Icon to display in the select |
| isMulti | boolean | Whether multiple selections are allowed |

#### Internal Logic
1. Renders a Radix UI SelectRoot component with custom event handlers and styling.
2. Handles the display of loading state, icons, and custom styling.
3. Manages the dropdown content and positioning.

### SelectItem
#### Description
A component for individual select options within the Select component.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| children | ReactNode | The content of the select item |
| className | string | Additional CSS classes for styling |
| isSelected | boolean | Whether the item is currently selected |
| isDisabled | boolean | Whether the item is disabled |
| customIcon | IconProp | Custom icon for the item |

#### Internal Logic
Renders a Radix UI SelectItem with custom styling and icon support.

### SelectClear
#### Description
A component for clearing the selected value in the Select component.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| children | ReactNode | The content of the clear option |
| className | string | Additional CSS classes for styling |
| isSelected | boolean | Whether the clear option is selected |
| isDisabled | boolean | Whether the clear option is disabled |
| onClear | () =&gt; void | Function to call when clearing the selection |
| selectValue | string | The current value of the select |

#### Internal Logic
Renders a special SelectItem that clears the current selection when activated.

## Dependencies
- React
- Radix UI (@radix-ui/react-select)
- FontAwesome (for icons)
- tailwind-merge (for merging CSS classes)
