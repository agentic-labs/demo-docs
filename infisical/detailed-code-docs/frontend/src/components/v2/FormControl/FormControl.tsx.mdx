## High-level description
This code defines a reusable FormControl component that provides a standardized way to render form inputs with labels, error messages, and optional elements. It is designed to be flexible and customizable, allowing for various input types and styling options.

## Code Structure
The main symbols in this code are:
- `FormControl`: The main component that wraps the form input and associated elements.
- `FormLabel`: A sub-component for rendering the label of the form control.
- `FormHelperText`: A sub-component for rendering helper or error text below the form input.

These components are designed to work together, with `FormControl` typically containing a `FormLabel` and potentially a `FormHelperText`.

## Symbols

### FormControl
#### Description
The `FormControl` component is a wrapper for form inputs that provides a consistent layout and styling. It includes options for labels, error states, and helper text.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Optional ID for the form control |
| isRequired | boolean | Indicates if the field is required |
| isOptional | boolean | Indicates if the field is optional |
| isError | boolean | Indicates if the field is in an error state |
| label | ReactNode | The label for the form control |
| helperText | ReactNode | Optional helper text to display |
| errorText | ReactNode | Error text to display when in error state |
| children | JSX.Element | The form input element |
| className | string | Additional CSS classes |
| icon | ReactNode | Optional icon to display |
| tooltipText | string | Optional tooltip text |

#### Internal Logic
1. Renders a `FormLabel` if a label is provided.
2. Clones the child element (form input) with additional props.
3. Renders `FormHelperText` for either helper text or error text, depending on the state.

### FormLabel
#### Description
A component for rendering the label of a form control, with options for required/optional indicators and icons.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Optional ID for the label |
| isRequired | boolean | Indicates if the field is required |
| isOptional | boolean | Indicates if the field is optional |
| label | ReactNode | The label text |
| icon | ReactNode | Optional icon to display |
| className | string | Additional CSS classes |
| tooltipText | string | Optional tooltip text |

### FormHelperText
#### Description
A component for rendering helper or error text below a form input.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| isError | boolean | Indicates if this is error text |
| text | ReactNode | The text to display |
