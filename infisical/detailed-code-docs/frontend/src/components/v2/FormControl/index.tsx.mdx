## High-level description
This code defines and exports components for creating form controls in a React application. It provides a set of reusable components for form labels, helper text, and a wrapper component for form controls, offering a consistent and customizable way to build form interfaces.

## Code Structure
The file exports three main components: `FormControl`, `FormHelperText`, and `FormLabel`, along with their respective prop types. These components are defined in the `FormControl.tsx` file and re-exported in the `index.tsx` file for easier import in other parts of the application.

## Symbols

### FormControl
#### Description
`FormControl` is the main component that wraps form elements, providing a consistent structure for labels, input fields, and helper/error text.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| children | JSX.Element | The form input element to be wrapped |
| isRequired | boolean | Indicates if the field is required |
| isOptional | boolean | Indicates if the field is optional |
| label | ReactNode | The label for the form field |
| helperText | ReactNode | Additional helper text to display |
| errorText | ReactNode | Error message to display when isError is true |
| id | string | ID for the form field |
| isError | boolean | Indicates if the field has an error |
| icon | ReactNode | Icon to display next to the label |
| className | string | Additional CSS classes |
| tooltipText | string | Text to display in a tooltip |

#### Outputs
Returns a JSX.Element representing the complete form control.

#### Internal Logic
1. Renders a `FormLabel` if the label is a string, otherwise renders the label as-is.
2. Clones the child element (form input) with additional props.
3. Renders `FormHelperText` for either helper text or error text based on the `isError` prop.

### FormLabel
#### Description
`FormLabel` is a component for rendering form labels with optional required/optional indicators and icons.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | ID for the associated form field |
| isRequired | boolean | Indicates if the field is required |
| isOptional | boolean | Indicates if the field is optional |
| label | ReactNode | The label text or element |
| icon | ReactNode | Icon to display next to the label |
| className | string | Additional CSS classes |
| tooltipText | string | Text to display in a tooltip |

#### Outputs
Returns a JSX.Element representing the form label.

### FormHelperText
#### Description
`FormHelperText` is a component for rendering helper or error text below form fields.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| isError | boolean | Indicates if the text is an error message |
| text | ReactNode | The helper or error text to display |

#### Outputs
Returns a JSX.Element representing the helper or error text.

## Configuration
The components use Tailwind CSS classes for styling, which can be customized by passing additional classes through the `className` prop.

## Error Handling
Error states are handled through the `isError` prop in `FormControl` and `FormHelperText`. When `isError` is true, error styling is applied, and error text is displayed instead of helper text.
