## High-level description
The `UpgradePlanModal` component is a reusable modal dialog that prompts users to upgrade their subscription plan to access certain features. It is designed to be used across different parts of the application where premium features are gated behind higher-tier subscriptions.

## Code Structure
The component is structured as a functional React component that uses various hooks and sub-components to render a modal with upgrade information and action buttons.

## Symbols

### UpgradePlanModal
#### Description
This is the main component that renders the upgrade plan modal. It uses the `Modal` and `ModalContent` components to structure the dialog, and includes text explaining the benefits of upgrading along with action buttons.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| isOpen | boolean | Controls the visibility of the modal |
| onOpenChange | function | Callback function to handle opening/closing the modal |
| text | string | Custom text to display in the modal body |

#### Internal Logic
1. The component uses the `useSubscription` and `useOrganization` hooks to access subscription and organization data.
2. It determines the appropriate link for upgrading based on the current subscription status.
3. The `handleUpgradeBtnClick` function manages the upgrade process, either starting a pro trial or directing the user to the billing page.
4. The modal content includes the provided text, additional information about plan benefits, and action buttons.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| @app/context | Provides hooks for accessing subscription and organization data |
| @app/hooks/api | Provides the `useGetOrgTrialUrl` hook for fetching trial URLs |
| ../Button | Reusable button component |
| ../Modal | Reusable modal components |

## Error Handling
The component includes basic error handling in the `handleUpgradeBtnClick` function, logging any errors to the console.

## API/Interface Reference
This component is designed to be used as a child component within other parts of the application. It exposes the following props interface:

| Prop | Type | Description |
|:-----|:-----|:------------|
| isOpen | boolean | Controls the visibility of the modal |
| onOpenChange | (isOpen: boolean) =&gt; void | Callback function to handle opening/closing the modal |
| text | string | Custom text to display in the modal body |

The component is exported as `UpgradePlanModal` and can be imported and used in other parts of the application where upgrade prompts are needed.
