## High-level description
This code defines a set of React components for creating a tabbed interface using the Radix UI Tabs primitive. It provides a customizable and accessible tab system with styling applied using Tailwind CSS classes.

## Code Structure
The code exports several components: `Tabs`, `TabList`, `Tab`, and `TabPanel`. These components work together to create a complete tab interface. The `Tabs` component serves as the root container, `TabList` holds the tab triggers, `Tab` represents individual tab buttons, and `TabPanel` contains the content for each tab.

## Symbols

### `Tabs`
#### Description
The root component that wraps the entire tab interface.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| className | string | Additional CSS classes to apply |
| children | React.ReactNode | Child components |
| ...props | TabsPrimitive.TabsProps | Additional props passed to the underlying Radix UI component |

#### Internal Logic
Applies default styling and merges custom classes using `twMerge`.
