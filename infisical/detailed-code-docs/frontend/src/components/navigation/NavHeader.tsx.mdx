## High-level description
The `NavHeader` component is a navigation header used in the Infisical application. It displays the current organization, workspace, and page information, along with optional environment selection and folder navigation. This component is designed to provide context and navigation options for users across different pages of the application.

## Code Structure
The `NavHeader` component is a functional React component that takes various props to customize its appearance and behavior. It uses several other components and hooks from the application, such as `Link`, `useRouter`, `useOrganization`, `useWorkspace`, `Select`, `SelectItem`, and `Tooltip`.

## Symbols

### NavHeader
#### Description
The main component that renders the navigation header.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| pageName | string | Name of the current page |
| isProjectRelated | boolean | Whether the page is related to a project |
| isOrganizationRelated | boolean | Whether the page is related to organization settings |
| currentEnv | string | Current environment slug |
| userAvailableEnvs | any[] | List of available environments for the user |
| onEnvChange | (slug: string) =&gt; void | Callback function when environment is changed |
| isFolderMode | boolean | Whether the page is in folder mode |
| secretPath | string | Current secret path |
| isProtectedBranch | boolean | Whether the current branch is protected |
| protectionPolicyName | string | Name of the protection policy |

#### Internal Logic
1. Renders the organization name and link
2. Conditionally renders project or organization-related information
3. Displays the current page name
4. Renders environment selection dropdown if applicable
5. Displays folder navigation if in folder mode
6. Shows a lock icon for protected branches
