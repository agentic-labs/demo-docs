## High-level description
This code defines a reusable Toggle component in React, styled to resemble an iPhone-style toggle switch. It's primarily used for overriding secrets with personal values, allowing users to enable or disable specific settings.

## Symbols

### Toggle
#### Description
The Toggle component is a functional React component that renders a switch-like toggle button. It uses the Switch component from the @headlessui/react library and applies custom styling to achieve the desired appearance.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| enabled | boolean | Determines whether the toggle is in the on (true) or off (false) state |
| setEnabled | function | A function to update the enabled state |
| addOverride | function | A function that adds an override for a specific secret |
| id | string | The identifier for the secret associated with this toggle |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| JSX.Element | React component | Returns a styled Switch component |

#### Internal Logic
1. The component renders a Switch from @headlessui/react.
2. The Switch's checked prop is set to the enabled state.
3. An onChange handler is defined to:
   - Call addOverride with an empty string if the toggle is being enabled, or undefined if it's being disabled.
   - Invert the enabled state using setEnabled.
4. The component applies conditional CSS classes to style the switch based on its state.
5. A span element is used to create the sliding effect of the toggle button.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| @headlessui/react | Provides the base Switch component |

## API/Interface Reference
| Prop | Type | Description |
|:-----|:-----|:------------|
| enabled | boolean | Current state of the toggle |
| setEnabled | (value: boolean) =&gt; void | Function to update the toggle state |
| addOverride | (value: string \| undefined, id: string) =&gt; void | Function to add an override for a secret |
| id | string | Identifier for the associated secret |

## Side Effects
When the toggle is switched, it calls the addOverride function, which may modify the state of the parent component or trigger other side effects related to secret management.
