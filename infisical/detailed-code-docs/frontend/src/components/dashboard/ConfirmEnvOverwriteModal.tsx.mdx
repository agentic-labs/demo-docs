## High-level description
This code defines a React component called `ConfirmEnvOverwriteModal` that renders a modal dialog for confirming the overwriting of duplicate environment secrets. It provides options to keep old values or overwrite with new ones, displaying the list of duplicate keys.

## Symbols

### ConfirmEnvOverwriteModal
#### Description
A functional React component that renders a modal dialog for confirming the overwriting of duplicate environment secrets.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| isOpen | boolean | Determines whether the modal is open or closed |
| onClose | () =&gt; void | Function to be called when the modal is closed |
| duplicateKeys | string[] | Array of duplicate secret keys |
| onOverwriteConfirm | (preserve: "old" \| "new") =&gt; void | Function to be called when the user confirms overwriting, with the choice of preserving old or new values |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| JSX.Element | React element | The rendered modal component |

#### Internal Logic
1. The component uses the `Modal` and `ModalContent` components from "../v2" to create the modal structure.
2. It displays a title "Duplicate Secrets" and a message about duplicate secrets in the file.
3. The duplicate keys are joined into a comma-separated string and displayed.
4. Two buttons are provided in the footer:
   - "Keep Old": Calls `onOverwriteConfirm("old")` when clicked.
   - "Overwrite": Calls `onOverwriteConfirm("new")` when clicked.
5. The modal can be closed using the `onClose` function.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|

## API/Interface Reference
| Prop | Type | Description |
|:-----|:-----|:------------|
| isOpen | boolean | Controls the visibility of the modal |
| onClose | () =&gt; void | Function to close the modal |
| duplicateKeys | string[] | List of duplicate secret keys to display |
| onOverwriteConfirm | (preserve: "old" \| "new") =&gt; void | Callback function when user confirms overwrite choice |

This component is designed to be used in a larger application context where environment secret management is required. It provides a user-friendly interface for handling conflicts when importing or updating environment secrets.
