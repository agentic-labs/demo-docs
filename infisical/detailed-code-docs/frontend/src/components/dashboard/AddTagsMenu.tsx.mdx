## High-level description
The `AddTagsMenu` component is a React component that renders a dropdown menu for adding tags to a secret. It displays available tags, allows users to select new tags, and provides an option to add more tags by navigating to the project settings page.

## Symbols

### AddTagsMenu
#### Description
This is the main functional component that renders the tags menu. It uses the Headless UI `Menu` component to create an accessible dropdown menu.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| allTags | Tag[] | All available tags for a certain project |
| currentTags | Tag[] | Currently selected tags for a certain secret |
| modifyTags | function | Function to modify tags for a certain secret |
| id | string | Identifier for the secret |

#### Internal Logic
1. Renders a menu button with a plus icon and the number of additional tags (if more than 2).
2. Creates a dropdown menu with all available tags.
3. For each tag, renders a button that:
   - Shows a checked square if the tag is already selected.
   - Shows an empty square if the tag is not selected.
   - Allows adding the tag if it's not already selected.
4. Includes an "Add more tags" button at the bottom that navigates to the project settings page.
