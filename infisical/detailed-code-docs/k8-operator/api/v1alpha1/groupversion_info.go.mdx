## High-level description
This file defines the API group version information for the secrets v1alpha1 API group in the Infisical Kubernetes operator. It sets up the necessary structures and functions to register the API types with the Kubernetes API machinery.

## Symbols

### `GroupVersion`
#### Description
`GroupVersion` is a variable that defines the group and version for the secrets API. It uses the `schema.GroupVersion` struct from the Kubernetes apimachinery package.

#### Inputs
N/A (It's a variable declaration)

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| GroupVersion | schema.GroupVersion | Represents the API group and version |

### `SchemeBuilder`
#### Description
`SchemeBuilder` is a variable that creates a new scheme builder for adding Go types to the GroupVersionKind scheme. It uses the `scheme.Builder` struct from the controller-runtime package.

#### Inputs
N/A (It's a variable declaration)

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| SchemeBuilder | *scheme.Builder | A builder for adding types to the scheme |

### `AddToScheme`
#### Description
`AddToScheme` is a variable that holds a function to add the types in this group-version to a given scheme. It's assigned the `AddToScheme` method of the `SchemeBuilder`.

#### Inputs
N/A (It's a variable declaration)

|:-----|:-----|:------------|
| AddToScheme | func(*runtime.Scheme) error | Function to add types to a scheme |
