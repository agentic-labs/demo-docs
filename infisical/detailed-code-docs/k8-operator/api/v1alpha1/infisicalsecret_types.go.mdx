## High-level description
This code defines the structure and types for the InfisicalSecret Custom Resource Definition (CRD) in Kubernetes. It specifies various authentication methods and configurations for managing secrets in Kubernetes clusters using Infisical.

## Code Structure
The code defines several structs that represent different parts of the InfisicalSecret CRD. The main structs are `InfisicalSecret`, `InfisicalSecretSpec`, and `InfisicalSecretStatus`. These are interconnected, with `InfisicalSecret` containing `Spec` and `Status` fields of the respective types.
