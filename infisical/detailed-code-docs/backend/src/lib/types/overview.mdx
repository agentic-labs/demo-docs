## High-level description
This code defines several TypeScript types that represent permissions and access control concepts within the application. It includes types for generic permissions, organization-level permissions, project-level permissions, and various utility types for working with these permissions.

## Code Structure
The code defines several types related to permissions and access control. These types are used independently and do not have a direct relationship with each other.

### `RequiredKeys&lt;T&gt;`
#### Description
A utility type that extracts the names of all required properties from a given type `T`.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| T | Type | The type to extract required keys from. |

#### Outputs
A union type of string literals representing the names of required properties in `T`.

### `PickRequired&lt;T&gt;`
#### Description
A utility type that creates a new type by picking only the required properties from a given type `T`.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| T | Type | The type to pick required properties from. |

#### Outputs
A new type containing only the required properties from `T`.

### `EnforcementLevel`
#### Description
An enum defining the levels of enforcement for access control policies.

### `SecretSharingAccessType`
#### Description
An enum defining the access types for secret sharing, determining who can access a shared secret.
