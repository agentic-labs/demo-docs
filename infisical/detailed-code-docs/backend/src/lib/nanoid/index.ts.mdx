## High-level description
This code defines a function `alphaNumericNanoId` that generates random, URL-friendly, 10-character strings using only alphanumeric characters. It leverages the `nanoid` library for this purpose.

## Symbols

### `alphaNumericNanoId`
#### Description
This function generates a random 10-character string consisting of alphanumeric characters (A-Z, a-z, 0-9). It utilizes the `customAlphabet` function from the `nanoid` library to achieve this.

#### Inputs
This function takes no inputs.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| (unnamed) | string | A randomly generated 10-character alphanumeric string. |

#### Internal Logic
The function uses a predefined constant `SLUG_ALPHABETS` containing all alphanumeric characters. This string is used as the alphabet for the `customAlphabet` function from the `nanoid` library. The `customAlphabet` function is called with the alphabet and the desired length (10) to generate the random string.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| nanoid | Provides the functionality for generating random IDs. |
