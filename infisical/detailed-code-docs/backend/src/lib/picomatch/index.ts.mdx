## Symbols

### `containsGlobPatterns`
#### Description
This function determines if a given string, representing a secret path, contains any glob pattern characters. It normalizes the input path before checking for the presence of glob characters.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| secretPath | string | The secret path string to check for glob patterns. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
|  | boolean | Returns `true` if the `secretPath` contains any glob pattern characters, otherwise returns `false`. |

#### Internal Logic
1. Defines an array `globChars` containing all the glob pattern characters.
2. Normalizes the input `secretPath` using `path.normalize()`.
3. Uses `some()` to iterate over the `globChars` array and checks if any of the characters are present in the normalized path.
4. Returns `true` if any glob character is found, otherwise returns `false`.
