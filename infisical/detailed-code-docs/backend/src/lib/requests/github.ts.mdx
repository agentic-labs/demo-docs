## High-level description
This code defines a function `fetchGithubEmails` that fetches a user's email addresses from the GitHub API using the provided access token. 

## Symbols

### `fetchGithubEmails`
#### Description
This function sends a GET request to the GitHub API to retrieve the authenticated user's email addresses.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| accessToken | string | GitHub access token for authentication |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| data | `TGithubEmail[]` | An array of objects containing email information (email, primary, verified, visibility) |

#### Internal Logic
1. Constructs the API endpoint URL using `INTEGRATION_GITHUB_API_URL` and "/user/emails".
2. Sends a GET request to the constructed URL using the provided `accessToken` for authorization.
3. Returns the `data` property of the response, which contains an array of `TGithubEmail` objects.
