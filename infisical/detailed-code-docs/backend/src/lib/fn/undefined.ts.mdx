## High-level description
This code defines a helper function `executeIfDefined` that safely executes a given function on an input value only if the input is not undefined. This prevents potential errors caused by calling a function with an undefined input.

## Symbols

### `executeIfDefined`
#### Description
This function takes a function `func` and an input `input` as arguments. It checks if the input is undefined. If the input is undefined, it returns undefined. Otherwise, it calls the provided function with the input and returns the result.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| func | `(input: T) =&gt; R` | A function that takes an argument of type `T` and returns a value of type `R`. |
| input | `T \| undefined` | The input value which can be of type `T` or undefined. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
|  | `R \| undefined` | Returns the result of calling `func` with `input` if `input` is defined, otherwise returns undefined. |

#### Internal Logic
1. Check if the `input` is undefined using the strict equality operator (`===`).
2. If the `input` is undefined, return undefined.
3. If the `input` is defined, call the provided `func` with the `input` and return the result.

#### Performance Considerations
This function is expected to have minimal performance overhead as it only involves a simple conditional check and a function call.
