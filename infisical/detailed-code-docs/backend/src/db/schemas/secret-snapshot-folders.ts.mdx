## High-level description
The code defines a Zod schema, `SecretSnapshotFoldersSchema`, for validating the structure and data types of Secret Snapshot Folder objects. It also defines related types for inserting and updating these objects in the database.

The schema defines the following properties for a `SecretSnapshotFolders` object:

- `folderVersionId`: A UUID string representing the folder version ID.
