## High-level description
The code defines a Zod schema named `LdapGroupMapsSchema` for validating `LdapGroupMaps` objects. It also defines related types for inserting and updating `LdapGroupMaps` data. The schema and types are automatically generated from the database.

### `LdapGroupMapsSchema`
#### Description
This symbol defines a Zod schema for validating `LdapGroupMaps` objects. It ensures that the objects have the correct data types and structure as defined in the schema.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| (implicit) | `z.ZodObject` | A Zod object schema representing the structure of an `LdapGroupMaps` object. |

#### Internal Logic
The schema defines the following fields:
- `id`: A UUID string representing the unique identifier of the LDAP group map.
- `ldapConfigId`: A UUID string representing the unique identifier of the associated LDAP configuration.
- `ldapGroupCN`: A string representing the common name (CN) of the LDAP group.
- `groupId`: A UUID string representing the unique identifier of the associated group.
