## High-level description
This code defines the `SecretRotationOutputV2Schema` using the `zod` library to validate the shape of data for a secret rotation output entity. It also defines related types for inserting and updating this data.

### `SecretRotationOutputV2Schema`
#### Description
This symbol defines a `zod` schema named `SecretRotationOutputV2Schema` to validate the structure and data types of a secret rotation output object.

#### Internal Logic

- `key`: A required string field.
- `secretId`: A required string field representing a UUID.
- `rotationId`: A required string field representing a UUID.
