## High-level description
This file defines TypeScript types used for super admin operations, including admin sign-up, user retrieval, and supported login methods. These types ensure data consistency and provide clear interfaces for interacting with super admin functionalities.

## Symbols

### `TAdminSignUpDTO`
#### Description
Defines the data structure for an admin sign-up request.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| email | `string` | Admin's email address. |
| password | `string` | Admin's password. |
| publicKey | `string` | Admin's public key. |
| salt | `string` | Salt used for password hashing. |
| lastName | `string` | Admin's last name (optional). |
| verifier | `string` | Verifier string for authentication. |
| firstName | `string` | Admin's first name. |
| protectedKey | `string` | Admin's protected key. |
| protectedKeyIV | `string` | Initialization vector for the protected key. |
| protectedKeyTag | `string` | Authentication tag for the protected key. |
| encryptedPrivateKey | `string` | Admin's encrypted private key. |
| encryptedPrivateKeyIV | `string` | Initialization vector for the encrypted private key. |
| encryptedPrivateKeyTag | `string` | Authentication tag for the encrypted private key. |
| ip | `string` | IP address of the sign-up request. |
| userAgent | `string` | User agent of the sign-up request. |

### `TAdminGetUsersDTO`
#### Description
Defines the data structure for retrieving a list of users.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| offset | `number` | Offset for pagination. |
| limit | `number` | Maximum number of users to retrieve. |
| searchTerm | `string` | Search term for filtering users. |

### `LoginMethod`
#### Description
An enum representing the different login methods supported by the application.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| EMAIL | `"email"` | Login using email and password. |
| GOOGLE | `"google"` | Login using Google OAuth. |
| GITHUB | `"github"` | Login using GitHub OAuth. |
| GITLAB | `"gitlab"` | Login using GitLab OAuth. |
| SAML | `"saml"` | Login using SAML. |
| LDAP | `"ldap"` | Login using LDAP. |
| OIDC | `"oidc"` | Login using OpenID Connect. |
