## High-level description
This file defines TypeScript types related to identity access tokens. It includes types for renewing access tokens and for the payload of JWT tokens used in identity access.

## Symbols

### `TRenewAccessTokenDTO`
#### Description
This type defines the structure for a Data Transfer Object (DTO) used when renewing an access token.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| accessToken | string | The current access token that needs to be renewed |

### `TIdentityAccessTokenJwtPayload`
#### Description
This type defines the structure for the payload of a JWT (JSON Web Token) used for identity access tokens.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| identityId | string | Unique identifier for the identity |
| clientSecretId | string | Identifier for the client secret |
| identityAccessTokenId | string | Unique identifier for the identity access token |
| authTokenType | string | Type of authentication token |

## References
This file is referenced and used in other parts of the codebase, particularly in the identity and authentication services. The `TIdentityAccessTokenJwtPayload` type is used in the `identity-kubernetes-auth-service.ts` file for creating JWT tokens.

## Dependencies
This file doesn't have any external dependencies. It only uses TypeScript's built-in type system.

Note: The types defined in this file are crucial for maintaining type safety and consistency across the application, especially in areas dealing with authentication and identity management.
