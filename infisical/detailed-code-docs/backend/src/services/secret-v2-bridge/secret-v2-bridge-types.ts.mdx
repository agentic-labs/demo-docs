## High-level description
This file defines TypeScript types used by the Secret V2 Bridge service in the backend. These types represent data structures for secrets, secret versions, tags, and various operations related to managing secrets.

## Code Structure
This file primarily defines a series of TypeScript types. Some types are standalone, while others are used as parameters or return values for functions defined in other files.
