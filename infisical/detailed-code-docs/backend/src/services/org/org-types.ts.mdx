## High-level description
This TypeScript file defines various Data Transfer Object (DTO) types used for organization-related operations in the backend. These types are used to structure the data for different operations such as updating memberships, inviting users, verifying users, and managing workspaces within an organization.

## Symbols

### `TUpdateOrgMembershipDTO`
#### Description
Defines the structure for updating an organization membership.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| userId | string | The ID of the user whose membership is being updated |
| orgId | string | The ID of the organization |
| membershipId | string | The ID of the membership being updated |
| role | string (optional) | The new role for the member |
| isActive | boolean (optional) | The active status of the membership |
| actorOrgId | string \| undefined | The organization ID of the actor performing the update |
| actorAuthMethod | ActorAuthMethod | The authentication method of the actor |
