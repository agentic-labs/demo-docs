## High-level description
This file defines TypeScript types related to webhooks, including data transfer objects (DTOs) for creating, updating, testing, listing, and deleting webhooks. It also defines an enum for different webhook types.

## References
- `@app/lib/types`: Uses `TProjectPermission` type.

## Symbols

### `TCreateWebhookDTO`
#### Description
Represents the data required to create a new webhook.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| environment | string | The environment the webhook belongs to. |
| secretPath? | string | Optional. The secret path pattern the webhook is triggered for. |
| webhookUrl | string | The URL of the webhook endpoint. |
| webhookSecretKey? | string | Optional. The secret key for webhook signature verification. |
| type | string | The type of the webhook. |
| ...TProjectPermission |  | Includes properties from `TProjectPermission` for authorization. |

#### Outputs
None. This is a data transfer object.
