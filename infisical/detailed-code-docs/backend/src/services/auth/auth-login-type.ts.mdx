## High-level description
This TypeScript file defines several type definitions related to authentication and login processes. It includes types for various authentication scenarios such as server key generation, client proof submission, multi-factor authentication (MFA) verification, OAuth login, and OAuth token exchange.

## Symbols

### `TLoginGenServerPublicKeyDTO`
#### Description
Defines the structure for a Data Transfer Object (DTO) used when generating a server public key during the login process.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| email | string | User's email address |
| clientPublicKey | string | Client's public key |
| providerAuthToken | string (optional) | Authentication token from a provider |

#### Description
Defines the structure for a DTO used when submitting client proof during the login process.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| email | string | User's email address |
