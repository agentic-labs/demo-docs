## High-level description
This file defines TypeScript types for Data Transfer Objects (DTOs) used in group project operations. It includes types for creating, updating, deleting, and listing project groups, incorporating project slug permissions and temporary membership modes.

## Symbols

### `TCreateProjectGroupDTO`
#### Description
Defines the structure for creating a project group.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| groupSlug | string | The unique identifier for the group |
| role | string | The role assigned to the group in the project |
| TProjectSlugPermission | type | Inherited type containing project slug and actor information |

### `TUpdateProjectGroupDTO`
#### Description
Defines the structure for updating a project group, including support for temporary roles.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| roles | array | An array of role objects, which can be either permanent or temporary |
| groupSlug | string | The unique identifier for the group |
| TProjectSlugPermission | type | Inherited type containing project slug and actor information |

#### Internal Logic
The `roles` array can contain two types of role objects:
1. Permanent role: `{ role: string, isTemporary?: false }`
2. Temporary role: `{ role: string, isTemporary: true, temporaryMode: ProjectUserMembershipTemporaryMode.Relative, temporaryRange: string, temporaryAccessStartTime: string }`

### `TDeleteProjectGroupDTO`
#### Description
Defines the structure for deleting a project group.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| groupSlug | string | The unique identifier for the group to be deleted |
| TProjectSlugPermission | type | Inherited type containing project slug and actor information |

### `TListProjectGroupDTO`
#### Description
Defines the structure for listing project groups.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| TProjectSlugPermission | type | Inherited type containing project slug and actor information |

This file serves as a central location for defining the data structures used in group project operations, ensuring type safety and consistency across the application. It incorporates project permissions and supports both permanent and temporary role assignments for groups within projects.
