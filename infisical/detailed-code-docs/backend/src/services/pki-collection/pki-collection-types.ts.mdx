## High-level description
This file defines TypeScript types and enums related to PKI (Public Key Infrastructure) collections. It includes data transfer objects (DTOs) for creating, retrieving, updating, and deleting PKI collections, as well as managing items within these collections.

## Symbols

### `TCreatePkiCollectionDTO`
#### Description
Defines the structure for creating a new PKI collection.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| name | string | The name of the PKI collection |
| description | string | A description of the PKI collection |
| TProjectPermission | type | Project permission details (inherited) |

### `TGetPkiCollectionByIdDTO`
#### Description
Defines the structure for retrieving a PKI collection by its ID.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| collectionId | string | The ID of the PKI collection |
| TProjectPermission | type | Project permission details (inherited, excluding projectId) |

### `TUpdatePkiCollectionDTO`
#### Description
Defines the structure for updating an existing PKI collection.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| collectionId | string | The ID of the PKI collection to update |
| name | string (optional) | The updated name of the PKI collection |
| description | string (optional) | The updated description of the PKI collection |
| TProjectPermission | type | Project permission details (inherited, excluding projectId) |

### `TDeletePkiCollectionDTO`
#### Description
Defines the structure for deleting a PKI collection.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| collectionId | string | The ID of the PKI collection to delete |
| TProjectPermission | type | Project permission details (inherited, excluding projectId) |

### `PkiItemType`
#### Description
An enum defining the types of PKI items.

#### Values
- CERTIFICATE: "certificate"
- CA: "ca"

### `pkiItemTypeToNameMap`
#### Description
A mapping of PKI item types to their human-readable names.

### `TGetPkiCollectionItems`
#### Description
Defines the structure for retrieving items from a PKI collection.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| collectionId | string | The ID of the PKI collection |
| type | PkiItemType (optional) | The type of PKI items to retrieve |
| offset | number | The offset for pagination |
| limit | number | The limit for pagination |
| TProjectPermission | type | Project permission details (inherited, excluding projectId) |

### `TAddItemToPkiCollectionDTO`
#### Description
Defines the structure for adding an item to a PKI collection.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| collectionId | string | The ID of the PKI collection |
| type | PkiItemType | The type of the item to add |
| itemId | string | The ID of the item to add |
| TProjectPermission | type | Project permission details (inherited, excluding projectId) |

### `TRemoveItemFromPkiCollectionDTO`
#### Description
Defines the structure for removing an item from a PKI collection.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| collectionId | string | The ID of the PKI collection |
| itemId | string | The ID of the item to remove |
| TProjectPermission | type | Project permission details (inherited, excluding projectId) |
