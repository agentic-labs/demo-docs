## High-level description
This file defines TypeScript enums and types related to certificate management in a backend application. It includes enums for certificate status, key algorithms, and revocation reasons, as well as types for various certificate-related operations.

## Symbols

### `CertStatus`
#### Description
An enum representing the possible statuses of a certificate.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ACTIVE | string | Represents an active certificate |
| REVOKED | string | Represents a revoked certificate |

### `CertKeyAlgorithm`
#### Description
An enum representing the supported key algorithms for certificates.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| RSA_2048 | string | RSA with 2048-bit key |
| RSA_4096 | string | RSA with 4096-bit key |
| ECDSA_P256 | string | ECDSA with P-256 curve |
| ECDSA_P384 | string | ECDSA with P-384 curve |

### `CrlReason`
#### Description
An enum representing the reasons for certificate revocation.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| UNSPECIFIED | string | Unspecified reason |
| KEY_COMPROMISE | string | Private key has been compromised |
| CA_COMPROMISE | string | CA has been compromised |
| AFFILIATION_CHANGED | string | The certificate subject's affiliation has changed |
| SUPERSEDED | string | The certificate has been superseded |
| CESSATION_OF_OPERATION | string | The certified entity has ceased operation |
| CERTIFICATE_HOLD | string | The certificate is on hold |
| PRIVILEGE_WITHDRAWN | string | The privilege granted in the certificate has been withdrawn |
| A_A_COMPROMISE | string | AA has been compromised |
