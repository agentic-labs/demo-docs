## High-level description
This file defines TypeScript types related to secret folders used within a project. It includes types for creating, updating, deleting, and retrieving folder information, as well as an enum for reserved folder names.

## Symbols
### `ReservedFolders`
#### Description
An enum defining reserved folder names that should not be used for user-created folders.

#### Internal Logic
- `SecretReplication`: Represents a reserved folder name used internally for secret replication.

### `TCreateFolderDTO`
#### Description
Represents the data required to create a new secret folder.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| environment | string | The environment where the folder resides. |
| path | string | The folder's path within the environment. |
| name | string | The name of the folder. |
| projectId | string | The ID of the project the folder belongs to. |
| userId | string | The ID of the user performing the action. |

### `TUpdateFolderDTO`
#### Description
Represents the data required to update an existing secret folder.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| environment | string | The environment where the folder resides. |
| path | string | The folder's path within the environment. |
| id | string | The ID of the folder to update. |
| name | string | The new name of the folder. |
| projectId | string | The ID of the project the folder belongs to. |
| userId | string | The ID of the user performing the action. |

### `TUpdateManyFoldersDTO`
#### Description
Represents the data required to update multiple secret folders within a project.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| projectSlug | string | The slug of the project containing the folders. |
| folders | `{ environment: string; path: string; id: string; name: string; }[]` | An array of objects containing update data for each folder. |
| userId | string | The ID of the user performing the action. |

### `TDeleteFolderDTO`
#### Description
Represents the data required to delete a secret folder.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| environment | string | The environment where the folder resides. |
| path | string | The folder's path within the environment. |
| idOrName | string | The ID or name of the folder to delete. |
| projectId | string | The ID of the project the folder belongs to. |
| userId | string | The ID of the user performing the action. |

### `TGetFolderDTO`
#### Description
Represents the data required to retrieve a secret folder.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| environment | string | The environment where the folder resides. |
| path | string | The folder's path within the environment. |
| projectId | string | The ID of the project the folder belongs to. |
| userId | string | The ID of the user performing the action. |

### `TGetFolderByIdDTO`
#### Description
Represents the data required to retrieve a secret folder by its ID.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | The ID of the folder to retrieve. |
| userId | string | The ID of the user performing the action. |
