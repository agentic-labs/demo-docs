## High-level description
The code defines TypeScript types related to PKI (Public Key Infrastructure) alerts. These types are used for creating, retrieving, updating, and deleting PKI alerts, and specify the expected data structure for each operation. 

## Code Structure
The code defines several TypeScript types: `TCreateAlertDTO`, `TGetAlertByIdDTO`, `TUpdateAlertDTO`, and `TDeleteAlertDTO`. Each type represents the data structure for a specific PKI alert operation. They all utilize and extend the `TProjectPermission` type for authorization purposes.

## References
- `TProjectPermission`: This type is imported from "@app/lib/types" and is used to define project permissions for PKI alerts.

## Symbols
### `TCreateAlertDTO`
#### Description
Defines the data structure for creating a new PKI alert.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| name | string | Name of the alert. |
| pkiCollectionId | string | ID of the PKI collection the alert belongs to. |
| alertBeforeDays | number | Number of days before expiry to trigger the alert. |
| emails | string[] | Array of email addresses to send the alert to. |
| ...TProjectPermission |  | Includes properties from the `TProjectPermission` type for authorization. |

#### Outputs
This is a data transfer object, it doesn't return any value by itself.
