## High-level description
This file defines TypeScript types for certificate template related operations in the application. These types are used for data transfer and validation purposes within the certificate template service.

## References
This file references the following types:
- `TProjectPermission` from "@app/lib/types"

## Symbols
### `TCreateCertTemplateDTO`
#### Description
This type defines the data structure for creating a new certificate template.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| caId | string | The ID of the certificate authority (CA) to associate with the template. |
| pkiCollectionId | string | (Optional) The ID of the PKI collection to associate with the template. |
| name | string | The name of the certificate template. |
| commonName | string | The common name to be included in the certificate. |
| subjectAlternativeName | string | The subject alternative name to be included in the certificate. |
| ttl | string | The time-to-live (TTL) for certificates generated from this template. |
| actorId | string | The ID of the actor performing the action. |
| actorAuthMethod | string | The authentication method used by the actor. |
| actor | string | The actor object. |
| actorOrgId | string | The ID of the organization the actor belongs to. |

#### Outputs
This is a data transfer object (DTO) and does not directly return any output. It is used as input to the `createCertTemplate` function.

### `TGetEstConfigurationDTO`
#### Description
This type defines the data structure for retrieving an EST (Enrollment over Secure Transport) configuration for a certificate template.

#### Inputs
This is a union type, meaning it can be one of two structures:

**Structure 1:**
| Name | Type | Description |
|:-----|:-----|:------------|
| isInternal | true | Flag indicating that the request is internal. |
| certificateTemplateId | string | The ID of the certificate template associated with the EST configuration. |

**Structure 2:**
| Name | Type | Description |
|:-----|:-----|:------------|
| isInternal | false | Flag indicating that the request is not internal. |
| certificateTemplateId | string | The ID of the certificate template associated with the EST configuration. |
| actorId | string | The ID of the actor performing the action. |
| actorAuthMethod | string | The authentication method used by the actor. |
| actor | string | The actor object. |
| actorOrgId | string | The ID of the organization the actor belongs to. |

#### Outputs
This is a data transfer object (DTO) and does not directly return any output. It is used as input to the `getEstConfiguration` function.
