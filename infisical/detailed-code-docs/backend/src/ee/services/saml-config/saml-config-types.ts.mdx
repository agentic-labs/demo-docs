## High-level description
This file defines TypeScript types and enums related to SAML (Security Assertion Markup Language) configuration for authentication in the application. It includes types for creating, updating, and retrieving SAML configurations, as well as for SAML login data transfer objects (DTOs).

## Symbols

### `SamlProviders`
#### Description
An enum that defines the supported SAML providers.

#### Inputs
N/A (Enum)

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| OKTA_SAML | string | Okta SAML provider |
| AZURE_SAML | string | Azure SAML provider |
| JUMPCLOUD_SAML | string | JumpCloud SAML provider |
| GOOGLE_SAML | string | Google SAML provider |
| KEYCLOAK_SAML | string | Keycloak SAML provider |

### `TCreateSamlCfgDTO`
#### Description
A type that defines the structure for creating a SAML configuration.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| authProvider | SamlProviders | The SAML provider |
| isActive | boolean | Whether the configuration is active |
| entryPoint | string | The SAML entry point URL |
| issuer | string | The SAML issuer |
| cert | string | The SAML certificate |
| TOrgPermission | type | Organization permission properties (inherited) |

### `TUpdateSamlCfgDTO`
#### Description
A type that defines the structure for updating a SAML configuration.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| authProvider? | SamlProviders | The SAML provider (optional) |
| isActive? | boolean | Whether the configuration is active (optional) |
| entryPoint? | string | The SAML entry point URL (optional) |
| issuer? | string | The SAML issuer (optional) |
| cert? | string | The SAML certificate (optional) |
| TOrgPermission | type | Organization permission properties (inherited) |

### `TGetSamlCfgDTO`
#### Description
A union type that defines different ways to retrieve a SAML configuration.

#### Inputs
Three possible structures:
1. Organization-based retrieval:
   - type: "org"
   - orgId: string
   - actor: ActorType
   - actorId: string
   - actorAuthMethod: ActorAuthMethod
   - actorOrgId: string | undefined
2. Organization slug-based retrieval:
   - type: "orgSlug"
   - orgSlug: string
3. SSO ID-based retrieval:
   - type: "ssoId"
   - id: string

### `TSamlLoginDTO`
#### Description
A type that defines the structure for SAML login data.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| externalId | string | External identifier for the user |
| email | string | User's email address |
| firstName | string | User's first name |
| lastName? | string | User's last name (optional) |
| authProvider | string | Authentication provider |
| orgId | string | Organization ID |
| relayState? | string | SAML relay state (optional) |
