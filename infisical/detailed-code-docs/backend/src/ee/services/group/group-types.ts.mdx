## High-level description
This file defines TypeScript types and interfaces for group-related operations in the backend of an application. It includes types for creating, updating, and deleting groups, as well as managing user memberships within groups. The types are designed to work with various data access layers (DALs) and database operations.

## Code Structure
The file consists of multiple type definitions that are used to structure the input and output of group-related functions. These types are likely used across different parts of the application to ensure type safety and consistency in group operations.

## Symbols

### `TCreateGroupDTO`
#### Description
Defines the structure for creating a new group.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| name | string | The name of the group |
| slug | string (optional) | A URL-friendly identifier for the group |
| role | string | The role associated with the group |

#### Description

#### Inputs
