## High-level description
This file contains utility functions for handling SCIM (System for Cross-domain Identity Management) operations. It provides functions for building SCIM user and group lists, parsing SCIM filters, extracting SCIM values from paths, and constructing SCIM user and group objects.

## Code Structure
The file exports several functions that work together to handle SCIM-related operations. These functions are used to format and process data for SCIM requests and responses.

## Symbols

### `buildScimUserList`
#### Description
Constructs a SCIM user list response object.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| scimUsers | TScimUser[] | Array of SCIM user objects |
| startIndex | number | Starting index for pagination |
| limit | number | Limit for pagination |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| TListScimUsers | object | SCIM user list response object |
