## High-level description
This file defines TypeScript types and interfaces for SCIM (System for Cross-domain Identity Management) operations in the backend of an application. It includes types for SCIM token management, user management, and group management, as well as request and response structures for various SCIM-related operations.

#### Description

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|

| ttlDays | number | Time-to-live in days for the token |
