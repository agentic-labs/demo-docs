## High-level description
This TypeScript file defines types and interfaces related to secret rotation functionality. It includes types for creating, listing, deleting, and restarting secret rotations, incorporating project permissions and specific rotation details.

## Symbols

### `TCreateSecretRotationDTO`
#### Description
Defines the structure for creating a new secret rotation.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| secretPath | string | Path to the secret |
| environment | string | Environment for the secret rotation |
| interval | number | Rotation interval |
| provider | string | Provider for the secret rotation |
| inputs | Record&lt;string, unknown&gt; | Input parameters for the rotation |
| outputs | Record&lt;string, string&gt; | Output mappings for the rotation |
| ... | TProjectPermission | Project permission properties |

### `TListByProjectIdDTO`
#### Description
Type alias for project permissions, used for listing secret rotations by project ID.

#### Inputs
Inherits all properties from `TProjectPermission`.

### `TDeleteDTO`
#### Description
Defines the structure for deleting a secret rotation.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| rotationId | string | ID of the rotation to be deleted |
| ... | Omit&lt;TProjectPermission, "projectId"&gt; | Project permission properties, excluding projectId |

### `TRestartDTO`
#### Description
Defines the structure for restarting a secret rotation.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| rotationId | string | ID of the rotation to be restarted |
| ... | Omit&lt;TProjectPermission, "projectId"&gt; | Project permission properties, excluding projectId |
