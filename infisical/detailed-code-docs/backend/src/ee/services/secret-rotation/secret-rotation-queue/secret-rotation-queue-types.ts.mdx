## High-level description
This file defines types related to secret rotation in the backend of an enterprise edition (EE) application. It includes types for encrypted secret rotation data, decrypted secret rotation data, and database function parameters for secret rotation.

## Symbols

### `TSecretRotationEncData`
#### Description
Defines the structure for encrypted secret rotation data.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| inputs | Record&lt;string, unknown&gt; | Encrypted input data for secret rotation |
| creds | Array&lt;{ outputs: Record&lt;string, unknown&gt;; internal: Record&lt;string, unknown&gt;; }&gt; | Array of credential objects containing encrypted output and internal data |

### `TSecretRotationData`
#### Description
Defines the structure for decrypted secret rotation data.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| inputs | Record&lt;string, unknown&gt; | Decrypted input data for secret rotation |
| outputs | Record&lt;string, unknown&gt; | Decrypted output data from secret rotation |
| internal | Record&lt;string, unknown&gt; | Decrypted internal data used in secret rotation |

### `TSecretRotationDbFn`
#### Description
Defines the parameters for a database function used in secret rotation.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| client | TDbProviderClients | The database client type |
| username | string | Database username |
| password | string | Database password |
| host | string | Database host |
| database | string | Database name |
| port | number | Database port |
| query | string | SQL query to execute |
| variables | unknown[] | Variables to be used in the query |
| ca | string | (Optional) CA certificate for SSL connection |
