## High-level description
This code defines and exports an array of secret rotation templates for various services and databases. It provides a centralized collection of templates that can be used for rotating credentials or API keys for different platforms such as Sendgrid, PostgreSQL, MySQL, and AWS IAM.

## Code Structure
The code imports template definitions from separate files and combines them into a single array called `rotationTemplates`. Each template in the array follows a consistent structure defined by the `TSecretRotationProviderTemplate` type.

## Symbols

### `rotationTemplates`
#### Description
An array of secret rotation templates for different services and databases. Each template contains metadata about the service and a reference to the actual template implementation.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| rotationTemplates | TSecretRotationProviderTemplate[] | An array of secret rotation template objects |

#### Internal Logic
The array is populated with four template objects, each representing a different service or database:
1. Twilio Sendgrid
2. PostgreSQL/CockroachDB
3. MySQL/MariaDB
4. AWS IAM

Each template object contains the following properties:
- `name`: A unique identifier for the template
- `title`: A human-readable title for the service
- `image`: The filename of an image associated with the service
- `description`: A brief description of what the template rotates
- `template`: A reference to the imported template implementation
