## High-level description
This TypeScript file defines types related to Certificate Revocation Lists (CRLs) for a Certificate Authority (CA) system. It includes types for retrieving CRLs by ID and for getting CA CRLs with specific permissions.

### `TGetCaCrlsDTO`
#### Description
This type defines the structure for a Data Transfer Object (DTO) used to retrieve CRLs for a specific Certificate Authority. It combines the CA ID with project permissions, excluding the project ID.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| caId | string | The unique identifier of the Certificate Authority |
| (other properties) | Omit&lt;TProjectPermission, "projectId"&gt; | Project permissions, excluding the project ID |
