## High-level description
This TypeScript file defines types and interfaces related to OpenID Connect (OIDC) configuration for an authentication system. It includes types for OIDC login, configuration retrieval, creation, and update operations, as well as an enum for configuration types.

## Symbols

### `OIDCConfigurationType`
#### Description
An enum that defines the types of OIDC configuration.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| CUSTOM | string | Represents a custom OIDC configuration |
| DISCOVERY_URL | string | Represents an OIDC configuration using a discovery URL |

___

### `TOidcLoginDTO`
#### Description
A type definition for the OIDC login data transfer object.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| externalId | string | The external identifier for the user |
| email | string | The user's email address |
| firstName | string | The user's first name |
| lastName? | string | The user's last name (optional) |
| orgId | string | The organization identifier |
| callbackPort? | string | The callback port (optional) |

___

### `TGetOidcCfgDTO`
#### Description
A union type for getting OIDC configuration, which can be either external or internal.

#### Inputs
For external type:
| Name | Type | Description |
|:-----|:-----|:------------|
| type | "external" | Indicates an external configuration |
| orgSlug | string | The organization slug |
| TGenericPermission | unknown | Generic permission type (imported) |

For internal type:
| Name | Type | Description |
|:-----|:-----|:------------|
| type | "internal" | Indicates an internal configuration |
| orgSlug | string | The organization slug |

___
