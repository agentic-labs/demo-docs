## High-level description
This file defines various types and enums related to licensing and billing for an enterprise software application. It includes definitions for different instance types, license contents, feature sets, and various DTOs (Data Transfer Objects) for handling organization-related operations.

## Symbols

### `InstanceType` (enum)
#### Description
Defines the different types of instances for the application.

#### Enum Values
| Value | Description |
|:------|:------------|
| OnPrem | Self-hosted instance |
| EnterpriseOnPrem | Enterprise self-hosted instance |
| EnterpriseOnPremOffline | Enterprise self-hosted offline instance |
| Cloud | Cloud-hosted instance |

### `TOfflineLicenseContents` (type)
#### Description
Represents the contents of an offline license.

#### Properties
| Name | Type | Description |
|:-----|:-----|:------------|
| license | TOfflineLicense | The offline license details |
| signature | string | The signature for license verification |

### `TOfflineLicense` (type)
#### Description
Defines the structure of an offline license.

#### Properties
| Name | Type | Description |
|:-----|:-----|:------------|
| issuedTo | string | The entity the license is issued to |
| licenseId | string | Unique identifier for the license |
| customerId | string \| null | Customer identifier |
| issuedAt | string | Date when the license was issued |
| expiresAt | string \| null | Expiration date of the license |
| terminatesAt | string \| null | Termination date of the license |
| features | TFeatureSet | Set of features included in the license |

### `TFeatureSet` (type)
#### Description
Defines the set of features and limits for a license.

#### Properties
Various boolean flags and numeric limits for different features.

### `TOrgPermission` (type)
#### Description
Imported type representing organization permissions.

### Various DTO types
#### Description
Multiple Data Transfer Object types are defined for different operations related to organization billing, plans, and licensing. These types typically extend `TOrgPermission` and may include additional properties specific to their use case.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| @app/lib/types | Imports TOrgPermission type |

This file serves as a central location for type definitions related to licensing and billing in the application. It provides a structured way to represent different aspects of licenses, instance types, and feature sets, which are likely used across various parts of the application for license management and feature access control.
