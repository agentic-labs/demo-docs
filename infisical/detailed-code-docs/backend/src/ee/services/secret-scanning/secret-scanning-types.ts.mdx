## High-level description
This file defines various types and enums related to secret scanning functionality in the backend of an application. It includes risk status enums, data transfer object (DTO) types for installation sessions, organization installation status, organization risks, and risk status updates.

## Symbols

### `SecretScanningRiskStatus`
#### Description
An enum that defines the possible statuses for secret scanning risks.

#### Enum Values
| Value | Description |
|:------|:------------|
| FalsePositive | Resolved as a false positive |
| Revoked | Resolved and revoked |
| NotRevoked | Resolved but not revoked |
| Unresolved | Not yet resolved |

### `TInstallAppSessionDTO`
#### Description
A type alias for `TOrgPermission`, representing the data transfer object for installing an app session.

### `TLinkInstallSessionDTO`
#### Description
Defines the data transfer object for linking an installation session.

#### Properties
| Name | Type | Description |
|:-----|:-----|:------------|
| installationId | string | The ID of the installation |
| sessionId | string | The ID of the session |
| ... | Omit&lt;TOrgPermission, "orgId"&gt; | Other properties from TOrgPermission, excluding orgId |

### `TGetOrgInstallStatusDTO`
#### Description
A type alias for `TOrgPermission`, representing the data transfer object for getting an organization's installation status.

### `TGetOrgRisksDTO`
#### Description
A type alias for `TOrgPermission`, representing the data transfer object for getting an organization's risks.

### `TUpdateRiskStatusDTO`
#### Description
Defines the data transfer object for updating a risk status.

#### Properties
| Name | Type | Description |
|:-----|:-----|:------------|
| riskId | string | The ID of the risk to update |
| status | SecretScanningRiskStatus | The new status for the risk |
| ... | TOrgPermission | Other properties from TOrgPermission |
