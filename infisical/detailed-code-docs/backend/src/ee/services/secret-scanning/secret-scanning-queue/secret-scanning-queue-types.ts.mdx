## High-level description
This file defines TypeScript types related to secret scanning events and results within the application. It includes types for representing secret matches found in code, as well as payloads for push events and full repository scan events.

## Symbols
### `SecretMatch`
#### Description
This type represents a single secret match found in the code. It contains detailed information about the match, the secret, the location in the code, and metadata about the commit and author.

#### Inputs
This is a type definition, not a function, so it doesn't have inputs.

#### Outputs
This is a type definition, not a function, so it doesn't have outputs.

### `TScanPushEventPayload`
#### Description
This type represents the payload of a push event that triggers a secret scan. It contains information about the organization, the commits that were pushed, the user who pushed the commits, the repository, and the installation ID.

#### Inputs
This is a type definition, not a function, so it doesn't have inputs.

#### Outputs
This is a type definition, not a function, so it doesn't have outputs.

### `TScanFullRepoEventPayload`
#### Description
This type represents the payload of an event that triggers a full repository scan. It contains information about the organization, the installation ID, and the repository.

#### Inputs
This is a type definition, not a function, so it doesn't have inputs.

#### Outputs
This is a type definition, not a function, so it doesn't have outputs.
