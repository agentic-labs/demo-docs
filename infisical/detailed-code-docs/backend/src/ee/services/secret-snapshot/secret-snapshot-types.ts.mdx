## High-level description
This file defines TypeScript types for various Data Transfer Objects (DTOs) used in the secret snapshot functionality of the application. These types are used to structure the data for operations like counting snapshots, listing snapshots, retrieving snapshot data, and rolling back snapshots.

## Symbols

### `TProjectSnapshotCountDTO`
#### Description
This type defines the structure for counting project snapshots.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| environment | string | The environment of the snapshot |
| path | string | The path of the snapshot |
| TProjectPermission | type | Inherited properties for project permissions |

### `TProjectSnapshotListDTO`
#### Description
This type defines the structure for listing project snapshots.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| environment | string | The environment of the snapshot |
| path | string | The path of the snapshot |
| offset | number (optional) | The offset for pagination |
| limit | number (optional) | The limit for pagination |
| TProjectPermission | type | Inherited properties for project permissions |

### `TGetSnapshotDataDTO`
#### Description
This type defines the structure for retrieving snapshot data.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | The ID of the snapshot |
| TProjectPermission | type | Inherited properties for project permissions, excluding projectId |

### `TRollbackSnapshotDTO`
#### Description
This type defines the structure for rolling back a snapshot.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | The ID of the snapshot to rollback |
| TProjectPermission | type | Inherited properties for project permissions, excluding projectId |

## Dependencies

## References
This file is referenced by and used in conjunction with `secret-snapshot-service.ts`, which implements the actual functionality using these types.

The types defined in this file are crucial for maintaining type safety and providing clear interfaces for the secret snapshot functionality throughout the application.
