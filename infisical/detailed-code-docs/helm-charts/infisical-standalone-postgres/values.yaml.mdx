## High-level description
This YAML file contains configuration values for deploying Infisical, a standalone PostgreSQL database, and Redis using Helm charts. It defines various settings for the Infisical application, ingress, PostgreSQL database, and Redis cache, allowing for customization of the deployment.

## Code Structure
The YAML file is structured into several main sections: general settings, Infisical configuration, ingress configuration, PostgreSQL configuration, and Redis configuration. Each section contains specific parameters for customizing the respective components of the deployment.

## Symbols

### nameOverride
#### Description
An optional field to override the name of the Helm release.

### fullnameOverride
#### Description
An optional field to override the full name of the Helm release.

### infisical
#### Description
Contains configuration settings for the Infisical application.

#### Internal Logic
- Enables the Infisical deployment
- Sets the name and replica count
- Configures the Docker image and pull policy
- Defines resource limits and requests
- Sets up service configuration

### ingress
#### Description
Configures the ingress for the Infisical application.

#### Internal Logic
- Disabled by default
- Allows setting up hostname, ingress class, and annotations
- Provides option for TLS configuration

### postgresql
#### Description
Configures the PostgreSQL database for Infisical.

#### Internal Logic
- Enables PostgreSQL deployment
- Sets the database name, username, password, and database name

### redis
#### Description
Configures the Redis cache for Infisical.

#### Internal Logic
- Enables Redis deployment
- Sets up Redis authentication
- Configures Redis as a standalone instance

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| infisical.enabled | boolean | true | Enables Infisical deployment |
| infisical.replicaCount | integer | 2 | Number of Infisical replicas |
| infisical.image.tag | string | "v0.46.3-postgres" | Infisical Docker image tag |
| ingress.enabled | boolean | false | Enables ingress for Infisical |
| postgresql.enabled | boolean | true | Enables PostgreSQL deployment |
| redis.enabled | boolean | true | Enables Redis deployment |

Note: This table includes only a subset of the available configuration options. The YAML file contains many more options for fine-tuning the deployment.
