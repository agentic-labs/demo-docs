## High-level description
This code defines the gRPC service for the AnalyticsService in the Flipt analytics package. It provides functionality for retrieving flag evaluation counts and includes both client and server-side implementations for the gRPC service.

## Code Structure
The code is structured into two main parts:
1. The gRPC service definition (AnalyticsServiceClient and AnalyticsServiceServer interfaces)
2. The implementation of the gRPC-Gateway, which translates gRPC into RESTful JSON APIs

## Symbols

#### Description
An interface that defines the client-side API for the AnalyticsService.
