## High-level description
This Go code defines a set of constants and variables related to operators used in the Flipt package. It establishes various types of operators and their valid usage contexts for different data types, such as strings, numbers, booleans, and entity IDs.

## Code Structure
The code consists of constant declarations for different operators and several map variables that group these operators based on their validity and applicability to different data types.

## Performance Considerations
The use of maps with empty struct values (`map[string]struct{}`) is a memory-efficient way to represent sets in Go. This approach allows for constant-time lookups when checking if an operator is valid for a particular context.
