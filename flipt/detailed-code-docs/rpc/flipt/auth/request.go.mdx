## High-level description
This code defines a set of request methods for authentication-related operations in the Flipt system. It provides functions to create requests for creating tokens, listing authentications, getting authentication details, and deleting authentications.

## Code Structure
The code defines four functions, each corresponding to a different authentication-related request type. These functions are methods on different request structs and they all return a `flipt.Request` object.

## Symbols

### `CreateTokenRequest.Request()`
#### Description
This method creates a request for token creation.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| (return value) | flipt.Request | A new request for creating a token |

#### Internal Logic
It uses `flipt.NewRequest()` to create a new request with `ResourceAuthentication`, `ActionCreate`, and a subject of `SubjectToken`.
