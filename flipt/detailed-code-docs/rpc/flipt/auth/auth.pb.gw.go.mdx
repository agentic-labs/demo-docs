## High-level description
This file contains generated code for gRPC gateway functionality related to authentication services in the Flipt project. It defines request/response types and service interfaces for various authentication methods like token-based, OIDC, Kubernetes, and GitHub authentication.

## Code Structure
The file defines several protobuf message types and gRPC service interfaces. The main services are:

- PublicAuthenticationService
- AuthenticationService  
- AuthenticationMethodTokenService
- AuthenticationMethodOIDCService
- AuthenticationMethodKubernetesService
- AuthenticationMethodGithubService

These services define methods for operations like listing authentication methods, creating tokens, authorizing URLs, handling callbacks, etc. The code also includes generated client and server implementations for these services.
