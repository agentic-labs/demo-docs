## High-level description
This YAML file configures StackHawk, a security testing tool, for the Flipt application. It defines basic settings for the application, including its identifier, environment, host, and OpenAPI specification file location.

## Symbols

### `app`
#### Description
The `app` section contains the main configuration settings for the StackHawk scan of the Flipt application.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| applicationId | string | A unique identifier for the Flipt application in StackHawk |
| env | string | The environment in which the application is running |
| host | string | The base URL where the application is hosted |
| openApiConf | object | Configuration for the OpenAPI specification |

#### Internal Logic
The `app` section defines the essential parameters needed for StackHawk to scan the Flipt application. It specifies where the application is running and how to access its API documentation.

### `openApiConf`
#### Description
A subsection of `app` that specifies the location of the OpenAPI specification file.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| filePath | string | The path to the OpenAPI specification file |
