## High-level description
This file is a `package.json` file that defines the dependencies and scripts for a Next.js application example that integrates with Flipt. It specifies the required packages for both development and production environments.

## References
This `package.json` file references the following external packages:

* `@flipt-io/flipt`: The main Flipt server-side SDK.
* `@flipt-io/flipt-client-browser`: The Flipt client-side SDK for browser-based integrations.

| @flipt-io/flipt-client-browser | Enables feature flag evaluation and management from the browser. |
| @next/font | Next.js built-in component for using self-hosted fonts efficiently. |

## Configuration
This `package.json` file defines scripts for running various development tasks:

| Option | Description |
|:-------|:------------|
| dev | Starts the Next.js development server. |
| build | Builds the Next.js application for production. |
| start | Starts the Next.js production server. |
| lint | Runs ESLint to check code style and quality. |
