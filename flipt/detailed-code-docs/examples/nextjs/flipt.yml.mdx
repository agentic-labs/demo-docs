## High-level description
This YAML file defines a Flipt feature flag configuration for a Next.js application. It sets up a "Language" feature flag with three variants (Spanish, English, and French) and a rule that distributes traffic evenly among them.

## Code Structure
The file defines a `namespace`, `flags`, and `segments`. The `flags` section contains a list of feature flags, each with its own `variants` and `rules`. The `rules` section references `segments` defined in the `segments` section.
