## High-level description
This code defines a Next.js custom `App` component, which serves as the root component for all pages in the application. It renders the current page component (`Component`) with its props (`pageProps`).

## Symbols

### `App`
#### Description
This is a functional component that acts as a wrapper for all pages in the Next.js application. It renders the provided `Component` (representing the current page) and passes through any `pageProps`.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| Component | React.ComponentType | The React component representing the current page to be rendered. |
| pageProps | Record&lt;string, any&gt; | An object containing properties to be passed to the page component. |

#### Outputs
- Returns a React element representing the rendered page component.

#### Internal Logic
1. Imports global CSS styles from `'../styles/globals.css'`.
2. Defines a functional component named `App` that accepts `Component` and `pageProps` as props.
3. Renders the `Component` with the provided `pageProps` spread into it.

|:-----------|:--------|
| next/app | Provides the `AppProps` type for type-safety. |
| react | Used implicitly for JSX and component rendering. |
