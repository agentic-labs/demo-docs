## High-level description
This file defines the default database root directory for the Flipt application when running on Linux systems. It provides a platform-specific implementation of the `defaultDatabaseRoot` function, which returns the default path for storing database files on Linux.

## Symbols

### `defaultDatabaseRoot`
#### Description
The `defaultDatabaseRoot` function returns the default directory path where the Flipt database should be stored on Linux systems. It provides a consistent location for database files across Linux installations.

#### Inputs
This function takes no inputs.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| string | string | The default database root directory path for Linux |
| error | error | Always nil in this implementation |

#### Internal Logic
The function simply returns a hardcoded string "/var/opt/flipt" as the default database root directory for Linux systems. It also returns a nil error, indicating that the operation always succeeds.

## Dependencies
This file has no external dependencies beyond the Go standard library.

## Configuration
The default database root path is hardcoded in this file:
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| Database Root | string | "/var/opt/flipt" | The default directory for storing Flipt database files on Linux |
