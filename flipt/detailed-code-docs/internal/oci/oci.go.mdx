## High-level description

This file defines constants, errors, and types related to OCI (Open Container Initiative) functionality in the Flipt project. It includes media types for Flipt features and namespaces, annotations for identifying namespace keys, and error types for common OCI-related issues.

## Symbols

#### Description
A constant representing the OCI media type for a Flipt features artifact.
