## High-level description
The `internal/server/otel/attributes.go` file defines a set of OpenTelemetry (OTel) attribute keys used for instrumenting Flipt's server-side operations with tracing and metrics data. These attributes provide context and details about Flipt's feature flag evaluations and other server operations.

## Code Structure
This file simply declares a set of global variables, each representing an OTel attribute key. These keys are used in other parts of the codebase to associate specific values with spans and metrics, enabling better monitoring and observability of Flipt's behavior.

## References
- `go.opentelemetry.io/otel/attribute`: This package provides the necessary types for defining and working with OTel attributes.
- `go.opentelemetry.io/otel/semconv/v1.26.0`: This package defines semantic conventions for OTel, including specific attributes for feature flags.
