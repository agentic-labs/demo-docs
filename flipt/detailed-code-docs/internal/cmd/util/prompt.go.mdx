## High-level description
This Go package provides utility functions for prompting user input in a command-line interface. It offers three main functions: one for plain text input, one for password input (hidden), and one for confirmation (yes/no) input. These functions utilize the `survey` library to handle user interactions.

## Symbols

### `PromptPlaintext`
#### Description
This function prompts the user to input plain text, with an optional default value.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| message | string | The prompt message displayed to the user |
| defaultVal | string | The default value for the input |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| value | string | The user's input, trimmed of leading and trailing whitespace |
| error | error | Any error that occurred during the prompt |

#### Internal Logic
1. Creates a `survey.Input` struct with the provided message and default value.
2. Uses `survey.AskOne` to prompt the user, with a required validator.
3. Trims whitespace from the input and returns it.

### `PromptPassword`
#### Description
This function prompts the user to input a password, hiding the input.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| message | string | The prompt message displayed to the user |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| password | string | The user's input password, trimmed of leading and trailing whitespace |
| error | error | Any error that occurred during the prompt |

#### Internal Logic
1. Creates a `survey.Password` struct with the provided message.
2. Uses `survey.AskOne` to prompt the user, with a required validator.
3. Trims whitespace from the input and returns it.

### `PromptConfirm`
#### Description
This function prompts the user for a yes/no confirmation.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| message | string | The prompt message displayed to the user |
| defaultVal | bool | The default value for the confirmation |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| value | bool | The user's confirmation (true for yes, false for no) |
| error | error | Any error that occurred during the prompt |

#### Internal Logic
1. Creates a `survey.Confirm` struct with the provided message and default value.
2. Uses `survey.AskOne` to prompt the user.
3. Returns the boolean result.
