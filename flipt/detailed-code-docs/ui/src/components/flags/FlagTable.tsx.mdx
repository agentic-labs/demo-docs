## High-level description
This code defines a `FlagTable` component in React that displays a table of feature flags with sorting, pagination, and filtering capabilities. It uses the `@tanstack/react-table` library for table functionality and integrates with Redux for state management.

## Code Structure
The main component `FlagTable` uses several hooks and utilities to manage the table's state and render the data. It defines columns for the table, sets up sorting and pagination, and renders the table with a search box and pagination controls.

## References
- Redux store (for namespace selection)
- React Router (for navigation)
- Custom hooks (useTimezone)
- Utility functions (truncateKey, flagTypeToLabel)

## Symbols

### FlagTable
#### Description
A React component that renders a table of feature flags with sorting, pagination, and filtering capabilities.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| flags | IFlag[] | An array of flag objects to be displayed in the table |

#### Internal Logic
1. Sets up state for sorting, pagination, and filtering
2. Defines table columns with custom cell renderers
3. Configures the table using `useReactTable` hook
4. Renders the table header with sorting controls
5. Renders the table body with flag data
6. Adds a search box for filtering (if there are enough flags)
7. Adds pagination controls

## Performance Considerations
The component uses pagination and filtering to handle large datasets efficiently. The search functionality is only rendered when there are 10 or more flags, improving performance for smaller datasets.
