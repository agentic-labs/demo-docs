## High-level description
This code defines a reusable Dropdown component in React, which creates a customizable dropdown menu with various actions. It uses the Headless UI library for accessibility and transitions, and allows for flexible styling and configuration of dropdown items.

## Code Structure
The main `Dropdown` component uses the `Menu` component from Headless UI to create the dropdown structure. It renders a button that toggles the dropdown visibility and a list of menu items based on the provided actions. The component is highly customizable, allowing for different styles and behaviors for each action item.

## Symbols

### Dropdown
#### Description
A functional component that renders a dropdown menu with customizable actions.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| props | DropdownProps | Contains the label for the dropdown button and an array of actions |

#### Internal Logic
1. Renders a `Menu.Button` with the provided label and a chevron icon.
2. Uses `Transition` component for smooth appearance/disappearance of the dropdown.
3. Maps over the `actions` array to render individual `Menu.Item` components.
4. Handles disabled states for actions and applies appropriate styling.
5. Executes the `onClick` function for each action when clicked.

### DropdownAction
#### Description
An interface defining the structure of each action item in the dropdown.

#### Properties
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the action |
| label | string | Text to display for the action |
| icon? | Icon | Optional icon component to display alongside the label |
| onClick | () =&gt; void | Function to execute when the action is clicked |
| disabled? | boolean | Optional flag to disable the action |
| title? | string | Optional title attribute for the action |
| className? | string | Optional CSS class for custom styling |
| activeClassName? | string | Optional CSS class for active state |
| inActiveClassName? | string | Optional CSS class for inactive state |

### DropdownProps
#### Description
An interface defining the props for the Dropdown component.

#### Properties
| Name | Type | Description |
|:-----|:-----|:------------|
| label | string | Text to display on the dropdown button |
| actions | DropdownAction[] | Array of action items to include in the dropdown |
