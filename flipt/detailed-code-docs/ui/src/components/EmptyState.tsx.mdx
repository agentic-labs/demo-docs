## High-level description
The `EmptyState` component is a reusable React component that displays a button with customizable text and icon. It's designed to be used when there's no content to display, providing a call-to-action for users to create new items or perform actions.

## Code Structure
The component is defined as a function that takes props and returns JSX. It uses conditional rendering to display different elements based on the provided props.

## Symbols

### EmptyState
#### Description
A functional component that renders a button with customizable text, icon, and behavior.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| text | string | Optional. The main text to display on the button |
| secondaryText | string | Optional. Additional text to display below the main text |
| disabled | boolean | Optional. Whether the button is disabled |
| Icon | Icon | Optional. A custom icon component to display |
| onClick | () =&gt; void | Optional. Function to call when the button is clicked |

#### Internal Logic
1. Destructures props with default values
2. Returns a button element with conditional rendering for icon, text, and secondary text
3. Uses the `cls` utility function to conditionally apply CSS classes
4. Disables the button if `onClick` is not provided or `disabled` is true

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| @heroicons/react/24/outline | Provides the default PlusCircleIcon |
| types/Icon | Defines the Icon type |
| utils/helpers | Provides the `cls` utility function for conditional class names |

## Usage Examples
The `EmptyState` component is used in various parts of the application, such as:
1. In the Flags component when there are no flags to display
2. In the Segments component when there are no segments to display
3. In the Console component when there's no response to display
4. In the Namespaces component when there are no namespaces to display
5. In the Tokens component when there are no tokens to display

In each case, it provides a consistent way to prompt users to create new items or perform actions when there's no existing content.
