## High-level description
This code defines a `SortableRollout` component that wraps the `Rollout` component with drag-and-drop functionality. It uses the `useSortable` hook from the `@dnd-kit/sortable` library to enable sorting of rollouts in a list.

## Code Structure
The `SortableRollout` component is the main export of this file. It uses props from its parent component and the `useSortable` hook to render a sortable `Rollout` component.

#### Outputs
Returns a JSX element representing the sortable rollout.

#### Internal Logic
1. Destructures the props to get individual values.
2. Uses the `useSortable` hook to get drag-and-drop related properties and functions.
3. Defines a `style` object based on the `transform` value from `useSortable`.
4. Sets a `className` based on whether the item is being dragged.
5. Returns a `Rollout` component with additional props for drag-and-drop functionality.
