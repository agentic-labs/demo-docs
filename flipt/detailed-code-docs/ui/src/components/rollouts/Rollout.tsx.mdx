## High-level description
This code defines a React component called `Rollout` that represents a single rollout rule in a feature flag management system. It displays the rollout's details and provides options for editing and deleting the rollout. The component is designed to be used within a draggable list of rollouts.

## Code Structure
The main `Rollout` component is a forwardRef component that renders a list item (`&lt;li&gt;`) containing the rollout's information and controls. It uses several sub-components and utilities from other parts of the application, such as `QuickEditRolloutForm` for inline editing of the rollout.

## Symbols

### `Rollout`
#### Description
A React component that renders a single rollout rule with its details and controls.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| flag | IFlag | The feature flag associated with this rollout |
| rollout | IRollout | The rollout rule data |
| segments | ISegment[] | Available segments for the rollout |
| onSuccess | () =&gt; void | Optional callback function called on successful operations |
| onEdit | () =&gt; void | Optional callback function for edit action |
| onDelete | () =&gt; void | Optional callback function for delete action |
| style | React.CSSProperties | Optional inline styles |
| className | string | Optional CSS class names |
| readOnly | boolean | Flag to indicate if the rollout is in read-only mode |

#### Internal Logic
1. Renders a list item with the rollout's rank, type, and description.
2. Provides a menu for edit and delete actions (if not in read-only mode).
3. Renders a `QuickEditRolloutForm` for inline editing of the rollout.
