## High-level description
This code defines a `NamespaceTable` component in React, which renders a table of namespaces with sorting, filtering, and pagination capabilities. It also includes functionality for editing and deleting namespaces, with appropriate UI elements and modals.

## Code Structure
The main `NamespaceTable` component uses several helper components and hooks from the `@tanstack/react-table` library to manage the table's state and functionality. It also imports custom components like `Pagination` and `Searchbox` for additional features.

## Symbols

### NamespaceEditAction
#### Description
A component that renders an editable link for a namespace name.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| cell | CellContext&lt;INamespace, string&gt; | The cell context from the table |
| setEditingNamespace | (namespace: INamespace) =&gt; void | Function to set the namespace being edited |
| setShowEditNamespaceModal | (show: boolean) =&gt; void | Function to show/hide the edit modal |

#### Internal Logic
When clicked, it sets the editing namespace and shows the edit modal.

### NamespaceDeleteAction
#### Description
A component that renders a delete action for a namespace.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| row | Row&lt;INamespace&gt; | The row data from the table |
| setDeletingNamespace | (namespace: INamespace) =&gt; void | Function to set the namespace being deleted |
| setShowDeleteNamespaceModal | (show: boolean) =&gt; void | Function to show/hide the delete modal |

#### Internal Logic
It checks if the namespace is protected, and if not, allows deletion by setting the deleting namespace and showing the delete modal.

### NamespaceTable
#### Description
The main component that renders the table of namespaces with sorting, filtering, and pagination.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| namespaces | INamespace[] | Array of namespace objects to display |
| setEditingNamespace | (namespace: INamespace) =&gt; void | Function to set the namespace being edited |
| setShowEditNamespaceModal | (show: boolean) =&gt; void | Function to show/hide the edit modal |
| setDeletingNamespace | (namespace: INamespace) =&gt; void | Function to set the namespace being deleted |
| setShowDeleteNamespaceModal | (show: boolean) =&gt; void | Function to show/hide the delete modal |
| readOnly | boolean | Flag to indicate if the table is in read-only mode |

#### Internal Logic
1. Sets up state for sorting, filtering, and pagination.
2. Defines table columns using `createColumnHelper`.
3. Uses `useReactTable` hook to create and manage the table instance.
4. Renders the table header with sorting functionality.
5. Renders the table body with rows and cells.
6. Includes a `Searchbox` component for filtering if there are enough namespaces.
7. Includes a `Pagination` component for navigating through pages.
