## High-level description
This code defines a Loading component in React, which renders a loading spinner. The component is customizable with props for primary color and full-screen display.

## Code Structure
The code exports a single functional component named `Loading`. It uses the `cls` utility function from a helper module to conditionally apply CSS classes.

## Symbols

### `Loading`
#### Description
A React functional component that renders a loading spinner.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| isPrimary | boolean | Optional. Determines if the spinner should use primary colors. |
| fullScreen | boolean | Optional. Determines if the spinner should be displayed full-screen. |

#### Internal Logic
1. Destructures the `isPrimary` and `fullScreen` props.
2. Uses the `cls` function to conditionally apply CSS classes based on the props.
3. Renders a div containing the spinner, with classes applied for styling and animation.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| cls | A utility function for conditionally joining CSS class names |

## Notes
- The component uses Tailwind CSS classes for styling.
- The spinner is created using border styles and animation.
- The component is flexible and can be used in various contexts within the application.
