## High-level description
This code defines TypeScript types and interfaces for rollout functionality in a feature flag system. It includes enums for rollout types, interfaces for rollout rules, and utility functions for working with rollouts.

## Code Structure
The code defines several interconnected types and interfaces:
- `RolloutType` enum defines the types of rollouts (SEGMENT and THRESHOLD).
- `IRolloutRuleSegment` and `IRolloutRuleThreshold` interfaces define the structure for segment and threshold rollout rules.
- `IRolloutBase` and `IRollout` interfaces define the base and extended rollout structures.
- `IRolloutList` interface extends `IPageable` to represent a list of rollouts.
