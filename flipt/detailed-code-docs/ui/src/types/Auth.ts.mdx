## High-level description
This file defines TypeScript interfaces for authentication methods and authentication objects in the Flipt UI. It provides a structured way to represent different authentication methods, their metadata, and authentication tokens.

## Symbols

### `IAuthMethod`
#### Description
Defines the structure for an authentication method.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| method | string literal union | The type of authentication method |
| enabled | boolean | Whether the method is enabled |
| sessionCompatible | boolean | Whether the method is compatible with sessions |
| metadata | object | Additional metadata for the method |

### `IAuthMethodList`
#### Description
Represents a list of authentication methods.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| methods | IAuthMethod[] | An array of authentication methods |

### `IAuth`
#### Description
Defines the structure for an authentication object.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the authentication |
| method | string | The authentication method used |
| expiresAt | string | Expiration timestamp |
| createdAt | string | Creation timestamp |
| updatedAt | string | Last update timestamp |
