## High-level description
This file defines TypeScript types and interfaces related to feature flags in a UI application. It includes enums, interfaces, and utility functions for working with flag types, flag data structures, and flag lists.

## Code Structure
The file defines several interconnected types and interfaces:
- `FlagType` enum defines the types of flags (VARIANT and BOOLEAN).
- `IFlagBase` interface defines the basic properties of a flag.
- `IFlag` interface extends `IFlagBase` with additional properties.
- `IFlagList` interface combines `IPageable` with an array of `IFlag` objects.
- `FilterableFlag` type combines `IFlag` with `IFilterable` properties.

## Symbols

### `FlagType` (enum)
#### Description
Defines the types of flags available in the system.

#### Values
| Name | Value | Description |
|:-----|:------|:------------|
| VARIANT | 'VARIANT_FLAG_TYPE' | Represents a variant flag type |
| BOOLEAN | 'BOOLEAN_FLAG_TYPE' | Represents a boolean flag type |

### `flagTypeToLabel` (function)
#### Description
Converts a `FlagType` enum value to a human-readable label.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| flagType | FlagType | The flag type to convert |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| label | string | The human-readable label for the flag type |

### `IFlagBase` (interface)
#### Description
Defines the basic properties of a flag.

#### Properties
| Name | Type | Description |
|:-----|:-----|:------------|
| key | string | Unique identifier for the flag |
| type | FlagType | Type of the flag |
| name | string | Name of the flag |
| enabled | boolean | Whether the flag is enabled |
| description | string | Description of the flag |
| defaultVariant? | IVariant | Optional default variant for the flag |

### `IFlag` (interface)
#### Description
Extends `IFlagBase` with additional properties for a complete flag representation.

#### Properties
| Name | Type | Description |
|:-----|:-----|:------------|
| createdAt | string | Timestamp of flag creation |
| updatedAt | string | Timestamp of last flag update |
| variants? | IVariant[] | Optional array of variants for the flag |

### `IFlagList` (interface)
#### Description
Represents a pageable list of flags.

#### Properties
| Name | Type | Description |
|:-----|:-----|:------------|
| flags | IFlag[] | Array of flags in the list |
| nextPageToken | string | Token for the next page of results |
| totalCount | number | Total count of flags |

### `FilterableFlag` (type)
#### Description
Combines `IFlag` with `IFilterable` properties for use in filtering and selection contexts.

## Dependencies
