## High-level description
This file defines TypeScript interfaces and enums for various metadata and configuration-related structures used in the UI of a feature flag management system. It includes definitions for system information, storage types, UI configuration, and loading statuses.

## Symbols

### `IInfo`
#### Description
An interface that represents system information about the application.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| version | string | Current version of the application |
| latestVersion | string (optional) | Latest available version |
| latestVersionURL | string (optional) | URL to the latest version |
| commit | string | Git commit hash |
| buildDate | string | Date when the application was built |
| goVersion | string | Version of Go used to build the application |
| updateAvailable | boolean | Indicates if an update is available |
| isRelease | boolean | Indicates if this is a release version |

### `IGit`
#### Description
An interface that represents Git-related information.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ref | string | Git reference |
| repository | string | Git repository URL or name |

### `IStorage`
#### Description
An interface that represents storage configuration.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| type | StorageType | Type of storage being used |
| readOnly | boolean (optional) | Indicates if the storage is read-only |
| git | IGit (optional) | Git-related information if applicable |

### `ITopbar`
#### Description
An interface that represents the configuration for the top bar in the UI.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| color | string (optional) | Color of the top bar |
| label | string (optional) | Label text for the top bar |

### `IUI`
#### Description
An interface that represents UI-related configuration.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| defaultTheme | Theme | Default theme for the UI |
| topbar | ITopbar | Configuration for the top bar |

### `IConfig`
#### Description
An interface that represents the overall configuration of the application.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| status | LoadingStatus | Current loading status of the configuration |
| storage | IStorage | Storage configuration |
| ui | IUI | UI configuration |
| analyticsEnabled | boolean | Indicates if analytics are enabled |

### `StorageType`
#### Description
An enum that represents different types of storage.

#### Values
- DATABASE
- GIT
- LOCAL
- OBJECT
- OCI

### `LoadingStatus`
#### Description
An enum that represents different loading statuses.

#### Values
- IDLE
- LOADING
- SUCCEEDED
- FAILED
