## High-level description
This file defines TypeScript interfaces for namespaces in a feature flag management system. It includes interfaces for basic namespace properties, full namespace details, and a list of namespaces with pagination support.

## Symbols

### `INamespaceBase`
#### Description
Defines the basic properties of a namespace.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| key | string | Unique identifier for the namespace |
| name | string | Display name for the namespace |
| description | string (optional) | Additional information about the namespace |

### `INamespace`
#### Description
Extends `INamespaceBase` with additional properties for a full namespace representation.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| protected | boolean | Indicates if the namespace is protected from deletion |
| createdAt | string | Timestamp of when the namespace was created |
| updatedAt | string | Timestamp of when the namespace was last updated |

### `INamespaceList`
#### Description
Represents a paginated list of namespaces.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| namespaces | INamespace[] | Array of namespace objects |
| nextPageToken | string | Token for retrieving the next page of results |
| totalCount | number | Total number of namespaces available |

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| IPageable | Imported from './Pageable' to provide pagination properties |

This file is crucial for defining the structure of namespace data used throughout the application. It provides a clear interface for working with individual namespaces and lists of namespaces, including pagination support for efficient data retrieval and display.
