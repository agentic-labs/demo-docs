## High-level description
This TypeScript file defines interfaces and an enum related to distribution configurations for feature flags. It provides structures for representing distribution variants and their rollout percentages.

## Symbols

### `IDistributionBase`
#### Description
An interface that defines the base structure for a distribution.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| variantId | string | Unique identifier for the variant |
| rollout | number | Percentage of users who will see this variant |

### `IDistributionVariant`
#### Description
An interface that extends `IDistributionBase` to include a variant key.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| variantKey | string | Key identifier for the variant |

### `IDistribution`
#### Description
An interface that extends `IDistributionBase` to include additional metadata.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | Unique identifier for the distribution |
| createdAt | string | Timestamp of when the distribution was created |
| updatedAt | string | Timestamp of when the distribution was last updated |

### `DistributionType`
#### Description
An enum that defines the types of distributions available.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| None | string | Represents no distribution |
| Single | string | Represents a single variant distribution |
| Multi | string | Represents a multi-variant distribution |
