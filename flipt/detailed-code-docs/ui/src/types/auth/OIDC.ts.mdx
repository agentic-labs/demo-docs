## High-level description
This TypeScript file defines interfaces related to OpenID Connect (OIDC) authentication method for the Flipt application. It extends the base authentication interfaces and provides specific structures for OIDC metadata and authentication details.

## Code Structure
The code defines four main interfaces: `AuthMethodOIDCMetadataProvider`, `IAuthMethodOIDC`, `IAuthMethodOIDCMetadata`, and `IAuthOIDCInternal`. These interfaces build upon and extend the base `IAuth` and `IAuthMethod` interfaces imported from the `Auth.ts` file.

## References
- `IAuth` and `IAuthMethod` from `~/types/Auth`

## Symbols

### `AuthMethodOIDCMetadataProvider`
#### Description
This interface defines the structure for an OIDC provider's metadata, specifically the authorization URL.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| authorize_url | string | The URL used for OIDC authorization |

### `IAuthMethodOIDC`
#### Description
This interface extends the `IAuthMethod` interface to specify the OIDC authentication method and its metadata.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| method | 'METHOD_OIDC' | Specifies that this is an OIDC authentication method |
| metadata | object | Contains providers, which is a record of OIDC provider metadata |

### `IAuthMethodOIDCMetadata`
#### Description
This interface defines the structure for OIDC-specific metadata, including various claims that can be present in an OIDC token.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| 'io.flipt.auth.oidc.provider' | string | The OIDC provider identifier |
| 'io.flipt.auth.oidc.email'? | string | Optional email claim |
| 'io.flipt.auth.oidc.email_verified'? | string | Optional email verification status |
| 'io.flipt.auth.oidc.name'? | string | Optional name claim |
| 'io.flipt.auth.oidc.picture'? | string | Optional picture URL claim |
| 'io.flipt.auth.oidc.preferred_username'? | string | Optional preferred username claim |

### `IAuthOIDCInternal`
#### Description
This interface extends the `IAuth` interface to represent an internal OIDC authentication object with OIDC-specific metadata.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| method | 'METHOD_OIDC' | Specifies that this is an OIDC authentication method |
| metadata | IAuthMethodOIDCMetadata | The OIDC-specific metadata |

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| ~/types/Auth | Imports base interfaces IAuth and IAuthMethod |

This file is crucial for implementing OIDC authentication in the Flipt application. It provides type definitions that ensure type safety when working with OIDC-related data throughout the application. The interfaces defined here are used in other parts of the codebase, such as the `SessionProvider` component and the API handling code, to manage OIDC authentication sessions and metadata.
