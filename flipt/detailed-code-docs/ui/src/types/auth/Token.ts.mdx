## High-level description
This file defines TypeScript interfaces for authentication tokens used in the Flipt application. It includes interfaces for token metadata, internal token representation, token lists, and public token information.

## Symbols

### `IAuthMethodTokenMetadata`
#### Description
Defines the metadata structure for authentication tokens.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| 'io.flipt.auth.token.name' | string | The name of the token |
| 'io.flipt.auth.token.description' | string (optional) | The description of the token |
| 'io.flipt.auth.token.namespace' | string (optional) | The namespace associated with the token |

### `IAuthTokenInternal`
#### Description
Extends the `IAuth` interface (imported from `~/types/Auth`) and includes token metadata.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| metadata | IAuthMethodTokenMetadata | The metadata associated with the token |

### `IAuthTokenInternalList`
#### Description
Defines a list of internal authentication tokens.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| authentications | IAuthTokenInternal[] | An array of internal authentication tokens |

### `IAuthTokenBase`
#### Description
Defines the base structure for an authentication token.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| name | string | The name of the token |
| description | string (optional) | The description of the token |
| namespaceKey | string (optional) | The namespace key associated with the token |
| expiresAt | string (optional) | The expiration date of the token |

### `IAuthToken`
#### Description
Extends `IAuthTokenBase` and includes additional fields for a complete token representation.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| id | string | The unique identifier of the token |
| createdAt | string | The creation date of the token |
| updatedAt | string | The last update date of the token |

### `IAuthTokenSecret`
#### Description
Defines the structure for a token secret, including the client token and authentication information.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| clientToken | string | The client token string |
| authentication | IAuthTokenInternal | The internal authentication information |

## References
This file references the `IAuth` interface from `~/types/Auth`.

## Dependencies
- `~/types/Auth`: Provides the `IAuth` interface used in `IAuthTokenInternal`.

The interfaces defined in this file are used throughout the application for managing and interacting with authentication tokens, particularly in components and API calls related to token creation, listing, and management.
