## High-level description
This TypeScript file defines interfaces related to GitHub authentication in the Flipt application. It extends the base authentication interfaces to provide specific structures for GitHub authentication methods and metadata.

## Symbols

### `IAuthMethodGithub`
#### Description
This interface extends the `IAuthMethod` interface to specify the GitHub authentication method.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| method | string | Fixed value 'METHOD_GITHUB' |
| metadata | object | Contains the authorize_url for GitHub authentication |

### `IAuthMethodGithubMetadata`
#### Description
This interface defines the structure for GitHub-specific authentication metadata.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| 'io.flipt.auth.github.email' | string (optional) | User's GitHub email |
| 'io.flipt.auth.github.name' | string (optional) | User's GitHub name |
| 'io.flipt.auth.github.picture' | string (optional) | URL of user's GitHub profile picture |
| 'io.flipt.auth.github.preferred_username' | string (optional) | User's preferred GitHub username |

### `IAuthGithubInternal`
#### Description
This interface extends the `IAuth` interface to represent an internal GitHub authentication object.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| method | string | Fixed value 'METHOD_GITHUB' |
| metadata | IAuthMethodGithubMetadata | GitHub-specific authentication metadata |

The `Github.ts` file is part of a larger authentication system in the Flipt application. It's used in conjunction with other authentication types (such as JWT and OIDC) to provide a flexible authentication mechanism. The interfaces defined here are used in various parts of the application, including the API client (`api.ts`), session management (`Session.ts`), and the `SessionProvider` component.
