## High-level description
This file contains utility functions for various operations in the UI, including CSS class manipulation, clipboard operations, string formatting, error handling, and generating commands for CLI and cURL. It provides reusable helper functions to support common tasks across the application.

## Code Structure
The file exports multiple utility functions that are not directly related to each other. Each function serves a specific purpose and can be used independently throughout the application.

## Symbols

### `cls`
#### Description
A utility function that combines CSS class names using the `clsx` and `twMerge` libraries.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| args | ClassValue[] | An array of class values to be combined |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| combinedClasses | string | The combined and merged CSS class names |

### `copyTextToClipboard`
#### Description
Copies the given text to the clipboard using the Clipboard API or the `execCommand` method as a fallback.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| text | string | The text to be copied to the clipboard |

|:-----|:-----|:------------|

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| str | string | The input string to be converted to title case |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| titleCasedString | string | The input string converted to title case |

### `stringAsKey`
#### Description

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| str | string | The input string to be converted to a key |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| keyString | string | The input string converted to a key format |

### `truncateKey`
#### Description
Truncates a string to a specified length and adds an ellipsis if necessary.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| str | string | The input string to be truncated |
| len | number | (Optional) The maximum length of the truncated string (default: 25) |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| truncatedString | string | The truncated string |

#### Inputs

| Name | Type | Description |
