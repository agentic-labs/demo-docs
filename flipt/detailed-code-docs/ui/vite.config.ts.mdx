## High-level description
This file configures Vite for a React-based UI project. It sets up the development server, build process, and proxy settings for API requests, enabling seamless integration with a Flipt backend service.

## Symbols

### defineConfig
#### Description
The `defineConfig` function is used to define the Vite configuration object. It provides type checking and autocompletion for the configuration options.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| config | Object | The configuration object for Vite |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| config | Object | The processed configuration object |

#### Internal Logic
The function sets up various configuration options for Vite:
1. Sets the base URL to an empty string.
2. Configures the React plugin.
3. Sets up path aliases for easier imports.
4. Enables manifest generation during build.
5. Configures environment variable prefix.
6. Sets up the development server with proxy settings for API requests.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| @vitejs/plugin-react | Provides React support for Vite |
| path | Used for resolving file paths |
| vite | The Vite build tool and its configuration utilities |

### Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| FLIPT_ADDRESS | string | 'http://localhost:8080' | The address of the Flipt backend service |

## Server Configuration
The development server is configured with the following settings:
- Host: Enabled for all network interfaces
- Origin: http://localhost:5173
- Proxy: Routes API requests to the Flipt backend service

| Endpoint | Proxy Target |
|:---------|:-------------|
| /api/v1 | FLIPT_ADDRESS |
| /auth/v1 | FLIPT_ADDRESS |
| /evaluate/v1 | FLIPT_ADDRESS |
| /internal/v1 | FLIPT_ADDRESS |
| /meta | FLIPT_ADDRESS |

## Build Configuration
- Manifest: Enabled, generating a manifest file during build

## Resolve Configuration
- Alias: Sets up '~' as an alias for the 'src' directory

## Environment Variables
- Prefix: 'FLIPT_' is used for environment variables specific to this project
