## High-level description
This HTML file serves as a development-specific landing page for the Flipt application. It provides a "Not Found" message and instructions for running the UI in development mode, along with styling and necessary scripts for development purposes.

## Symbols

### HTML Structure
#### Description
The HTML structure defines the layout and content of the development landing page for Flipt.

#### Internal Logic
1. Sets up the HTML document with necessary meta tags and title.
2. Includes scripts for React refresh and Vite development server.
3. Loads Tailwind CSS from a CDN.
4. Creates a centered container with Flipt logo, "Not Found" message, and instructions.
5. Provides a command example for running the UI in development mode.
6. Includes a link to the Flipt Discord for help.

### Script Imports
#### Description
The file includes several script imports for development purposes.

#### Internal Logic
1. Imports RefreshRuntime from the Vite development server.
2. Sets up global variables for React refresh functionality.
3. Imports the Vite client script.
4. Loads Tailwind CSS from a CDN.
5. Imports the main application script (main.tsx) from the Vite development server.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| React Refresh | Enables hot module replacement for React components during development |
| Vite | Provides the development server and build tooling |
| Tailwind CSS | Supplies utility-first CSS classes for styling |

## Notes
1. This file is specifically for development purposes, as indicated by the `.dev` in the filename and the use of `localhost` URLs.
2. The page provides a fallback UI when the main application is not running, guiding developers to start the UI development server.
3. The styling uses Tailwind CSS classes for responsive design and consistent appearance.
4. The file includes setup for React's hot module replacement, allowing for a smoother development experience with live updates.
