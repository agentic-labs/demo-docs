## High-level description
This code defines a package-level constant `DefaultNamespace` for the Flipt SDK. This constant represents the default namespace used in Flipt when no other namespace is explicitly specified.

## Symbols

### `DefaultNamespace`
#### Description
A constant string representing the default namespace in Flipt, which is "default". This namespace is protected and always present. When a namespace key is omitted, this default namespace is used.
