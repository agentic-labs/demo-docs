## High-level description
The code in `sdk/go/http/evaluation.sdk.gen.go` defines an HTTP client for the Flipt Evaluation service. It provides methods for performing boolean, variant, and batch evaluations using the Flipt API over HTTP. This generated code translates gRPC calls into HTTP requests and responses.

## Code Structure
The `EvaluationServiceClient` struct represents the HTTP client for the Evaluation service. It contains methods for each gRPC endpoint defined in the Evaluation service, such as `Boolean`, `Variant`, and `Batch`. Each method constructs an HTTP request, sends it to the Flipt server, and unmarshals the response into the corresponding protobuf message.
