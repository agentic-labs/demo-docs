## High-level description
The code defines a Go package `sdk` containing a client for interacting with the Flipt API. It provides functions to retrieve configuration and information from the Flipt server.

## Code Structure
The `Meta` struct represents the Flipt Meta API client. It uses a `meta.MetadataServiceClient` for transport and a `ClientAuthenticationProvider` for authentication. The `GetConfiguration` and `GetInfo` functions use the client to make API calls.
