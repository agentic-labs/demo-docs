## High-level description
This file, `headers.go`, is part of the `generator` package in the Loophole Labs project. It provides a utility function for generating file names with a specific extension for RPC (Remote Procedure Call) related files.

## Symbols

### `extension`
#### Description
A constant string that defines the file extension for RPC-related files.

#### Inputs
None (constant)

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| extension | string | The file extension ".frpc.go" |

### `FileName`
#### Description
A function that appends the predefined extension to a given file name.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| name | string | The base name of the file |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| string | string | The full file name with the ".frpc.go" extension appended |

#### Internal Logic
The function uses the `utils.AppendString` function to concatenate the input `name` with the `extension` constant.
