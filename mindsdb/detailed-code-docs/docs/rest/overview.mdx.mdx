MindsDB provides REST API endpoints, enabling incorporation of AI building blocks into applications.

This section introduces REST API endpoints provided by MindsDB to bring data and AI together.

Follow these steps to get started:

&lt;Steps&gt;
  &lt;Step title="Set up the development environment"&gt;
    Learn more about [usage here](/rest/usage). 
  &lt;/Step&gt;
  &lt;Step title="Connect a data source"&gt;
    Connect your data source to MindsDB via [this endpoint](/rest/databases/create-databases).&lt;br&gt;&lt;/br&gt;
    Explore all available [data sources here](/integrations/data-overview).
  &lt;/Step&gt;
  &lt;Step title="Create and deploy an AI/ML model"&gt;
    Create, train, and deploy AI/ML models within MindsDB via [this endpoint](/rest/models/train-model).&lt;br&gt;&lt;/br&gt;
    Explore all available [AI engines here](/integrations/ai-overview).
  &lt;/Step&gt;
  &lt;Step title="Get predictions"&gt;
    Query for predictions via [this endpoint](/rest/models/query-model).
  &lt;/Step&gt;
&lt;/Steps&gt;
