MindsDB provides SDKs for MongoDB, Python, and JavaScript, enabling incorporation of AI building blocks into these development environments.

This section introduces custom syntax provided by MindsDB to bring data and AI together inside MongoDB, Python, and JavaScript development environments.

Follow these steps to get started:

&lt;Steps&gt;
  &lt;Step title="Set up the development environment"&gt;
    For MongoDB, use [MongoDB Compass](/sdks/mongo/connect/mongo-compass) or [MongoDB Shell](/sdks/mongo/connect/mongo-shell).&lt;br&gt;&lt;/br&gt;
    For Python, [install the package](/sdks/python/installation).&lt;br&gt;&lt;/br&gt;
    For JavaScript, [install the package](/sdks/javascript/installation).&lt;br&gt;&lt;/br&gt;
  &lt;/Step&gt;
  &lt;Step title="Connect a data source"&gt;
    Connect a data source in [MongoDB](/sdks/mongo/databases/insertOne), [Python](/sdks/python/create_database), and [JavaScript](/sdks/javascript/create_database).&lt;br&gt;&lt;/br&gt;
    Explore all available [data sources here](/integrations/data-overview).
  &lt;/Step&gt;
  &lt;Step title="Configure an AI engine"&gt;
    Configure an AI engine in [MongoDB](/sdks/mongo/ml_engines/insertOne), [Python](/sdks/python/create_ml_engine), and [JavaScript](/sdks/javascript/create_ml_engine).&lt;br&gt;&lt;/br&gt;
    Explore all available [AI engines here](/integrations/ai-overview).
  &lt;/Step&gt;
  &lt;Step title="Create and deploy an AI/ML model"&gt;
    Create and deploy an AI/ML model in [MongoDB](/sdks/mongo/models/insertOne), [Python](/sdks/python/create_model), and [JavaScript](/sdks/javascript/create_model).
  &lt;/Step&gt;
  &lt;Step title="Query for predictions"&gt;
    Query for predictions in [MongoDB](/sdks/mongo/models/get-batch-predictions), [Python](/sdks/python/get-batch-predictions), and [JavaScript](/sdks/javascript/batchQuery).
  &lt;/Step&gt;
  &lt;Step title="Automate customized workflows"&gt;
    Automate tasks by scheduling docs in [MongoDB](/sdks/mongo/jobs/insertOne), [Python](/sdks/python/create_job), and [JavaScript](/sdks/javascript/create_job).
  &lt;/Step&gt;
&lt;/Steps&gt;
