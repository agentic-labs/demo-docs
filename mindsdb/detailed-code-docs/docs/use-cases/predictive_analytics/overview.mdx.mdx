title: Predictive Analytics
sidebarTitle: Overview

MindsDB enables seamless time-series forecasting so you can look into the future and get predictions based on historical data. You can also use historical data to detect anomalies, for example, in quality control systems.

  &lt;img src="/assets/use_cases/predictive_analytics.jpg" /&gt;
&lt;/p&gt;

This section covers the following use cases:
    
    - Time-series forecasting
    - Anomaly detection
