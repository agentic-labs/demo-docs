title: AI Workflow Automation
sidebarTitle: Overview
---

With MindsDB, you can create, customize, and automate AI workflows that comprise of connecting a [data source](/integrations/data-overview), deploying an [AI/ML model](/integrations/ai-overview), and streaming predictions and forecast into your application. Use [jobs](/mindsdb_sql/sql/create/jobs) and [triggers](/mindsdb_sql/sql/create/trigger) to create custom automation.

This section covers the following use cases:
    
    - Chatbot automation with jobs
    - Alert systems
