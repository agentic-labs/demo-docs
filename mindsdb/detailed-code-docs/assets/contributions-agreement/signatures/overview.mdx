## High-level description

This file contains a JSON object that stores information about contributors who have signed a Contributor License Agreement (CLA) for a specific project. It includes details such as the contributor's GitHub username, ID, the pull request number associated with their contribution, and the timestamp of their agreement.

## Symbols

### `signedContributors`
#### Description
An array of objects, each representing a contributor who has signed the CLA. Each object contains the following properties:

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| name | string | GitHub username of the contributor |
| id | number | GitHub user ID of the contributor |
| comment_id | number | ID of the comment where the CLA was signed |
| created_at | string | Timestamp of when the CLA was signed |
| repoId | number | ID of the repository |
| pullRequestNo | number | Number of the pull request associated with the contribution |

## Dependencies
This file does not have any explicit dependencies on external libraries or modules. It is a standalone JSON data file.

## Configuration
This file does not contain any configuration options. It is purely a data storage file.
