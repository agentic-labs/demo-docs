## High-level description
This code defines utility functions for working with MongoDB-related data types and boolean operations. It provides functions for converting between ObjectId and integer representations, as well as simple boolean evaluation functions.

## Symbols

### `is_true(val)`
#### Description
This function evaluates whether the given value is considered "true" in a boolean context.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| val | Any | The value to be evaluated |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | bool | True if the input is considered true, False otherwise |

#### Internal Logic
The function first converts the input to a boolean using `bool(val)`, then compares it to `True` using the `is` operator.

### `int_to_objectid(n)`
#### Description
This function converts an integer to a MongoDB ObjectId.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| n | int | The integer to be converted |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | bson.ObjectId | The resulting ObjectId |

#### Internal Logic
1. Converts the input integer to a string.
2. Pads the string with leading zeros to ensure it's 24 characters long.
3. Creates and returns a new `bson.ObjectId` using the padded string.

### `objectid_to_int(obj)`
#### Description
This function converts a MongoDB ObjectId to an integer.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| obj | bson.ObjectId | The ObjectId to be converted |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | int | The integer representation of the ObjectId |

#### Internal Logic
Converts the ObjectId to a string and then to an integer.
