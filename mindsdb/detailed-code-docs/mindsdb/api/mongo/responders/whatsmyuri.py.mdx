## High-level description
This code defines a responder for the MongoDB "whatsmyuri" command in the MindsDB MongoDB API. It returns the host and port of the MongoDB server configuration when queried.

## Code Structure
The main symbol in this code is the `Responce` class, which inherits from the `Responder` class. The `responder` object is an instance of the `Responce` class.

## References
This code references the `Responder` class from `mindsdb.api.mongo.classes` and the `helpers` module from `mindsdb.api.mongo.functions`.
