## High-level description
This file serves as an initialization module for the `api_utilities` package within the MindsDB integrations utilities handlers. It imports and exposes the `MSGraphAPIBaseClient` class from the `microsoft` module, making it available for use in other parts of the MindsDB project.

## Symbols

### `MSGraphAPIBaseClient`
#### Description
`MSGraphAPIBaseClient` is a class imported from the `microsoft` module. It likely provides a base client for interacting with the Microsoft Graph API, which is a RESTful web API that enables access to Microsoft Cloud service resources.
