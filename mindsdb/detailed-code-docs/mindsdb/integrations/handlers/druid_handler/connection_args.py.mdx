## High-level description
This code defines the connection arguments required for establishing a connection to Apache Druid within the MindsDB framework. It specifies the structure and types of connection parameters, including both required and optional fields, using an OrderedDict.

## Symbols

### `connection_args`
#### Description
An OrderedDict that defines the connection arguments for Apache Druid. Each key-value pair represents a connection parameter with its properties.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| host | dict | Configuration for the host parameter |
| port | dict | Configuration for the port parameter |
| path | dict | Configuration for the path parameter |
| scheme | dict | Configuration for the scheme parameter |
| user | dict | Configuration for the user parameter |
| password | dict | Configuration for the password parameter |

#### Internal Logic
- Each parameter is defined with properties such as type, description, required status, and label.
- The `type` property uses values from the `HANDLER_CONNECTION_ARG_TYPE` enum imported from `mindsdb.integrations.libs.const`.
- The `password` parameter has an additional `secret` property set to `True`.

### `connection_args_example`
#### Description
An OrderedDict providing an example of connection arguments for Apache Druid.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| host | str | Example host value |
| port | int | Example port value |
| path | str | Example path value |
| scheme | str | Example scheme value |

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| collections.OrderedDict | Used to create ordered dictionaries for connection arguments |
| mindsdb.integrations.libs.const.HANDLER_CONNECTION_ARG_TYPE | Imported as ARG_TYPE, used to specify the type of connection arguments |

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| host | ARG_TYPE.STR | None | The host name or IP address of Apache Druid |
| port | ARG_TYPE.INT | None | The port that Apache Druid is running on |
| path | ARG_TYPE.STR | None | The query path |
| scheme | ARG_TYPE.STR | 'http' | The URI schema (optional) |
| user | ARG_TYPE.STR | None | The user name for authentication (optional) |
| password | ARG_TYPE.PWD | None | The password for authentication (optional, marked as secret) |

This configuration defines the structure for connecting to Apache Druid, specifying both required and optional parameters with their respective types and descriptions.
