## High-level description
This code defines the Aerospike integration handler for MindsDB. It imports necessary components, sets up the handler class, and defines metadata for the integration such as name, type, and icon path.

## Code Structure
The code imports various components and attempts to import the `AerospikeHandler` class. If the import fails, it sets the `Handler` to `None` and captures the import error. It then defines metadata for the Aerospike integration, including its name, type, and icon path.

## Symbols

### Handler
#### Description
The `Handler` variable is assigned the `AerospikeHandler` class if the import is successful, otherwise it's set to `None`.

### import_error
#### Description
Captures any exception that occurs during the import of `AerospikeHandler`. It's `None` if the import is successful.

### title
#### Description
A string variable set to "Aerospike", representing the display name of the integration.

### name
#### Description
A string variable set to "aerospike", representing the internal name of the integration.

### type
#### Description
Set to `HANDLER_TYPE.DATA`, indicating that this is a data handler integration.

### icon_path
#### Description
A string variable set to "icon.svg", specifying the path to the integration's icon file.
