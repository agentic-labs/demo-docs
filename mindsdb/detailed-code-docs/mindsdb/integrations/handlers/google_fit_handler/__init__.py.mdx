## High-level description
This code defines the Google Fit integration handler for MindsDB. It imports necessary components, sets up the handler class, and defines metadata for the integration such as its name, type, and icon.

## Symbols

### Handler
#### Description
The `Handler` variable is assigned the `GoogleFitHandler` class, which is imported from the `google_fit_handler` module. This class is responsible for handling the integration with Google Fit.

#### Inputs
None

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| Handler | Class | The GoogleFitHandler class |

### import_error
#### Description
This variable captures any exception that occurs during the import of the `GoogleFitHandler`. If an error occurs, `Handler` is set to `None`, and the error is stored in `import_error`.

#### Inputs
None

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
