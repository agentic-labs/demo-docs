## High-level description
This code defines the connection arguments required for establishing a connection to a Rockset database in the MindsDB integration handler. It specifies the structure and types of connection parameters, along with an example of how these parameters should be filled.

## Symbols

### `connection_args`
#### Description
An OrderedDict that defines the structure and types of connection arguments required for connecting to a Rockset database.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| user | ARG_TYPE.STR | Rockset user name |
| password | ARG_TYPE.PWD | Rockset password (marked as secret) |
| api_key | ARG_TYPE.STR | Rockset API key |
| api_server | ARG_TYPE.STR | Rockset API server |
| host | ARG_TYPE.STR | Rockset host |
| port | ARG_TYPE.INT | Rockset port |
| database | ARG_TYPE.STR | Rockset database |

#### Internal Logic
Each key in the OrderedDict represents a connection parameter, and its value is another dictionary containing the type and description of the parameter. The `password` parameter is additionally marked as `secret: True`.

### `connection_args_example`
#### Description
An OrderedDict providing example values for the connection arguments defined in `connection_args`.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|

| password | str | Example Rockset password |
| api_key | str | Example Rockset API key |
| api_server | str | Example Rockset API server |
| host | str | Example Rockset host |
| port | str | Example Rockset port |
| database | str | Example Rockset database |

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|

The code defines the following configuration options for Rockset connection:
| Option | Type | Description |
|:-------|:-----|:------------|
| user | ARG_TYPE.STR | Rockset user name |
| password | ARG_TYPE.PWD | Rockset password (secret) |
| api_key | ARG_TYPE.STR | Rockset API key |
| api_server | ARG_TYPE.STR | Rockset API server |
| host | ARG_TYPE.STR | Rockset host |
| port | ARG_TYPE.INT | Rockset port |

This configuration structure allows for flexible and secure handling of connection parameters for the Rockset integration in MindsDB.
