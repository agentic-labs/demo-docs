## High-level description
This code defines the connection arguments required for the OilPriceAPI handler in the MindsDB platform. It specifies the structure and validation rules for the API key used to authenticate with the OilPriceAPI service.

## Code Structure

- `connection_args` defines the schema for the connection arguments. It specifies the type, description, required status, label, and whether the argument is a secret for each argument.
- `connection_args_example` provides an example of how the connection arguments should be structured.
