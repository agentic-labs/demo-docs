## High-level description
This code defines the connection arguments required for connecting to an SAP SQL Anywhere database in the MindsDB integration handler. It specifies the structure and types of connection parameters, along with an example of how these parameters should be used.

## Symbols

### `connection_args`
#### Description
An OrderedDict that defines the structure and types of connection arguments required for connecting to an SAP SQL Anywhere database.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| host | ARG_TYPE.STR | The IP address/host name of the SAP SQL Anywhere instance host |
| port | ARG_TYPE.STR | The port number of the SAP SQL Anywhere instance |
| user | ARG_TYPE.STR | Specifies the user name |
| password | ARG_TYPE.PWD | Specifies the password for the user (marked as secret) |
| server | ARG_TYPE.STR | Specifies the name of the server to connect to |
| database | ARG_TYPE.STR | Specifies the name of the database to connect to |
| encrypt | ARG_TYPE.BOOL | Enables or disables TLS encryption |

#### Internal Logic
Each key in the OrderedDict represents a connection parameter, and its value is another dictionary containing the type and description of the parameter. The `password` parameter is additionally marked as `secret: True`.

### `connection_args_example`
