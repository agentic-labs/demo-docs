## High-level description
This code defines the connection arguments required for connecting to a Google Sheets document in the MindsDB integration handler for Google Sheets. It specifies the structure of the connection arguments and provides an example of how these arguments should be used.

## Symbols

### `connection_args`
#### Description
An OrderedDict that defines the required connection arguments for the Google Sheets handler. It specifies two arguments: `spreadsheet_id` and `sheet_name`.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| spreadsheet_id | ARG_TYPE.STR | The unique ID of the Google Sheet |
| sheet_name | ARG_TYPE.STR | The name of the sheet within the Google Sheet |

#### Internal Logic
The `connection_args` OrderedDict uses the argument names as keys and dictionaries as values. Each dictionary contains two key-value pairs:
1. `'type'`: Specifies the type of the argument using the `ARG_TYPE` enum from the `mindsdb.integrations.libs.const` module.
2. `'description'`: Provides a brief description of the argument.

### `connection_args_example`
#### Description
An OrderedDict that provides an example of how the connection arguments should be filled out for the Google Sheets handler.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| spreadsheet_id | str | An example Google Sheet ID |
| sheet_name | str | An example sheet name |

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| collections.OrderedDict | Used to create ordered dictionaries for `connection_args` and `connection_args_example` |

## References
This code references the `HANDLER_CONNECTION_ARG_TYPE` (aliased as `ARG_TYPE`) from the `mindsdb.integrations.libs.const` module. This enum is used to specify the types of the connection arguments.

The `HANDLER_CONNECTION_ARG_TYPE` enum includes the following types:
- STR: String
- INT: Integer
- BOOL: Boolean
- URL: URL
- PATH: File path
- DICT: Dictionary
- PWD: Password

In this specific code, only the `STR` (string) type is used for both `spreadsheet_id` and `sheet_name`.
