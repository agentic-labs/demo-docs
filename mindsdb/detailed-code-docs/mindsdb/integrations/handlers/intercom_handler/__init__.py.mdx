## High-level description
This code defines the Intercom handler for MindsDB, which allows integration with the Intercom API. It sets up the necessary attributes and imports for the handler, including version information, handler type, and error handling for import issues.

## Symbols

### Handler
#### Description
The `Handler` symbol is intended to be the main class for interacting with the Intercom API. It is imported from the `intercom_handler` module if the import is successful.

#### Inputs
N/A (This is a class import, not a function)

#### Outputs
N/A (This is a class import, not a function)

### import_error
#### Description
This variable captures any exception that occurs during the import of the `IntercomHandler` class. It's used for error handling and reporting.

#### Inputs
N/A (This is a variable assignment)

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| import_error | Exception or None | Contains the exception if an error occurred during import, otherwise None |

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| title | str | "Intercom" | The display title for the Intercom handler |
| name | str | "intercom" | The internal name for the Intercom handler |
| type | HANDLER_TYPE | HANDLER_TYPE.DATA | Specifies that this is a data handler |
| icon_path | str | "icon.svg" | Path to the icon file for the Intercom handler |

## Error Handling
The code implements a basic error handling mechanism for the import of the `IntercomHandler` class. If an exception occurs during the import, it is caught and stored in the `import_error` variable, while the `Handler` is set to `None`.

## API/Interface Reference
This file defines the following symbols that can be imported when using the Intercom handler:

| Symbol | Type | Description |
|:-------|:-----|:------------|
| Handler | class or None | The main IntercomHandler class (if import was successful) |
| version | str | Version of the Intercom handler |
| name | str | Internal name of the handler |
| type | HANDLER_TYPE | Type of the handler (DATA) |
| title | str | Display title for the handler |
| description | str | Description of the handler |
| import_error | Exception or None | Any error that occurred during import |
| icon_path | str | Path to the handler's icon file |

These symbols are explicitly listed in the `__all__` variable, making them available when importing from this module.
