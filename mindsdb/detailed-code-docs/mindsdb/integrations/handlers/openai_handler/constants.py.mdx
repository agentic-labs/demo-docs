## High-level description
This file defines constants used in the OpenAI handler for MindsDB. It includes various model names, API base URL, and categorizes models into different types such as chat models, completion models, fine-tuning models, embedding models, and image models.

## Symbols

### `OPENAI_API_BASE`
#### Description
Defines the base URL for the OpenAI API.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| N/A | str | Hardcoded string value |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| OPENAI_API_BASE | str | The base URL for OpenAI API |
