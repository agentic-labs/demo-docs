## High-level description
This SQL script is designed to set up a test database and table for the MSSQL handler in MindsDB. It creates a new database, a table with various data types, and inserts sample data into the table.

## Symbols

### Database Creation
#### Description
The script starts by dropping the existing database if it exists and then creates a new database named `mdb_db_handler_test`.

#### Internal Logic
1. Drops the existing database if it exists using `DROP DATABASE IF EXISTS`.
2. Creates a new database using `CREATE DATABASE`.
3. Switches to the newly created database using `USE`.

### Table Creation
#### Description
Creates a table named `test` with four columns of different data types.

#### Internal Logic
Creates a table with the following structure:
- `col_one`: Integer
- `col_two`: Integer
- `col_three`: Float
- `col_four`: VARCHAR(20)

### Data Insertion
#### Description
Inserts three rows of sample data into the `test` table.

#### Internal Logic
Uses `INSERT INTO` statements to add the following data:
1. (1, -1, 0.1, 'A')
2. (2, -2, 0.2, 'B')
3. (3, -3, 0.3, 'C')

## Side Effects
- Drops the existing `mdb_db_handler_test` database if it exists.
- Creates a new database named `mdb_db_handler_test`.
- Creates a new table named `test` in the `mdb_db_handler_test` database.
- Inserts three rows of data into the `test` table.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| MSSQL Server | The script is written in T-SQL and requires a Microsoft SQL Server environment to execute. |

## Performance Considerations
This script is relatively lightweight and should execute quickly. However, dropping and recreating a database can be a resource-intensive operation, especially if the database is large or if there are active connections.
