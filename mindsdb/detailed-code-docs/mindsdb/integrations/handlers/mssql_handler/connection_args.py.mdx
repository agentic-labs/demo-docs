## High-level description
This code defines the connection arguments required for establishing a connection to a Microsoft SQL Server database in the MindsDB integration handler. It specifies the necessary parameters, their types, descriptions, and whether they are required or optional.

## Symbols

### `connection_args`
#### Description
An OrderedDict that defines the connection arguments for connecting to a Microsoft SQL Server database. Each key-value pair represents a connection parameter with its properties.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| user | dict | Specifies the username for authentication |
| password | dict | Specifies the password for authentication |
| database | dict | Specifies the database name to connect to |
| host | dict | Specifies the host name or IP address of the server |
| port | dict | Specifies the TCP/IP port of the server |
| server | dict | Specifies the server name (for named instances or Azure SQL Database) |

#### Internal Logic
Each connection argument is defined with the following properties:
- `type`: The data type of the argument (STR, PWD, or INT)
- `description`: A brief explanation of the argument's purpose
- `required`: A boolean indicating if the argument is mandatory
- `label`: A user-friendly label for the argument
- `secret`: (Only for password) A boolean indicating if the value should be treated as sensitive

### `connection_args_example`
#### Description
An OrderedDict providing an example of how to use the connection arguments with sample values.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| host | str | Example host address |
| port | int | Example port number |
| user | str | Example username |
| password | str | Example password |
| database | str | Example database name |

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| collections.OrderedDict | Used to create ordered dictionaries for connection arguments |
| mindsdb.integrations.libs.const.HANDLER_CONNECTION_ARG_TYPE | Imported as ARG_TYPE, used to specify argument types |

This code provides a structured way to define and validate connection parameters for the Microsoft SQL Server integration in MindsDB. It ensures that all necessary information is collected from the user to establish a successful database connection.
