## High-level description
This code defines the configuration settings for the Microsoft Teams handler in the MindsDB integration. It includes default scopes for API access, table column definitions, and test data structures for various Teams entities like chats, messages, and channels.

## Code Structure
The code defines a single class `MSTeamsHandlerConfig` which inherits from `BaseSettings`. This class contains all the configuration settings as class attributes. At the end of the file, an instance of this class is created as `ms_teams_handler_config`.
