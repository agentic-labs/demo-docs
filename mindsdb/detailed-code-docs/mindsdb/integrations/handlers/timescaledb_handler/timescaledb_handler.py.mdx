## High-level description
This code defines a `TimeScaleDBHandler` class that extends the `PostgresHandler` class to handle connections and operations with TimeScaleDB, a time-series database built on top of PostgreSQL. It also specifies the connection arguments required for establishing a connection to a TimeScaleDB instance.

## Code Structure
The code consists of a single class `TimeScaleDBHandler` and two dictionaries `connection_args` and `connection_args_example`. The `TimeScaleDBHandler` inherits from `PostgresHandler`, while the dictionaries define the connection parameters and provide an example configuration.

## Symbols

### TimeScaleDBHandler
#### Description
A handler class for TimeScaleDB that extends the PostgreSQL handler. It maintains the same functionality as the PostgreSQL handler but is specifically named for TimeScaleDB.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| name | str | The name of the handler instance |
| **kwargs | dict | Additional keyword arguments |

#### Internal Logic
The class initializes by calling the superclass (`PostgresHandler`) constructor with the provided name and keyword arguments.

### connection_args
#### Description
An ordered dictionary that defines the connection arguments required for connecting to a TimeScaleDB instance.

#### Internal Logic
Defines five connection parameters:
1. host: The hostname or IP address of the TimeScaleDB server
2. database: The name of the database to connect to
3. user: The username for authentication
4. password: The password for authentication
5. port: The port number for the connection

Each parameter is defined with a type (from `ARG_TYPE`) and a description.

### connection_args_example
#### Description
An ordered dictionary providing an example configuration for connecting to a TimeScaleDB instance.

#### Internal Logic
Contains sample values for each of the connection parameters defined in `connection_args`:
- host: '127.0.0.1'
- port: 36806
- password: 'P455W0rD'
- user: 'tsdbadmin'
- database: 'tsdb'

## Configuration
The configuration options for the TimeScaleDB connection are defined in the `connection_args` dictionary:
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| host | ARG_TYPE.STR | None | The host name or IP address of the TimeScaleDB server/database |
| database | ARG_TYPE.STR | None | The database name to use when connecting with the TimeScaleDB server |
| user | ARG_TYPE.STR | None | The user name used to authenticate with the TimeScaleDB server |
| password | ARG_TYPE.STR | None | The password to authenticate the user with the TimeScaleDB server |
| port | ARG_TYPE.INT | None | Specify port to connect TimeScaleDB |
