## High-level description
This code defines the connection arguments required for integrating with the Symbl API in the MindsDB framework. It specifies the necessary credentials (App ID and App Secret) for authenticating with the Symbl service.

## Symbols

### `connection_args`
#### Description
An OrderedDict that defines the structure and properties of the connection arguments required for the Symbl integration.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| app_id | dict | Specifies the properties of the App ID argument |
| app_secret | dict | Specifies the properties of the App Secret argument |

#### Internal Logic
- Both `app_id` and `app_secret` are defined as password-type arguments (ARG_TYPE.PWD).
- `app_id` is labeled as "app_id" and is required.
- `app_secret` is labeled as "app_secret", is required, and is marked as secret.

### `connection_args_example`
#### Description
An OrderedDict providing example values for the connection arguments.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| app_id | str | Example value for the App ID |
| app_secret | str | Example value for the App Secret |

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| collections.OrderedDict | Used to create ordered dictionaries for structured data |
| mindsdb.integrations.libs.const.HANDLER_CONNECTION_ARG_TYPE | Imported as ARG_TYPE, used to specify argument types |

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| app_id | ARG_TYPE.PWD | N/A | App ID for Symbl authentication |
| app_secret | ARG_TYPE.PWD | N/A | App Secret for Symbl authentication |
