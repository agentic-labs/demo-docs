## High-level description
This code defines the connection arguments for the Milvus handler in MindsDB. It specifies the required and optional parameters needed to establish a connection with a Milvus server, as well as configuration options for search and table creation operations.

## Symbols

### `connection_args`
#### Description
An OrderedDict that defines the connection arguments for the Milvus handler. It specifies the type, description, and whether each argument is required.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| alias | STR | Alias of the Milvus connection to construct |
| host | STR | IP address of the Milvus server |
| port | INT | Port of the Milvus server |
| user | STR | Username of the Milvus server |
| password | PWD | Password of the username of the Milvus server |
| search_default_limit | INT | Default limit to be passed in select statements |
| search_metric_type | STR | Metric type used for searches |
| search_ignore_growing | BOOL | Whether to ignore growing segments during similarity searches |
| search_params | DICT | Specific to the `search_metric_type` |
| create_auto_id | BOOL | Whether to auto generate id when inserting records with no ID |
| create_id_max_len | STR | Maximum length of the id field when creating a table |
| create_embedding_dim | INT | Embedding dimension for creating table |
| create_dynamic_field | BOOL | Whether or not the created tables have dynamic fields |
| create_content_max_len | INT | Max length of the content column |
| create_content_default_value | STR | Default value of content column |
| create_schema_description | STR | Description of the created schemas |
| create_alias | STR | Alias of the created schemas |
| create_index_params | DICT | Parameters of the index created on embeddings column |
| create_index_metric_type | STR | Metric used to create the index |
| create_index_type | STR | The type of index |

### `connection_args_example`
#### Description
An OrderedDict that provides example values for the connection arguments defined in `connection_args`.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| collections.OrderedDict | Used to create ordered dictionaries for connection arguments |
| mindsdb.integrations.libs.const.HANDLER_CONNECTION_ARG_TYPE | Provides constants for argument types |

## Notes
1. The `password` argument is marked as `secret: True`, indicating that it should be handled securely.
2. Many of the `create_*` arguments are optional and have default values specified in their descriptions.
3. The `search_params` argument is a dictionary that is specific to the `search_metric_type`.
4. The `create_index_params` argument is an empty dictionary by default, allowing for custom index parameters.

This code provides a comprehensive set of connection arguments for the Milvus handler, covering both basic connection details and advanced configuration options for search and table creation operations.
