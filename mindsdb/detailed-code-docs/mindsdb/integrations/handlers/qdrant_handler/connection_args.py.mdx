## High-level description
This code defines the connection arguments for the Qdrant handler in MindsDB. It specifies the required and optional parameters needed to establish a connection with a Qdrant instance, including both local and remote configurations.

## Symbols

### `connection_args`
#### Description
An OrderedDict that defines the connection arguments for the Qdrant handler. Each key-value pair represents a connection parameter with its properties.

#### Inputs
N/A (This is a data structure definition)

#### Outputs
N/A (This is a data structure definition)

#### Internal Logic
The OrderedDict contains the following key connection parameters:
1. `location`: Specifies the Qdrant instance location (in-memory or remote URL).
2. `url`: Full URL of the Qdrant service.
3. `host`: Hostname of the Qdrant service.
4. `port`: REST API port (default: 6333).
5. `grpc_port`: gRPC interface port (default: 6334).
6. `prefer_grpc`: Boolean to prefer gRPC interface when possible.
7. `https`: Boolean to use HTTPS protocol.
8. `api_key`: API key for Qdrant Cloud authentication.
9. `prefix`: Additional path prefix for REST URL.
10. `timeout`: Timeout for API requests.
11. `path`: Persistence path for local Qdrant instance.
12. `collection_config`: Configuration for collection creation (required).

Each parameter is defined with its type (from `ARG_TYPE`), description, and whether it's required or not. The `api_key` is marked as secret.

## Configuration
The `connection_args` OrderedDict serves as a configuration template for the Qdrant handler. Each key in the dictionary represents a configurable option:

| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| location | STR | None | Specifies Qdrant instance location |
| url | STR | None | Full URL of Qdrant service |
| host | STR | None | Hostname of Qdrant service |
| port | INT | 6333 | REST API port |
| grpc_port | INT | 6334 | gRPC interface port |
| prefer_grpc | BOOL | false | Preference for gRPC interface |
| https | BOOL | None | Use of HTTPS protocol |
| api_key | PWD | None | API key for Qdrant Cloud |
| prefix | STR | None | Additional path prefix for REST URL |
| timeout | INT | 5.0 (REST), unlimited (gRPC) | Timeout for API requests |
| path | STR | None | Persistence path for local instance |
| collection_config | DICT | None | Collection creation configuration |

Note: The `collection_config` is the only required parameter in this configuration.
