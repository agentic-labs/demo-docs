## High-level description
This code defines the connection arguments for the LibSQL handler in MindsDB. It specifies the required and optional parameters for establishing a connection to a LibSQL database, including the database file, sync URL, and authentication token.

## Symbols

### `connection_args`
#### Description
An OrderedDict that defines the connection arguments for the LibSQL handler. It specifies the type and description of each argument.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| database | ARG_TYPE.STR | The database file where the data will be stored. |
| sync_url | ARG_TYPE.STR | The database URL where the data is synced with. |
| auth_token | ARG_TYPE.STR | The JWT auth token to authenticate with the sync database. |

#### Internal Logic
- Uses `OrderedDict` to maintain the order of arguments.
- Each argument is defined with a type (from `ARG_TYPE`) and a description.
- The `auth_token` argument is marked as `secret: True` for security purposes.
