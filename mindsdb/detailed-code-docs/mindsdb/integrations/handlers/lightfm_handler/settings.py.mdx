## High-level description
This code defines a `ModelParameters` class using Pydantic's `BaseModel`. It specifies default parameters for a LightFM recommendation model, including learning rate, loss function, and number of epochs.

## Symbols

### `ModelParameters`
#### Description
`ModelParameters` is a Pydantic model that defines the configuration parameters for a LightFM recommendation model. It inherits from `pydantic.BaseModel`, which provides data validation and settings management.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| learning_rate | float | The learning rate for the model, defaulting to 0.05 |
| loss | str | The loss function to be used, defaulting to "warp" |
| epochs | int | The number of training epochs, defaulting to 10 |

#### Internal Logic
The class defines three attributes with default values:
1. `learning_rate`: A float value defaulting to 0.05, which controls the step size at each iteration while moving toward a minimum of the loss function.
2. `loss`: A string defaulting to "warp" (Weighted Approximate-Rank Pairwise), which specifies the loss function used in the LightFM model.
3. `epochs`: An integer defaulting to 10, which determines the number of complete passes through the training dataset.

These parameters are commonly used in machine learning models, particularly in recommendation systems like LightFM.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|

## Configuration
The `ModelParameters` class itself serves as a configuration object for the LightFM model. The default values are:

| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| learning_rate | float | 0.05 | The learning rate for the model |
| loss | str | "warp" | The loss function to be used |
| epochs | int | 10 | The number of training epochs |

These default values can be overridden when creating an instance of `ModelParameters`.
