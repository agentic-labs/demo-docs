## High-level description
The `CockroachHandler` class is a specialized handler for CockroachDB, inheriting from the `PostgresHandler`. It provides a connection and execution interface for CockroachDB statements, leveraging the PostgreSQL-compatible nature of CockroachDB.

## Code Structure
The `CockroachHandler` class extends the `PostgresHandler` class, inheriting all its functionality while specifying a unique name for the handler.
